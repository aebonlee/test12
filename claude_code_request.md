제목: 국내 투자유치 뉴스 웹 스크래핑 및 SQLite DB 저장 파이썬 스크립트 작성 요청

설명:
다음 요구사항을 충족하는 파이썬 스크립트를 작성해 주세요. 이 스크립트는 지정된 한국 뉴스 웹사이트들로부터 투자 유치 관련 뉴스 기사를 수집하고, 수집된 데이터를 SQLite 데이터베이스에 저장해야 합니다.

주요 요구사항:
1.  **대상 웹사이트:** 스크래핑 대상 뉴스 웹사이트 목록은 다음과 같습니다.
    *   thevc.kr (더브이씨)
    *   www.venturesquare.net (벤처스퀘어)
    *   platum.kr (플래텀)
    *   startuptoday.kr (스타트업투데이)
    *   startupn.kr (스타트업엔)
    *   outstanding.kr (아웃스탠딩)
    *   mobiinside.co.kr (모비인사이드)
    *   www.zdnet.co.kr (지디넷코리아)
    *   www.thebell.co.kr (더벨)
    *   nextunicorn.kr (넥스트유니콘 - 파인더)
    *   www.epnc.co.kr (테크월드뉴스)
    *   www.aitimes.com (AI타임스)
    *   www.vmnews.co.kr (벤처경영신문)
    *   www.newstopkorea.com (뉴스톱)
    *   www.bloter.net (블로터)
    *   www.economist.co.kr (이코노미스트)
    *   www.mk.co.kr/news/it (매일경제 - MK테크리뷰)
    *   news.daum.net/section/2/venture (다음뉴스 - 벤처/스타트업 섹션)
    *   www.korea.kr (대한민국 정책브리핑)
    *   www.kvca.or.kr (한국벤처캐피탈협회)

2.  **데이터 수집 범위:**
    *   `2026년 1월 1일` 이후에 발행된 기사만 수집해야 합니다.
    *   기사 제목 또는 본문에 다음 키워드 중 하나 이상이 포함된 경우에만 수집해야 합니다: "투자유치", "스타트업", "시리즈A", "시리즈B", "시리즈C", "프리A", "시드", "펀딩", "투자", "VC", "벤처캐피탈", "엑셀러레이터".

3.  **수집할 데이터 필드:** 각 기사로부터 다음 정보를 추출하여 저장해야 합니다.
    *   `title` (기사 제목)
    *   `url` (기사 원문 URL)
    *   `date` (발행일자 및 시간 - 가능하면 정확하게, 불가능하면 일자만이라도)
    *   `source` (뉴스 웹사이트 이름, 예: "벤처스퀘어", "플래텀")
    *   `keywords` (수집 조건에 부합한 키워드 중 기사에 포함된 키워드 목록)

4.  **데이터베이스 저장:**
    *   수집된 모든 데이터는 `news_data.db`라는 이름의 SQLite 데이터베이스 파일에 저장되어야 합니다.
    *   데이터베이스 스키마는 `id` (PRIMARY KEY, AUTOINCREMENT), `title`, `url`, `date`, `source`, `keywords` 필드를 포함해야 합니다.
    *   URL을 기반으로 중복 기사를 방지하는 로직을 포함해야 합니다. (기존에 동일 URL이 있으면 업데이트하거나 건너뛰기)

5.  **예외 처리 및 견고성:**
    *   각 웹사이트의 구조가 다를 수 있으므로, 웹 스크래핑 시 발생하는 오류(예: 페이지 없음, 요소 찾기 실패)를 적절히 처리하여 스크립트가 중단되지 않고 다음 웹사이트로 진행될 수 있도록 해야 합니다.
    *   과도한 요청으로 인한 웹사이트 차단을 방지하기 위해 요청 사이에 적절한 지연 시간을 두어야 합니다.

6.  **사용 라이브러리:** `requests`, `BeautifulSoup` 등 파이썬 표준 및 일반적인 웹 스크래핑 라이브러리를 사용할 수 있습니다. `selenium`과 같은 브라우저 자동화 도구는 복잡성 증가 및 실행 환경 제약으로 인해 가급적 사용하지 않는 방향으로 고려해 주세요.

이 스크립트는 사용자가 직접 실행할 수 있어야 합니다.
