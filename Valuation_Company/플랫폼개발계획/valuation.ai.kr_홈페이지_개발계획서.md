## 4. 기업가치평가 페이지 (/valuation)

### 4.1 5가지 평가방법 설명

#### DCF 평가 (800만원)
```
🎯 추천 대상: 투자유치, M&A 진행 기업
📊 특징: 미래 현금흐름 기반 정밀 평가
📋 산출물: 20-30페이지 DCF 모델 및 평가보고서
⏰ 소요기간: 7-10일
```

#### 상대가치 평가 (500만원)
```
🎯 추천 대상: 간편 비교 평가 필요 기업
📊 방법: PER, PBR, PSR, EV/EBITDA
📋 산출물: 15-20페이지 멀티플 분석 보고서
⏰ 소요기간: 5-7일
```

#### 상증법 평가 (1,000만원)
```
🎯 추천 대상: 가업승계, 증여 예정 기업
📊 특징: 세법 기준 준수, 절세 전략 포함
📋 산출물: 국세청 인정 평가보고서
⏰ 소요기간: 10-14일
```

#### IPO 평가 (2,000만원)
```
🎯 추천 대상: 상장 준비 기업
📊 특징: 공모가 산정, 증권신고서용
📋 산출물: 40-50페이지 IPO 평가보고서
⏰ 소요기간: 14-21일
```

#### 자산가치 평가 (600만원)
```
🎯 추천 대상: 자산 중심 기업, 지주회사
📊 특징: 순자산가치 기반 평가
📋 산출물: 자산가치 평가보고서
⏰ 소요기간: 7-10일
```

### 4.2 평가 프로세스
```
1단계: 상담 및 견적 (1일)
2단계: 자료 수집 (2-3일)
3단계: AI 초안 생성 (1일)
4단계: 회계사 검증 (2-3일)
5단계: 최종 보고서 (1일)
```

## 5. 기업 랭킹 페이지 (/ranking)

### 5.1 랭킹 기준 (투명 공개)
```
📊 종합 점수 계산 (100점 만점)
- DCF 가치: 40점
- 성장성: 20점 (매출 성장률, 시장 확장성)
- 수익성: 20점 (영업이익률, ROE)
- 안정성: 10점 (부채비율, 현금흐름)
- 경영진 역량: 10점 (경력, 성과)
```

### 5.2 필터 기능
```
🔍 상세 필터링
- 업종: IT, 바이오, 제조, 서비스, 금융
- 규모: 50억 미만, 50-100억, 100-500억, 500억+
- 단계: 시드, 시리즈A/B/C, Pre-IPO
- 지역: 서울, 경기, 부산, 대구, 기타
- 설립연도: 3년 미만, 3-7년, 7-15년, 15년+
```

### 5.3 랭킹 카테고리
```
🏆 전체 종합 랭킹 (TOP 100)
💎 가장 저평가된 기업 (TOP 20)
🚀 급성장 기업 (성장률 기준 TOP 20)
💰 투자 효율성 TOP (ROI 기준 TOP 20)
⭐ 이달의 주목 기업 (에디터 선정)
```

### 5.4 기업 상세 페이지 (/company/[id])
```
📊 기업 대시보드
- 기본 정보 (업종, 설립연도, 직원 수)
- 평가 정보 (추정 가치, 평가 날짜)
- 랭킹 정보 (전체 순위, 업종 순위)
- 재무 하이라이트 (매출, 영업이익, 성장률)

🤖 AI 아바타 IR (유료 기업만)
- 실시간 Q&A 채팅
- 24시간 언제든 문의 가능
- 투자 관련 질문 특화

📈 관심도 분석
- 최근 7일 조회수
- 투자자 관심도 지수
- 경쟁사 대비 관심도

💼 투자자 액션
- [관심 기업 저장]
- [상세 보고서 구매] (10만원)
- [연락처 요청]
```

## 6. AI 아바타 IR 페이지 (/ai-avatar)

### 6.1 서비스 소개
```
🤖 24시간 언제든 기업 담당자와 대화
- 투자 관련 질문 즉시 답변
- 사업계획, 재무현황, 성장전략 상담
- 한국어, 영어, 중국어, 일본어 지원
- 실제 담당자 수준의 전문성
```

### 6.2 AI 아바타 예시
```
💬 대화 미리보기

투자자: "올해 매출 목표는 어떻게 되나요?"
AI 아바타: "안녕하세요! 저희 올해 매출 목표는 120억원입니다. 
작년 대비 40% 성장을 목표로 하고 있으며, 주요 
성장동력은 신제품 출시와 해외시장 진출입니다."

투자자: "경쟁사 대비 차별점이 뭔가요?"
AI 아바타: "저희만의 핵심 차별점은 3가지입니다. 
첫째, 특허받은 독자 기술로 30% 효율성 향상을 
달성했습니다. 둘째..."
```

### 6.3 가격 및 신청
```

- 월 이용료: 200만원
- 설정 기간: 7일
- 포함 사항: 
  * 24시간 무제한 대화
  * 다국어 지원
  * 대화 분석 리포트

├── 기대 수익률: [10%] [20%] [30%] [50%+]

```

## 13. 차별화 전략

### vs 일반 평가 서비스
- 우리: 평가 + 랭킹 + 매칭
- 타사: 평가만

### vs 투자 플랫폼
- 우리: 객관적 평가 기반
- 타사: 자체 작성 IR

### vs 정보 제공 서비스
- 우리: 실시간 투자자 매칭
- 타사: 단순 정보 제공
