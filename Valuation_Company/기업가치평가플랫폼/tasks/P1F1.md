# P1F1: 웹사이트 목업 HTML 생성

**Phase**: 1 - 코어 엔진 개발
**영역**: Frontend
**담당 AI**: ui-designer
**작업 ID**: P1F1

---

## 📋 작업 개요

5가지 평가법(DCF, 상대가치, 자산가치, 배당할인, 청산가치)과 6가지 시뮬레이터 소개를 포함한 웹사이트 목업 HTML을 생성합니다.

---

## 🎯 작업 목표

1. 5가지 평가법 입력 화면 목업 디자인
2. 인터랙티브 HTML 데모 제작
3. 사용자가 시각적으로 확인 가능한 형태로 제작
4. 각 평가법별 입력 항목 명확히 정의

---

## 📝 상세 요구사항

### 1. 5가지 평가법 입력 폼

#### DCF (현금흐름할인법)
- 기본 정보 입력 (회사명, 평가기준일)
- FCFF 예측 테이블 (5개년)
- WACC 계산 섹션 (CAPM 모델)
- Terminal Value 계산 (Gordon Growth Model)
- 비영업자산/부채 입력
- 주식 정보 (발행주식수)

#### 상대가치평가법
- 기본 정보 입력
- 비교 기업 선택 (최소 3개)
- 재무 지표 입력 (P/E, P/B, P/S, EV/EBITDA)
- 멀티플 조정 (프리미엄/디스카운트)

#### 자산가치평가법 (NAV)
- 자산 항목별 장부가/시장가 입력
- 부채 항목 입력
- 순자산가치 계산

#### 배당할인모형 (DDM)
- 과거 배당 이력 입력
- 배당 성장률 예측
- 할인율 (요구수익률) 입력
- Gordon Growth Model 적용

#### 청산가치평가법
- 자산별 청산가치 입력 (강제청산/정리청산)
- 청산 비용 입력
- 채무 우선순위 반영

### 2. UI/UX 요구사항

- **탭 네비게이션**: 5가지 평가법 전환
- **진행 단계 표시**: 각 평가법마다 Step-by-Step
- **실시간 결과 미리보기**: 입력 시 계산 결과 표시
- **반응형 디자인**: 데스크탑/태블릿/모바일 대응
- **시각적 계층 구조**: 섹션별 명확한 구분

### 3. 스타일링

- Gradient 배경 사용
- 깔끔한 카드 레이아웃
- 가독성 좋은 폰트 (sans-serif)
- 입력 필드 명확한 라벨링
- 색상 코드:
  - Primary: #667eea (보라)
  - Secondary: #764ba2 (진한 보라)
  - Success: #48bb78 (녹색)
  - Warning: #f6ad55 (주황)

## 🔧 기술 스택

- HTML5
- CSS3 (인라인 스타일 또는 `<style>` 태그)
- JavaScript (탭 전환, 실시간 계산)
- 외부 라이브러리 없이 순수 HTML로 제작

### HTML 구조
```html
<!DOCTYPE html>
<html>
<head>
    <title>기업가치평가 플랫폼 - 입력 폼 데모</title>
    <style>/* 스타일 */</style>
</head>
<body>
    <div class="container">
        <h1>기업가치평가 플랫폼</h1>

        <!-- 탭 네비게이션 -->
        <div class="tabs">
            <button class="tab active">DCF법</button>
            <button class="tab">상대가치평가</button>
            <!-- ... -->
        </div>

        <!-- 각 평가법 폼 -->
        <div id="dcf" class="tab-content active">
            <!-- DCF 입력 폼 -->
        </div>
        <!-- ... -->
    </div>
    <script>/* 탭 전환 로직 */</script>
</body>
</html>
```
