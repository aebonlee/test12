## 📝 OWASP Top 10 체크리스트

1. **A01: Broken Access Control** - RLS 정책 적용
2. **A02: Cryptographic Failures** - 재무 데이터 AES-256 암호화
3. **A03: Injection** - SQL Injection 방어
4. **A04: Insecure Design** - 인간 승인 시스템 워크플로우
5. **A05: Security Misconfiguration** - 환경 변수 관리
6. **A06: Vulnerable Components** - Dependabot 자동 패치
7. **A07: Authentication Failures** - JWT 보안, 2FA
8. **A08: Data Integrity Failures** - 계산 결과 무결성 검증
9. **A09: Logging Failures** - 감사 로그 기록
10. **A10: SSRF** - API 요청 검증
