## 1. DCF 평가법 - 판단 필요 항목

#### C. WACC (가중평균자본비용) ⭐⭐⭐

**왜 판단이 필요한가?**
- 기업 고유 위험을 반영해야 함
- 비상장 기업 → 베타 추정 어려움
- 신생 기업 → 위험 프리미엄 추가

**AI 제시 방식**:
```
┌─────────────────────────────────────────────┐
│ 📊 WACC (할인율) 시나리오                    │
├─────────────────────────────────────────────┤
│                                             │
│ [자동 계산 결과]                             │
│                                             │
│ 1. 무위험이자율 (Rf): 3.5% (국고채 3년물)   │
│ 2. 업종 베타: 1.15 (소프트웨어)             │
│ 3. 시장위험 프리미엄 (MRP): 5.5%            │
│                                             │
│ → 자기자본비용 (Re):                         │
│    3.5% + 1.15 × 5.5% = 9.83%              │
│                                             │
│ 4. 타인자본비용 (Rd): 4.5% (차입금 이자율)  │
│ 5. 자본구조: 자기자본 85% / 부채 15%        │
│                                             │
│ → 기본 WACC:                                │
│    9.83% × 85% + 4.5% × (1-25%) × 15%      │
│    = 8.86%                                  │
│                                             │
│ ─────────────────────────────────────       │
│                                             │
│ ❗ 추가 위험 고려 필요:                       │
│                                             │
│ 🔵 시나리오 1: 낙관적 (기본 WACC)            │
│    8.86% (추가 위험 없음)                    │
│                                             │
│ 🟢 시나리오 2: 중립적 ⭐ 추천                │
│    10.0% (비상장 위험 프리미엄 +1%)         │
│                                             │
│ 🟡 시나리오 3: 보수적                       │
│    12.0% (신생 기업 위험 프리미엄 +3%)      │
│                                             │
│ [ 시나리오 1 ]  [ 시나리오 2 ]  [ 시나리오 3 ] │
│                                             │
│ 또는 직접 입력: WACC = [____]%              │
│                                             │
└─────────────────────────────────────────────┘
```
