## DCF 평가법 입력 데이터

### 재무 데이터 (Historical)

| 항목 | 필수 | 연도 범위 | 단위 | 출처 |
|------|------|---------|------|------|
| **매출액** | ✅ | 최근 5년 | 백만원 | 손익계산서 |
| **영업이익** | ✅ | 최근 5년 | 백만원 | 손익계산서 |
| **EBITDA** | ✅ | 최근 5년 | 백만원 | 계산 (영업이익+감가상각) |
| **당기순이익** | ✅ | 최근 5년 | 백만원 | 손익계산서 |
| **CAPEX** | ✅ | 최근 5년 | 백만원 | 현금흐름표 |
| **감가상각비** | ✅ | 최근 5년 | 백만원 | 현금흐름표 |
| **운전자본 변화** | ⚪ | 최근 5년 | 백만원 | 재무상태표 |
| **법인세** | ✅ | 최근 5년 | 백만원 | 손익계산서 |

**계산 로직**:
```
Free Cash Flow (FCF) = EBIT × (1 - 세율) + 감가상각비 - CAPEX - 운전자본 증가
```

| **시가총액** | ⚪ | 백만원 | 상장사만 (KRX) |

### WACC 계산 요소

| 항목 | 필수 | 데이터 타입 | 예시 | 출처 |
|------|------|-----------|------|------|
| **무위험수익률 (Rf)** | ✅ | percentage | 3.5% | 국고채 3년물 |
| **베타 (β)** | ⚪ | float | 1.2 | Bloomberg/자체 계산 |
| **시장 위험 프리미엄 (MRP)** | ✅ | percentage | 6.0% | 역사적 평균 |
| **부채비용 (Rd)** | ⚪ | percentage | 3.0% | 재무제표 주석 |
| **법인세율** | ✅ | percentage | 22% | 손익계산서 |

**계산 로직**:
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - 세율)
Re = Rf + β × MRP  (CAPM)
```

```
- 부채총계
- 자기자본

현금흐름표:
- CAPEX
- 감가상각비
```

### 우선순위 2: 재무제표 주석

```
- 발행주식수
- 유형자산 상세
- 우발부채
- 특수관계자 거래
```

### 우선순위 3: 사업보고서

```
- 회사 개요
- 업종 정보
- 사업 내용
```

## 데이터 검증 체크리스트

### 필수 검증 항목

- [ ] 발행주식수 확인
- [ ] 재무제표 합계 일치 (자산 = 부채 + 자본)
- [ ] 연도별 데이터 연속성
- [ ] 음수값 이상 확인
- [ ] 단위 일치 (백만원 vs 억원 vs 원)

### 논리적 검증

- [ ] ROE = 순이익 / 자기자본 (합리적 범위?)
- [ ] PER = 시가총액 / 순이익 (양수?)
- [ ] 성장률 (급격한 변화 여부)
- [ ] WACC (3~15% 범위?)
