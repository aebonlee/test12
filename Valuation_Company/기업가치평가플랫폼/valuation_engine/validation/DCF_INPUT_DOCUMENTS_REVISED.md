# DCF 평가 입력 문서 목록 (실무 기준)

**작성일**: 2025-10-17
**원칙**: 입력이 아닌 **문서 업로드**로 최대한 자동화

---

## 핵심 원칙

```
❌ 잘못된 접근: 회사에게 "입력하세요"
✅ 올바른 접근: 회사에게 "문서를 업로드하세요"

→ AI가 문서에서 자동 추출
→ 회사가 추출된 데이터만 검증
```

## 3. AI 자동 처리 데이터 (외부 수집)

### 3.1 거시경제 지표 (API 자동 수집)

| 항목 | 데이터 소스 | API | 자동 업데이트 |
|------|-------------|-----|---------------|
| **무위험이자율 (Rf)** | 한국은행 | 경제통계시스템 API | 일간 |
| **시장위험프리미엄 (MRP)** | 역사적 데이터 | 자체 DB | 분기 |
| **법인세율** | 국세청 | 고정값 (22%) | 연간 |
| **GDP 성장률** | 한국은행 | 경제통계시스템 API | 분기 |

**API 호출 예시**:
```python
# 한국은행 OpenAPI
def get_risk_free_rate():
    """10년 국고채 수익률 자동 수집"""
    url = "https://ecos.bok.or.kr/api/StatisticSearch/..."
    params = {
        'auth_key': BOK_API_KEY,
        'stat_code': '817Y002',  # 국고채(10년) 수익률
        'cycle': 'D',
        'start_date': today
    }
    response = requests.get(url, params=params)
    return response.json()['value']  # 예: 4.0%
```

### 3.2 업종 데이터 (웹 스크래핑/DB)

| 항목 | 데이터 소스 | 수집 방법 | 업데이트 주기 |
|------|-------------|-----------|---------------|
| **업종별 베타 (β)** | KPMG/PwC 밸류에이션 가이드 | PDF 파싱 or DB | 분기 |
| **업종 평균 마진** | 한국은행 기업경영분석 | 웹 스크래핑 | 연간 |
| **업종 성장률** | 산업 리서치 보고서 | 웹 스크래핑 | 분기 |

**업종별 베타 DB 예시**:
```python
INDUSTRY_BETA = {
    '소프트웨어 개발 및 공급업': 1.15,
    '전자상거래업': 1.25,
    '제조업 (전자부품)': 1.10,
    '제조업 (화학)': 0.95,
    '금융업': 1.05,
    # ... 전체 업종 커버
}

# 자동 매핑
company_industry = "소프트웨어 개발 및 공급업"
beta = INDUSTRY_BETA.get(company_industry, 1.0)  # 기본값 1.0
```

## 4. 평가자 (전문가) 입력 항목

### 4.1 실제로 평가자가 판단해야 할 것 (최소화)

| 항목 | 설명 | 입력 방법 | 기본값 제공 |
|------|------|-----------|-------------|

## 5. 문서 업로드 워크플로우

### 5.1 단계별 프로세스

```
Step 1: 회사 문서 준비 안내
├─ 📋 체크리스트 제공
├─ 📄 사업계획서 템플릿 다운로드
└─ ⚠️  필수 항목 안내

Step 2: 문서 업로드 (회사)
├─ 📤 사업자등록증 업로드
├─ 📤 법인등기부등본 업로드
├─ 📤 외부감사보고서 업로드 (3개년)
├─ 📤 차입금 명세서 업로드
├─ 📤 사업계획서 업로드
└─ 📤 기타 문서 업로드 (선택)

Step 3: AI 자동 데이터 추출
├─ 🤖 OCR로 기본 정보 추출
├─ 🤖 재무제표 테이블 인식
├─ 🤖 사업계획서 숫자 추출
├─ 🤖 재무 비율 자동 계산
└─ 🤖 외부 데이터 수집 (금리, 베타)

Step 4: 회사 데이터 검증
├─ ✅ 추출된 데이터 확인
├─ ✏️  오류 수정
└─ ✔️  확정

Step 5: 평가자 최종 검토
├─ 👁️  전체 데이터 리뷰
├─ 🔧 WACC/영구성장률 조정
└─ ✅ 계산 실행

Step 6: DCF 자동 계산
└─ 📊 결과 생성

Step 7: 보고서 생성
└─ 📄 PDF 다운로드
```

def extract_business_registration(pdf_file):
    """사업자등록증 OCR 추출"""

    # 정규표현식 패턴 매칭
    patterns = {
        'company_name': r'상호\s*:\s*(.+)',
        'business_number': r'(\d{3}-\d{2}-\d{5})',

def extract_business_plan(pdf_file):
    """사업계획서 재무 테이블 추출"""
    # PDF에서 테이블 인식
    tables = camelot.read_pdf(pdf_file, pages='all')

    # "매출" 또는 "Revenue" 키워드가 있는 테이블 찾기
    financial_table = None
    for table in tables:
        df = table.df
        if any('매출' in str(cell) or 'Revenue' in str(cell)
               for cell in df.values.flatten()):
            financial_table = df
            break

    # 연도별 데이터 추출
    revenue_forecast = []
    margin_forecast = []

    for col in financial_table.columns[1:]:  # 첫 컬럼은 라벨
        revenue = extract_number(financial_table[col][매출행])
        margin = extract_number(financial_table[col][마진행])
        revenue_forecast.append(revenue)
        margin_forecast.append(margin)

    return {
        'revenue_forecast': revenue_forecast,
        'margin_forecast': margin_forecast,
        'growth_rate': calculate_growth_rates(revenue_forecast)
    }
