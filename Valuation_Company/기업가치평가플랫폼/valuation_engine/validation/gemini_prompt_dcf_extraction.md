**DCF(Discounted Cash Flow) 방법론 설명:**

DCF는 기업의 미래 현금흐름을 예측하고, 이를 적절한 할인율로 할인하여 현재가치를 구하는 평가방법입니다.

주요 계산 흐름:
1. **역사적 재무데이터 분석** → 과거 3-5년 실적
2. **미래 현금흐름 예측** (5-10년):
   - 매출액 예측 (성장률 적용)
   - 영업이익 = 매출액 × 영업이익률
   - NOPLAT = EBIT × (1 - 세율) = 세후영업이익
   - FCF (잉여현금흐름) = NOPLAT + 감가상각비 - CAPEX - 운전자본증감
3. **영구가치 계산** (Terminal Value):
   - TV = 최종년도 FCF × (1 + 영구성장률) / (WACC - 영구성장률)
4. **할인율(WACC) 계산**:
   - Re (자기자본비용) = Rf + β × MRP
   - Rd (타인자본비용, 세후) = 차입금리 × (1 - 세율)
   - WACC = Re × E/(E+D) + Rd × D/(E+D)
5. **현재가치 계산**:
   - 각 연도 FCF를 WACC로 할인
   - 기업가치 = Σ(FCF 현재가치) + TV 현재가치
6. **주주가치 계산**:
   - 주주가치 = 기업가치 - 순차입금 + 비영업자산
   - 주당가치 = 주주가치 / 발행주식수

**중요 지침:**
1. 모든 숫자는 보고서에 명시된 **실제 값**을 그대로 추출하세요
2. 단위(백만원, 억원 등)를 확인하고 일관되게 변환하세요
3. 값이 없거나 찾을 수 없는 항목은 `null`로 표시하세요
4. 퍼센트는 소수점으로 변환하세요 (예: 12% → 0.12)
5. 보고서에 나온 계산식이나 가정도 함께 추출하세요
6. 표나 그래프에 있는 숫자도 모두 추출하세요
7. 민감도 분석 표가 있다면 그 범위 값들도 추출하세요

추출이 완료되면, **검증을 위해** 다음을 확인해주세요:
- FCF 계산이 맞는지 (NOPLAT + 감가상각비 - CAPEX - 운전자본증감)
- 기업가치 계산이 맞는지 (예측기간 FCF 현재가치 합 + 계속가치 현재가치)
- 주주가치 계산이 맞는지 (기업가치 - 순차입금 + 비영업자산)

## 사용 방법

1. Gemini에 위 프롬프트를 복사
2. DCF 보고서 PDF 파일 첨부
3. 응답받은 JSON을 파일로 저장: `샘플명_dcf_input.json`
4. DCF 엔진에 입력하여 검증
