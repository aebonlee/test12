## 8. 자기주식 보유법인 특례 (조심2023서0142)

### 80% 하한선 규정

상증세법 시행령 제54조 단서:
```
가중평균액 < 순자산가치 × 80% 인 경우
→ 순자산가치의 80%를 평가액으로 함
```

### 적용 예시

```
[사례]
순손익가치: 30,000원
순자산가치: 100,000원

가중평균액 = (30,000 × 3 + 100,000 × 2) / 5
           = (90,000 + 200,000) / 5
           = 58,000원

80% 하한선 = 100,000 × 0.8 = 80,000원

→ 58,000원 < 80,000원 이므로
→ 최종 평가액 = 80,000원 (하한선 적용)
```

**본 사례 검증**:
```
가중평균액 = 110,000원
80% 하한선 = 50,000 × 0.8 = 40,000원

→ 110,000원 > 40,000원 이므로
→ 하한선 미적용, 가중평균액 그대로 사용 ✓
```
