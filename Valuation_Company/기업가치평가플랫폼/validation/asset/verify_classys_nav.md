#### 1. 공정가치 조정 로직 강화
**문제**: 엔진이 자산을 0으로 계산하는 버그 발견
**해결**:
```python
# 현재 (문제)
total_assets_fv = 0  # 자산이 0으로 계산됨

# 개선 후
def _value_assets(self, balance_sheet, fair_value_data):
    # 각 자산 항목별 공정가치 평가
    current_assets_fv = balance_sheet.get('current_assets', 0)
    tangible_assets_fv = self._value_fixed_assets(balance_sheet, fair_value_data)
    intangible_assets_fv = self._value_intangible_assets(balance_sheet, fair_value_data)
    investments_fv = self._value_investments(balance_sheet, fair_value_data)

    return {
        'current_assets': current_assets_fv,
        'tangible_assets': tangible_assets_fv,
        'intangible_assets': intangible_assets_fv,
        'investments': investments_fv,
        'total': sum([current_assets_fv, tangible_assets_fv, intangible_assets_fv, investments_fv])
    }
```

#### 3. 비상장 할인율 자동 적용

```python
def apply_marketability_discount(self, nav, is_listed, company_size):
    if is_listed:
        return nav  # 할인 없음

    # 기업 규모별 할인율
    if company_size == 'large':  # 자산 1조 이상
        discount = 0.10
    elif company_size == 'medium':  # 1천억~1조
        discount = 0.20
    else:  # 1천억 미만
        discount = 0.30

    return nav * (1 - discount)
```
