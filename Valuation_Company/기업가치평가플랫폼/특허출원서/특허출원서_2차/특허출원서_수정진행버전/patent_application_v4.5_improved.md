【해결하고자 하는 과제】

본 발명은 상기한 종래 기술의 문제점을 해결하기 위한 것으로, 기업가치평가 정보의 활용도가 낮고 투자자와 기업 간 매칭 효율성이 부족한 문제를 4+2 구조의 혁신적 통합과 동적 가중치 학습을 통해 해결하고자 한다.

구체적으로는 첫째, 4개 핵심 시스템의 순차적 캐스케이드 활성화를 통해 시너지 효과를 달성하고, 둘째, 실제 투자 성사 데이터를 학습하여 매칭 알고리즘을 지속적으로 개선하며, 셋째, 2개 지원 시스템의 선택적 통합을 통해 핵심 시스템의 효과를 증폭시키고, 넷째, 동적 가중치 및 예측 분석을 통한 지능형 자동화를 구현하며, 다섯째, 지속가능한 포인트 경제 순환을 통한 완전한 투자 생태계를 구축하는 것을 목적으로 한다.

【과제의 해결 수단】

상기한 과제를 해결하기 위하여, 본 발명에 따른 기업가치평가 정보 기반의 AI를 활용한 상시 IR 및 투자자 매칭 시스템은 4개 핵심 시스템(평가정보 등록 시스템, 랭킹 시스템, AI 아바타 상시 IR 시스템, AI 투자자 매칭 시스템)이 순차적 캐스케이드 활성화를 통해 연동되고, 2개 지원 시스템(관심도 리포트 시스템, 포인트 순환경제 시스템)이 선택적으로 통합되어 전체 시스템의 효과를 증폭하는 4+2 구조를 형성한다.

특히, 본 발명의 핵심 혁신은 실제 투자 성사 이력을 학습하여 가중치를 자동으로 조정하는 동적 학습 메커니즘이다. 이는 다음 수식으로 구현된다:

W(t+1) = W(t) + η × ∇Performance(t)

여기서 W(t)는 시점 t의 가중치, η는 학습률, ∇Performance는 성과 개선 기울기이다.

【실시예 1: 4+2 구조 통합 시스템】

제1도를 참조하면, 본 발명의 일 실시예에 따른 기업가치평가 정보 기반의 AI를 활용한 상시 IR 및 투자자 매칭 시스템(100)은 4개 핵심 시스템과 2개 지원 시스템이 통합된 4+2 구조로 구성된다.

4개 핵심 시스템은:
- 평가정보 등록 시스템(110): 기업가치평가 정보의 표준화 및 품질 보증

- AI 투자자 매칭 시스템(140): 지능형 자동 매칭

2개 지원 시스템은:
- 관심도 리포트 시스템(150): 투자자 행동 분석 및 인사이트

【실시예 2: 동적 가중치 학습 메커니즘】

본 발명의 핵심 혁신인 동적 가중치 학습은 다음과 같이 구현된다:

1. **데이터 수집**: 과거 30일간의 투자 성사/실패 데이터 수집
2. **특징 추출**: 각 케이스의 특징 벡터 [섹터, 규모, 단계, 리스크] 추출
3. **성과 측정**: 예측 매칭 점수 vs 실제 투자 성사 여부 비교
4. **가중치 업데이트**: 
   ```
   W(t+1) = W(t) + η × (실제결과 - 예측결과) × 특징벡터
   ```
5. **수렴 확인**: 가중치 변화량이 임계값 이하로 감소 시 안정화

이러한 학습 과정을 통해 시스템은 지속적으로 개선된다.

【실시예 3: 캐스케이드 활성화 프로세스】

**Phase 1: 평가정보 등록 (기준점 설정)**

- 다중 검증 알고리즘으로 유효성 확인
- 표준화 처리로 데이터 품질 확보

**Phase 2: 랭킹 생성 (첫 번째 시너지)**
- 표준화된 데이터로 정확한 평가 가능
- 동적 가중치 적용으로 실시간 시장 반영
- 카테고리별 객관적 순위 생성

**Phase 3: AI 아바타 활성화 (두 번째 시너지)**
- 높은 품질의 평가정보와 랭킹 데이터로 학습
- 기업별 맞춤형 AI 모델 생성
- 24시간 투자자 질의응답 서비스 시작

**Phase 4: 투자자 매칭 (세 번째 시너지)**
- 풍부한 데이터 기반 정밀 매칭

【실시예 6: 구현 상세】

동적 가중치 학습 알고리즘의 구체적 구현:

```python
class DynamicWeightLearning:
    def __init__(self):
        self.weights = {
            'sector': 0.25,
            'size': 0.25,
            'stage': 0.25,
            'risk': 0.25
        }
        self.learning_rate = 0.01
        self.history = []
    
    def update_weights(self, success_data):
        # 성과 기반 가중치 조정
        for factor in self.weights:
            gradient = self.calculate_gradient(factor, success_data)
            self.weights[factor] += self.learning_rate * gradient
            
        # 정규화 (합이 1이 되도록)
        total = sum(self.weights.values())
        self.weights = {k: v/total for k, v in self.weights.items()}
        
        # 학습률 감소 (안정화)
        self.learning_rate *= 0.95
        
        return self.weights
```
