## 제1부: 가치평가의 기초와 프레임워크

### 1. 가치평가 개요

#### 1.1 가치평가의 정의와 목적

**가치평가(Valuation)의 정의**
- 기업, 자산, 또는 증권의 경제적 가치를 체계적으로 추정하는 과정
- 미래 경제적 효익을 현재가치로 환산하는 분석 기법
- 객관적 데이터와 주관적 판단이 결합된 종합적 평가 프로세스

**가치평가의 목적**

| 구분 | 주요 목적 | 활용 분야 |
|------|----------|-----------|
| **거래 목적** | M&A, 지분투자, 매각 | 인수합병, PE/VC 투자 |
| **재무보고 목적** | 손상평가, 공정가치 측정 | K-IFRS 적용, 회계감사 |
| **세무 목적** | 증여세, 상속세 평가 | 세무신고, 세무조사 대응 |
| **법적 목적** | 주식매수청구권, 소송 | 법적 분쟁, 손해배상 |
| **내부관리 목적** | 성과평가, 전략수립 | 경영계획, 투자의사결정 |

#### 1.2 가치의 개념과 종류

**가치의 유형별 특성**

```
┌─────────────────────────────────────────────────┐
│                   가치의 스펙트럼                    │
├─────────────────────────────────────────────────┤
│  장부가치 → 청산가치 → 시장가치 → 투자가치 → 시너지가치  │
│                                                     │
│  객관성 ◀────────────────────────────▶ 주관성       │
│  과거지향 ◀──────────────────────────▶ 미래지향     │
└─────────────────────────────────────────────────┘
```

**주요 가치 개념**

1. **공정가치(Fair Value)**
   - 정의: 측정일에 시장참여자 사이의 정상거래에서 자산 매도 또는 부채 이전 시 수취/지급 가격
   - 특징: 시장 기반 측정, 출구가격 개념
   - 적용: K-IFRS 재무보고

2. **시장가치(Market Value)**
   - 정의: 충분한 정보를 가진 자발적 거래당사자 간 거래 가능 가격
   - 특징: 객관적, 제3자 관점
   - 적용: 상장주식 평가, 부동산 감정

3. **투자가치(Investment Value)**
   - 정의: 특정 투자자에게 특별한 가치
   - 특징: 주관적, 투자자별 차별화
   - 적용: M&A, 전략적 투자

4. **내재가치(Intrinsic Value)**
   - 정의: 자산의 본질적 가치
   - 특징: 펀더멘털 분석 기반
   - 적용: 가치투자, 주식분석

### 2. 가치평가 방법론 체계

#### 2.1 가치평가 접근법 분류

```
                    가치평가 방법론
                          │
        ┌─────────────────┼─────────────────┐
        │                 │                 │
   수익접근법          시장접근법        원가접근법
        │                 │                 │
   ┌────┴────┐      ┌────┴────┐      ┌────┴────┐
   │         │      │         │      │         │
  DCF     EVA    상대가치  거래사례  순자산  대체원가
   │              │         │      │         │
┌──┼──┐      ┌──┼──┐      │      │         │
│  │  │      │  │  │      │      │         │
FCFF FCFE DDM PER PBR EV/EBITDA  거래배수  청산가치
```

#### 2.2 수익접근법 (Income Approach)

**DCF(Discounted Cash Flow) 모형의 구조**

```
기업가치 = Σ(FCFt / (1+WACC)^t) + TV / (1+WACC)^n

여기서:
- FCF: 잉여현금흐름
- WACC: 가중평균자본비용
- TV: 영구가치(Terminal Value)
- n: 추정기간
```

**DCF 모형의 변형**

| 모형 | 현금흐름 | 할인율 | 산출 가치 | 적용 상황 |
|------|----------|--------|-----------|-----------|
| FCFF DCF | 기업 잉여현금흐름 | WACC | 기업가치 | 일반 기업 |
| FCFE DCF | 주주 잉여현금흐름 | 자기자본비용 | 자기자본가치 | 금융기관 |
| DDM | 배당금 | 자기자본비용 | 주식가치 | 성숙 기업 |
| APV | 무부채 현금흐름 | 무부채 자본비용 | 조정기업가치 | LBO, 구조조정 |

#### 2.3 시장접근법 (Market Approach)

**주요 가치평가 배수**

```
┌──────────────────────────────────────────────────────┐
│                    가치평가 배수 체계                    │
├──────────────────────────────────────────────────────┤
│  자기자본 배수                기업가치 배수              │
│  ─────────────              ─────────────            │
│  • PER (P/E)                 • EV/EBITDA               │
│  • PBR (P/B)                 • EV/EBIT                 │
│  • PSR (P/S)                 • EV/Sales                │
│  • PEG                       • EV/FCF                  │
└──────────────────────────────────────────────────────┘
```

**배수 선택 기준**

1. **산업 특성**
   - 제조업: EV/EBITDA, PER
   - IT/바이오: PSR, EV/Sales
   - 금융업: PBR, ROE
   - 부동산: NAV, P/FFO

2. **기업 생명주기**
   - 스타트업: 사용자 기반 지표
   - 성장기: PSR, PEG
   - 성숙기: PER, EV/EBITDA
   - 쇠퇴기: PBR, 청산가치

#### 2.4 원가접근법 (Cost Approach)

**순자산가치 평가 프레임워크**

```
순자산가치(NAV) = 자산의 시장가치 - 부채의 시장가치

조정 항목:
+ 무형자산 재평가
+ 우발자산 인식
- 우발부채 반영
± 시장가치 조정
= 조정 순자산가치
```

## 제2부: 수익접근법 - DCF 평가모형

### 1. DCF 모형의 이론적 기초

#### 1.1 현재가치 이론

**화폐의 시간가치**

```
PV = FV / (1 + r)^n

복리 현재가치: PV = Σ(CFt / (1+r)^t)

여기서:
- PV: 현재가치
- FV: 미래가치
- CF: 현금흐름
- r: 할인율
- t: 기간
```

**DCF 가치평가의 핵심 원리**

1. **현금흐름 중심**: 회계이익이 아닌 현금흐름 기준
2. **미래지향적**: 과거가 아닌 미래 현금창출능력 평가
3. **위험 반영**: 할인율을 통한 리스크 조정
4. **시간가치 반영**: 현금흐름 발생 시점 고려

#### 1.2 DCF 모형의 구성요소

```
            기업가치(EV)
                 │
    ┌────────────┴────────────┐
    │                         │
현재가치(PV)              영구가치(TV)
    │                         │
    ├── FCF₁/(1+r)           │
    ├── FCF₂/(1+r)²          │
    ├── FCF₃/(1+r)³          │
    └── ...FCFₙ/(1+r)ⁿ       │
                              │
                    TVₙ/(1+r)ⁿ
```

### 2. 현금흐름 추정

#### 2.1 FCFF vs FCFE

**잉여현금흐름 산출 체계**

```
┌───────────────────────────────────────────────────────┐
│                    FCFF 산출 과정                      │
├───────────────────────────────────────────────────────┤
│  EBIT(1-T)                                            │
│  + 감가상각비                                          │
│  - 자본적 지출(CapEx)                                  │
│  - 순운전자본 증가                                     │
│  = FCFF (기업 잉여현금흐름)                            │
├───────────────────────────────────────────────────────┤
│                    FCFE 산출 과정                      │
├───────────────────────────────────────────────────────┤
│  당기순이익                                            │
│  + 감가상각비                                          │
│  - 자본적 지출(CapEx)                                  │
│  - 순운전자본 증가                                     │
│  + 신규 차입                                           │
│  - 차입금 상환                                         │
│  = FCFE (주주 잉여현금흐름)                            │
└───────────────────────────────────────────────────────┘
```

#### 2.2 매출액 추정

**매출 예측 방법론**

| 접근법 | 방법 | 적용 상황 | 장단점 |
|--------|------|-----------|--------|
| **Top-down** | 시장규모 × 시장점유율 | 신규 시장 진입 | 객관적, 거시적 |
| **Bottom-up** | 단가 × 수량 | 기존 사업 확장 | 구체적, 미시적 |
| **시계열 분석** | 과거 추세 연장 | 안정적 산업 | 단순, 보수적 |
| **회귀 분석** | 동인 기반 예측 | 상관관계 명확 | 통계적, 복잡 |

**성장률 시나리오 설정**

```
성장률 패턴:
      높음 ──────┐
       │         │
       │         ├──── 안정기
  초기 │         │
       │    ┌────┘
       │    │
      낮음 ──┴───────────────────
          시간 →
```

#### 2.3 영업이익률 추정

**원가 구조 분석**

```
매출액                     100%
├─ 매출원가                (60%)
│   ├─ 변동비              (40%)
│   └─ 고정비              (20%)
├─ 판관비                  (25%)
│   ├─ 인건비              (15%)
│   ├─ 마케팅비            (5%)
│   └─ 기타                (5%)
└─ 영업이익                 15%
```

#### 2.4 투자 추정

**운전자본 추정**

```
순운전자본 = 영업용 유동자산 - 영업용 유동부채
         = (매출채권 + 재고자산) - 매입채무

순운전자본 회전일수:
- 매출채권 회전일 = 매출채권/일평균매출
- 재고자산 회전일 = 재고자산/일평균매출원가
- 매입채무 회전일 = 매입채무/일평균매출원가

CCC = 매출채권회전일 + 재고회전일 - 매입채무회전일
```

**자본적 지출(CapEx) 추정**

| 구분 | 산식 | 설명 |
|------|------|------|
| **유지 CapEx** | 감가상각비 × 조정계수 | 현상 유지 투자 |
| **성장 CapEx** | 매출증가액 × 투자율 | 성장 위한 신규 투자 |
| **총 CapEx** | 유지 + 성장 CapEx | 총 자본적 지출 |

### 3. 할인율 산정 - WACC

#### 3.1 WACC 구조

**가중평균자본비용 공식**

```
WACC = E/(E+D) × Re + D/(E+D) × Rd × (1-T)

여기서:
- E: 자기자본의 시장가치
- D: 부채의 시장가치
- Re: 자기자본비용
- Rd: 타인자본비용
- T: 법인세율
```

#### 3.2 자기자본비용 산정 - CAPM

**CAPM 모형**

```
Re = Rf + β × (Rm - Rf)

구성요소:
┌────────────────────────────────────┐
│  무위험수익률(Rf)                   │
│  + 체계적 위험 프리미엄              │
│    = β × 시장위험프리미엄(Rm-Rf)    │
│  + 추가 위험 프리미엄(선택적)        │
│    - 규모 프리미엄                  │
│    - 비유동성 프리미엄              │
│  = 자기자본비용(Re)                 │
└────────────────────────────────────┘
```

#### 3.3 베타(β) 추정

**베타 추정 방법**

1. **과거 베타 (Historical Beta)**
   ```
   β = Cov(Ri, Rm) / Var(Rm)
   
   회귀분석: Ri = α + β×Rm + ε
   ```

2. **산업 베타 (Industry Beta)**
   ```
   βL = βU × [1 + (1-T) × D/E]
   
   여기서:
   - βL: 부채 베타(Levered)
   - βU: 무부채 베타(Unlevered)
   ```

**베타 조정 프로세스**

```
상장사 베타 수집
      ↓
언레버링 (부채 효과 제거)
      ↓
산업 평균 베타 산출
      ↓
리레버링 (목표기업 자본구조 반영)
      ↓
조정 베타 산출
```

#### 3.4 시장위험프리미엄

**시장위험프리미엄 추정 방법**

| 방법 | 산식/내용 | 장단점 |
|------|-----------|--------|
| **과거 수익률** | 과거 주식-채권 수익률 차이 | 객관적, 후행적 |
| **서베이** | 투자자 설문조사 | 선행적, 주관적 |
| **내재 ERP** | 현재 주가에서 역산 | 시장 반영, 복잡 |

### 4. 영구가치 산정

#### 4.1 영구가치 산정 방법

**Gordon 성장 모형**

```
TV = FCFn+1 / (WACC - g)

여기서:
- FCFn+1: 영구기간 첫해 현금흐름
- g: 영구성장률
- WACC > g (필수 조건)
```

**Exit Multiple 방법**

```
TV = EBITDAn × Exit Multiple

Exit Multiple 선정:
- 현재 유사기업 배수
- 과거 거래 배수
- 산업 평균 배수
```

#### 4.2 영구성장률 설정

**영구성장률 벤치마크**

```
┌─────────────────────────────────┐
│         0%  ← → GDP 성장률       │
│              ↑                   │
│        보수적 추정               │
│              ↓                   │
│      영구성장률 범위             │
└─────────────────────────────────┘
```

## 제3부: 시장접근법과 원가접근법

### 1. 상대가치 평가법

#### 1.1 비교기업 선정

**비교기업 선정 기준**

```
필수 기준:
□ 동일 산업 분류 (KSIC)
□ 유사 사업 모델
□ 비슷한 규모 (±50%)

추가 고려사항:
□ 성장률 유사성
□ 수익성 유사성
□ 위험 프로파일
□ 지역적 범위
```

**비교가능성 매트릭스**

| 특성 | 높음 | 중간 | 낮음 | 가중치 |
|------|------|------|------|--------|
| 산업 | 동일 세분류 | 동일 중분류 | 동일 대분류 | 30% |

| 성장성 | ±5%p | ±10%p | ±20%p | 20% |

| 지역 | 국내 | 아시아 | 글로벌 | 10% |

#### 1.2 배수 산정 및 적용

**배수 정규화 과정**

```
원시 데이터 수집
      ↓
이상치 제거 (상하위 10%)
      ↓
회계 차이 조정
      ↓
성장률/수익성 조정
      ↓
가중평균 산출
      ↓
최종 배수 결정
```

**PER 조정 예시**

```
기본 PER = 주가 / EPS

조정 PER = 기본 PER × (1+g_target)/(1+g_comp) × ROE_target/ROE_comp

여기서:
- g: 성장률
- ROE: 자기자본수익률
```

#### 1.3 EV/EBITDA 배수

**EV/EBITDA 활용**

```
기업가치(EV) 산정:
EV = EBITDA × EV/EBITDA Multiple

자기자본가치 도출:
자기자본가치 = EV - 순부채 + 비영업자산

순부채 = 차입금 + 리스부채 - 현금성자산
```

**산업별 EV/EBITDA 범위**

| 산업 | 범위 | 특징 |
|------|------|------|
| 제조업 | 5-8x | 자본집약적, 안정적 |

| IT/소프트웨어 | 12-20x | 높은 성장, 확장성 |
| 바이오/헬스케어 | 15-25x | 높은 진입장벽 |
| 유틸리티 | 6-9x | 안정적, 규제산업 |

## 제4부: M&A와 특수 상황 가치평가

### 1. M&A 가치평가

#### 1.1 M&A 가치평가 프레임워크

**M&A 가치 구성**

```
┌────────────────────────────────────────┐
│         M&A 거래가격 구성              │
├────────────────────────────────────────┤
│  독립기업가치 (Stand-alone Value)      │
│  + 시너지 가치 (Synergy Value)         │
│  + 지배권 프리미엄 (Control Premium)   │
│  = 인수가격 (Acquisition Price)        │
│                                        │
│  인수가격                              │
│  - 인수자 기여 시너지                  │
│  = 매도자 몫 (Seller's Share)          │
└────────────────────────────────────────┘
```

#### 1.2 시너지 평가

**시너지 유형과 평가**

| 시너지 유형 | 원천 | 평가 방법 | 실현 기간 |
|------------|------|-----------|-----------|
| **매출 시너지** | 교차판매, 시장확대 | 증분 매출 × 마진율 | 2-3년 |
| **비용 시너지** | 규모의 경제, 중복제거 | 비용 절감액 직접 산정 | 1-2년 |
| **세무 시너지** | 이월결손금, 세율 차이 | 세금 절감액 현재가치 | 즉시 |
| **재무 시너지** | 자본비용 감소 | WACC 감소 효과 | 1년 |

**시너지 현재가치 계산**

```
PV(시너지) = Σ[시너지t × (1-세율) / (1+r)^t]

시너지 실현률 반영:
조정 시너지 = 예상 시너지 × 실현 확률 × (1-실행비용률)
```

#### 1.3 지배권 프리미엄

**지배권 가치 요소**

```
지배권 프리미엄 구성:
├─ 경영권 가치
│  ├─ 이사 선임권
│  ├─ 전략 결정권
│  └─ 배당 정책
├─ 유동성 프리미엄
│  └─ 비상장→상장 가능성
└─ 정보 비대칭 해소
   └─ 내부 정보 접근
```

**프리미엄 수준 결정 요인**

| 요인 | 높은 프리미엄 | 낮은 프리미엄 |
|------|--------------|--------------|
| 지분율 | 50% 초과 | 50% 미만 |
| 시너지 | 높음 | 낮음 |
| 경쟁 입찰 | 복수 입찰자 | 단독 협상 |
| 시장 상황 | 호황기 | 불황기 |
| 대상회사 실적 | 성장/흑자 | 정체/적자 |

### 2. 스타트업 가치평가

#### 2.1 벤처캐피털 방법론

**VC Method 공식**

```
현재가치 = 미래가치 / (1 + 요구수익률)^n
        = Exit Value × 지분율 / (1 + IRR)^n

Exit Value = 예상 순이익 × Exit P/E Multiple
           또는
         = 예상 매출 × Exit Revenue Multiple
```

**단계별 가치평가 조정**

```
  Seed → Series A → Series B → Series C → IPO
   │        │          │          │        │
  아이디어  제품출시   시장검증   확장     성숙
   │        │          │          │        │
할인율 50-70% 40-50%   30-40%    20-30%   15-20%
```

#### 2.2 스타트업 특수 지표

**산업별 핵심 지표**

| 산업 | 주요 지표 | 가치평가 배수 |
|------|----------|--------------|
| SaaS | ARR, CAC, LTV | 5-15x ARR |
| 이커머스 | GMV, Take Rate | 0.5-2x GMV |
| 모빌리티 | MAU, 거래량 | $50-500/MAU |
| 핀테크 | TPV, 수수료율 | 2-5x Revenue |
| 게임 | ARPU, Retention | 0.5-1.5x Revenue |

**유닛 이코노믹스 분석**

```
LTV/CAC Ratio = 고객생애가치 / 고객획득비용

건전성 기준:
- LTV/CAC > 3: 우수
- LTV/CAC = 2-3: 양호
- LTV/CAC < 2: 개선 필요

Payback Period = CAC / (ARPU × Gross Margin)
목표: < 12개월
```

### 3. 특수증권 가치평가

#### 3.1 전환사채(CB) 평가

**CB 가치 구성**

```
CB 가치 = 채권 가치 + 전환권 가치

채권 가치 = Σ[쿠폰 / (1+YTM)^t] + 액면가 / (1+YTM)^n

전환권 가치 = Black-Scholes 또는 이항모형
```

**Black-Scholes 모형 적용**

```
Call Value = S×N(d1) - K×e^(-rt)×N(d2)

여기서:
d1 = [ln(S/K) + (r + σ²/2)×t] / (σ×√t)
d2 = d1 - σ×√t

S: 현재 주가
K: 전환가격
r: 무위험이자율
σ: 변동성
t: 만기
N(): 표준정규분포 누적확률
```

#### 3.2 상환전환우선주(RCPS) 평가

**RCPS 가치 요소**

```
RCPS 가치 구성:
├─ 우선주 가치 (배당 우선권)
├─ 전환권 가치 (보통주 전환)
├─ 상환권 가치 (Put Option)
└─ 기타 특약 가치
   ├─ 동반매도권 (Tag-along)
   ├─ 우선매수권 (ROFR)
   └─ 리픽싱 조항
```

**시나리오별 가치 분석**

| 시나리오 | 확률 | 수익 구조 | 현재가치 |
|----------|------|-----------|----------|
| IPO | 30% | 전환 후 매도 | 높음 |
| M&A | 40% | 상환 또는 전환 | 중간 |
| 청산 | 20% | 잔여재산 분배 | 낮음 |
| 계속기업 | 10% | 배당 수취 | 중간 |

### 4. 구조조정 기업 평가

#### 4.1 부실기업 가치평가

**가치평가 접근법 선택**

```
재무 상태에 따른 접근법:
                    
정상 → 위기 → 부실 → 회생/청산
 │      │      │        │
DCF   조정DCF  자산   청산가치
      APV    접근법
```

**기업구조조정 프로세스**
- Workout: 자율협약에 의한 구조조정
- 채권단 자율협약: 채권금융기관 주도
- 기업개선약정: 채권단과 기업 간 협약
- 법정관리/회생절차: 법원 주도

**조정 DCF (Adjusted DCF)**

```
기업가치 = Σ[FCF × 생존확률 / (1+r)^t]

생존확률 산정:
- Altman Z-Score
- 신용등급 기반 부도율
- 산업 평균 생존율
```

#### 4.2 청산가치 평가

**청산가치 할인율**

| 자산 유형 | 정상 처분 | 강제 청산 |
|----------|-----------|-----------|
| 현금성 자산 | 100% | 100% |
| 매출채권 | 85-95% | 50-70% |
| 재고자산 | 50-80% | 20-40% |
| 토지/건물 | 80-90% | 60-70% |
| 기계장치 | 40-60% | 10-30% |
| 무형자산 | 0-50% | 0% |

## 제5부: 실무 적용과 사례 연구 및 특수 평가

### 1. 산업별 특성과 평가

#### 1.1 제조업 가치평가

**제조업 특성 반영**

```
핵심 가치동인:
├─ 생산능력 (Capacity)
├─ 가동률 (Utilization)
├─ 원가 경쟁력
├─ 기술력/품질
└─ 고객 기반

주요 평가지표:
- EV/EBITDA: 5-8x
- PER: 8-15x
- PBR: 0.8-1.5x
```

**설비투자 주기 반영**

```
투자 주기:
     ↑ CapEx
     │     ╱╲
     │    ╱  ╲
     │   ╱    ╲
     │  ╱      ╲_____ 
     └─────────────────→ 시간
       3-5년 주기

DCF 반영: 주기적 대규모 CapEx 별도 추정
```

#### 1.2 IT/플랫폼 기업 평가

**네트워크 효과 반영**

```
사용자 가치 = k × N^α

여기서:
- N: 사용자 수
- α: 네트워크 계수 (1.5-2.0)
- k: 상수

메칼프의 법칙: 가치 ∝ N²
```

**주요 KPI와 가치평가**

| KPI | 정의 | 활용 |
|-----|------|------|
| MAU | 월 활성 사용자 | 사용자당 가치 |
| ARPU | 사용자당 평균 수익 | 수익성 지표 |
| Churn Rate | 이탈률 | LTV 계산 |
| CAC | 고객 획득 비용 | 효율성 지표 |
| GMV | 총 거래액 | 플랫폼 규모 |

#### 1.3 바이오/제약 평가

**개발 단계별 가치**

```
가치 곡선:
     가치
      ↑     승인
      │      ╱
      │     ╱ Phase 3
      │    ╱
      │   ╱ Phase 2
      │  ╱
      │ ╱ Phase 1
      │╱ 전임상
     └──────────────→ 개발단계
     
리스크 조정: rNPV = Σ[CFt × POSt / (1+r)^t]
POS: Probability of Success
```

**파이프라인 가치평가**

| 단계 | 성공 확률 | 기간 | 비용 |
|------|----------|------|------|
| 전임상 | 10% | 3년 | $5M |
| Phase 1 | 20% | 1년 | $15M |
| Phase 2 | 30% | 2년 | $50M |
| Phase 3 | 60% | 3년 | $150M |
| 승인/출시 | 90% | 1년 | $50M |

### 2. 종합 사례 연구

#### 2.1 사례: 전통 제조업 M&A

**기업 개요**
- 산업: 자동차 부품
- 매출: 5,000억원
- EBITDA: 500억원
- 순부채: 1,000억원

**가치평가 적용**

```
1. DCF 평가
   - FCFF 5년 추정
   - WACC: 8.5%
   - 영구성장률: 2%
   - 기업가치: 6,200억원

2. 상대가치 평가
   - EV/EBITDA: 7x
   - 기업가치: 3,500억원

3. 가중평균
   - DCF 60% + 상대 40%
   - 최종 EV: 5,120억원
   - 자기자본가치: 4,120억원
```

#### 2.2 사례: 스타트업 Series B

**기업 개요**
- 산업: SaaS (B2B)
- ARR: $10M
- 성장률: 150% YoY
- Burn Rate: $500K/월

**가치평가 과정**

```
VC Method:
- Exit 시점: 5년 후
- Exit ARR: $150M
- Exit Multiple: 8x
- Exit Value: $1.2B
- 요구 지분율: 20%
- 요구 수익률: 40%

Post-money Valuation:
= $1.2B × 20% / (1.4)^5
= $240M / 5.38
= $45M

Pre-money: $35M (투자금 $10M)
```

### 3. 메자닌 금융과 특수증권

#### 3.1 메자닌(Mezzanine) 금융 구조

**메자닌 금융의 정의**
- 채권과 주식의 중간 형태 금융상품
- 투자자: 높은 수익률 + 상승 참여 기회
- 발행자: 유연한 자본조달 + 낮은 비용

**주요 메자닌 상품**

| 상품 | 특징 | 평가 방법 |
|------|------|----------|
| 전환사채(CB) | 채권+주식전환권 | 채권가치+옵션가치 |
| 신주인수권부사채(BW) | 채권+신주인수권 | 채권가치+워런트가치 |
| 교환사채(EB) | 타사 주식 교환권 | 채권가치+교환권가치 |
| RCPS | 우선주+전환+상환 | 시나리오별 가치 |

#### 3.2 전환사채(CB) 상세 평가

**실제 CB 발행 조건 예시**
- 총 투자금액: 200억원
- 사채 만기: 4년
- Coupon: 0.0%, YTM: 0.0%
- Put Option: 발행 2년 후 매 3개월 (YTP 0.0%)
- 전환가능기간: 발행 1년 후 ~ 만기 1개월 전
- 전환가격: 발행 시 주가의 130%
- Refixing: 주가 하락 시 85%까지 조정

**CB 가치평가 모형**

```
CB 가치 = 채권가치 + 전환권가치

채권가치 = Σ[쿠폰/(1+YTM)^t] + 액면가/(1+YTM)^n

전환권가치 = Black-Scholes 또는 이항모형
```

#### 3.3 상환전환우선주(RCPS) 평가

**RCPS 발행 단계별 투자자**
```
Start-up → VC → Growth Capital → PEF
  ↓        ↓         ↓              ↓
고위험   중위험    중저위험      저위험
```

**RCPS 시나리오별 가치 분석**

| 시나리오 | 확률 | 수익 구조 | 현재가치 |
|----------|------|-----------|----------|
| IPO | 30% | 전환 후 매도 | 높음 |
| M&A | 40% | 상환 또는 전환 | 중간 |
| 청산 | 20% | 잔여재산 분배 | 낮음 |
| 계속기업 | 10% | 배당 수취 | 중간 |

**RCPS 특약 조항**
- 동반매도권 (Tag-along)
- 우선매수권 (ROFR)
- 리픽싱 조항
- 청산 시 우선분배권

### G. IPO 공모가 산정과 Book Building

**수요예측(Book Building) 프로세스**

```
실사/평가 → 기준가 산정 → 희망공모가밴드 → 수요예측 → 최종공모가
     ↓           ↓              ↓            ↓           ↓
  기업분석    DCF/상대가치    ±20% 범위    기관투자자    확정가격
```

**2024년 IPO 제도 개선사항**

1. **수요예측 초일 가점제 개편**
   - 기존: 1일차 3점, 2일차 2점, 3-5일차 1점
   - 개선: 1-3일차 1.5점, 4-5일차 1점
   - 목적: 초일 쏠림 현상 완화

2. **기관투자자 의무보유 강화**
   - 공모주 lock-up 기간 연장
   - 단기차익 실현 제한
   - 중장기 투자 유도

3. **공모가 산정 표준화**
   - 금융투자협회 가이드라인
   - 증권사별 내부 표준안 수립
   - 적정 공모가 산정 강화

**IPO Underpricing 현상**

한국 IPO 시장의 구조적 문제:
- 평균 30-40% 초기 수익률
- 정보 비대칭으로 인한 할인
- 기관투자자 우대 배정
- 일반투자자 불리한 구조

**코너스톤 투자자 제도 (도입 예정)**
- 증권신고서 제출 전 사전 배정
- 6-12개월 보호예수 조건
- 공모가 안정화 효과
- 해외 선진시장 사례 벤치마킹
