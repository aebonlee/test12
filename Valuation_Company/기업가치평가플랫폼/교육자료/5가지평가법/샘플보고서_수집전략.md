### 1. DCF 평가법 샘플

#### 출처 1: DART (전자공시시스템) - 합병/분할 공시
**URL:** https://dart.fss.or.kr

**검색 방법:**
```
1. DART 접속
2. 공시검색 → 주요사항보고서
3. 보고서명: "합병" or "분할" or "주식매수선택권"
4. 첨부파일 중 "평가보고서" 다운로드
```

**키워드:**
- "합병 평가보고서"
- "분할 평가보고서"
- "주식매수선택권 평가보고서"
- "자기주식 취득 평가"

**예상 발견 위치:**
- 합병 공시 → 첨부파일: 합병가액 산정 평가보고서
- 분할 공시 → 첨부파일: 분할가액 산정 평가보고서
- 주식매수선택권 부여 → 첨부파일: 공정가치 평가보고서

**구체적 검색 단계:**
```
1. DART 메인 → [공시검색]
2. [정기공시] 체크 해제
3. [주요사항보고서] 체크
4. 보고서명 입력: "합병"
5. 기간: 최근 1년
6. 검색
7. 각 공시 클릭 → 첨부파일 확인 → "평가보고서" 다운로드
```

**실제 사례 (예시):**
- (주)A + (주)B 합병 (2024.03.15)
  → 첨부: "합병가액 산정을 위한 기업가치평가 보고서.pdf"
  → DCF + 상대가치평가 병행

### Phase 2: 수집 후 분류

```python
import os
import PyPDF2

def classify_report(pdf_path):
    """
    PDF 내용 분석하여 평가법 자동 분류
    """
    with open(pdf_path, 'rb') as file:
        reader = PyPDF2.PdfReader(file)
        text = ""
        for page in reader.pages:
            text += page.extract_text()

    # 키워드 기반 분류
    if "DCF" in text or "현금흐름할인" in text:
        return "DCF"
    elif "PER" in text or "PBR" in text or "유사기업" in text:
        return "상대가치"
    elif "순자산가치" in text or "NAV" in text:
        return "자산가치"
    elif "자본시장법" in text or "시행령 제176조" in text:
        return "자본시장법"
    elif "상속세" in text or "증여세" in text or "시행령 제54조" in text:
        return "상증세법"
    else:
        return "미분류"

# 다운로드한 파일 분류
reports_dir = "./downloaded_reports"
for filename in os.listdir(reports_dir):
    if filename.endswith(".pdf"):
        category = classify_report(os.path.join(reports_dir, filename))
        # 카테고리별 폴더로 이동
        os.rename(
            os.path.join(reports_dir, filename),
            os.path.join(reports_dir, category, filename)
        )
```

## 샘플 활용 계획

### 1. 보고서 구조 분석
```
각 샘플에서 추출:
- 목차 구조 (몇 장, 몇 페이지)
- 챕터별 내용 (회사 개요, 재무 분석, 평가 결과 등)
- 표/차트 종류 (재무제표, 배수 비교표, FCF 차트 등)
- 주석 및 가정 사항
```

### 2. 계산 로직 검증
```
샘플 데이터 → 우리 엔진 → 결과 비교
- 오차 범위: ±5% 이내 목표
- 오차 원인 분석
- 알고리즘 개선
```

### 3. 템플릿 설계
```
샘플 3개 이상 → 공통 구조 추출 → 템플릿 설계
- 표준 목차
- 표준 표 양식
- 표준 차트 형식
```

### 4. Claude 프롬프트 학습
```
샘플 보고서 → Claude API에 예시로 제공
"다음과 같은 형식으로 작성하세요: [샘플 첨부]"
```

## 예상 소요 시간

**수동 수집:**
- DART 검색 및 다운로드: 2일
- 법원/국세청 사례 수집: 1일
- 분류 및 정리: 1일
- **총 4일**

**자동화 수집 (스크립트 개발 후):**
- 스크립트 개발: 3일
- 크롤링 실행: 1시간
- 검증 및 분류: 1일
- **총 4일 (이후 재사용 가능)**
