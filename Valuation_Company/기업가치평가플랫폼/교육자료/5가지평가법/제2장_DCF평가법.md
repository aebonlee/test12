## 섹션 1: DCF 평가법이란 무엇인가

### 1.1 정의와 핵심 개념

**DCF(Discounted Cash Flow) 평가법**은 기업이나 자산이 미래에 창출할 현금흐름을 현재가치로 환산하여 가치를 평가하는 방법입니다. 기업가치평가 실무에서 가장 광범위하게 사용되는 방법론입니다.

DCF의 핵심은 **화폐의 시간가치(Time Value of Money)** 원리입니다. 오늘의 100만원과 5년 후의 100만원은 가치가 다릅니다. 오늘 받은 100만원은 투자하여 수익을 낼 수 있고, 인플레이션과 위험도 고려해야 하기 때문입니다.

**기본 공식:**
```
기업가치(Enterprise Value) = Σ[FCFt / (1+WACC)^t] + Terminal Value / (1+WACC)^n
```

여기서:
- FCF (Free Cash Flow): 기업이 영업활동으로 창출한 잉여현금흐름
- WACC (Weighted Average Cost of Capital): 가중평균자본비용 (할인율)
- Terminal Value: 추정기간 이후의 영속가치
- n: 추정기간 (통상 5~10년)

### 1.2 DCF 평가법이 필요한 이유

**1) 내재가치(Intrinsic Value) 발견**
- 주가나 시장 분위기와 무관하게 기업의 본질적 가치 산출
- 기업의 현금창출능력에 기반한 객관적 평가

**2) 투자 의사결정의 근거**
- VC/PE 투자 타당성 검증
- M&A 가격 협상의 기준점 설정
- IPO 공모가 산정

**3) 법적/제도적 요구사항 충족**
- 자본시장법상 외부평가의견서 제출 (비상장주식 발행 시)
- 상증세법상 보충적 평가방법으로 활용
- 금융감독원 "기업평가방법 수행기준" 준수

### 1.3 다른 평가법과의 차이

| 구분 | DCF (수익접근법) | 상대가치평가 (시장접근법) | 자산가치평가 (원가접근법) |
|------|------------------|------------------------|----------------------|
| 원리 | 미래 현금흐름의 현재가치 | 비교기업 배수 비교 | 순자산 공정가치 |
| 장점 | 논리적 엄밀성, 미래지향적 | 빠르고 직관적 | 명확한 하한선 제시 |
| 단점 | 계산 복잡, 가정 민감 | 시장 왜곡 반영 | 수익성 미반영 |
| 적합 대상 | 안정적 수익 기업, 성장기업 | 상장사, 비교 대상 多 | 자산집약적 기업, 청산 |

**DCF가 적합한 경우:**
- 합병, 중요한 영업/자산 양수도 시 (자본시장법 제161조)
- 상장사는 주주와 자산가치를 비교, 비상장사는 이익접근법과 원가접근법을 가중 평균
- 합병비율 평가 (비상장주식 평가방법론)에서도 DCF를 주로 활용

## 섹션 2: DCF 평가법의 계산 방법 (실무 단계별 가이드)

### 개요: DCF 평가 5단계 프로세스

DCF 평가는 다음 5단계로 진행됩니다. 각 단계는 재무제표 분석부터 최종 가치 산출까지 체계적으로 연결됩니다.

```
1단계: 재무제표 수집 및 과거 실적 분석
2단계: 미래 현금흐름 추정 (FCF 계산)
3단계: 할인율 산정 (WACC 계산)
4단계: Terminal Value 계산
5단계: 현재가치 할인 및 기업가치/주식가치 도출
```

---

### 2.1 1단계: 재무제표 수집 및 과거 실적 분석

#### 2.1.1 필요한 재무제표와 데이터 출처

**최소 요구사항: 과거 3~5개년 재무제표**

**1. 재무상태표 (Balance Sheet)**

**자산 항목:**
- 유동자산: 현금및현금성자산, 단기금융상품, 매출채권, 재고자산, 선급금, 기타유동자산
- 비유동자산: 유형자산(토지, 건물, 기계장치, 건설중인자산), 무형자산(영업권, 산업재산권, 소프트웨어), 투자자산(장기금융상품, 지분법적용투자주식)

**부채 항목:**
- 유동부채: 매입채무, 단기차입금, 유동성장기부채, 미지급금, 선수금, 미지급비용
- 비유동부채: 장기차입금, 사채, 퇴직급여충당부채, 이연법인세부채

**자본 항목:**
- 자본금, 자본잉여금, 이익잉여금(이익준비금, 미처분이익잉여금)

**2. 손익계산서 (Income Statement)**

**영업손익:**
- 매출액 (제품매출, 상품매출, 용역매출)
- 매출원가 (재료비, 노무비, 제조경비)
- **매출총이익**
- 판매비와관리비 (급여, 퇴직급여, 복리후생비, 광고선전비, 운반비, 감가상각비, 연구개발비)
- **영업이익 (EBIT)**

**영업외손익:**
- 금융수익 (이자수익, 배당금수익, 외환차익, 외화환산이익)
- 금융비용 (이자비용, 외환차손, 외화환산손실)
- 기타수익 (유형자산처분이익, 투자자산처분이익, 잡이익)
- 기타비용 (유형자산처분손실, 기부금, 잡손실)
- **법인세비용차감전순이익**

**법인세 및 당기순이익:**
- 법인세비용
- **당기순이익**

**3. 현금흐름표 (Cash Flow Statement)**

- **영업활동 현금흐름**: 당기순이익에서 출발하여 비현금항목 조정
- **투자활동 현금흐름**: 유형자산의 취득(-), 처분(+), 무형자산의 취득(-), 투자자산의 취득(-), 처분(+)
  - **중요: 자본적지출(CapEx) = 유형자산 취득 + 무형자산 취득**
- **재무활동 현금흐름**: 차입금의 증가(+), 상환(-), 배당금 지급(-)

**데이터 출처:**

**상장사:**
- **DART (dart.fss.or.kr)**: 금융감독원 전자공시시스템
  - 사업보고서 > 재무제표 다운로드 (PDF 또는 XBRL)
  - 과거 5개년 비교 가능
- **KIND (kind.krx.or.kr)**: 한국거래소 상장사 정보
  - 재무정보 > 주요재무정보 > 엑셀 다운로드
- **TS2000, FnGuide, Bloomberg**: 전문 데이터베이스 (유료)

**비상장사:**
- 경영진 제공 재무제표 (감사받은 재무제표 권장)
- 외부감사보고서 (자산 5억 이상 또는 매출 10억 이상 기업)
- 국세청 제출 법인세 신고서 (재무제표 첨부)

**비교기업 데이터:**
- KIND: 동종업종 상장사 검색
- 한국은행 기업경영분석: 업종별 평균 재무비율
- 통계청 KOSIS: 산업별 매출 성장률

#### 2.1.2 과거 실적 분석 및 정규화 (Normalization)

**왜 정규화가 필요한가?**

재무제표에는 일회성 항목, 비경상적 손익, 회계정책 차이가 포함되어 있습니다. DCF는 "지속가능한 영업 현금흐름"을 평가하므로, 이러한 왜곡 요소를 제거해야 합니다.

**정규화 대상 항목:**

**1) 비경상적 손익 제거**
- 자산처분손익: 부동산 매각, 투자자산 처분, 유형자산 처분
- 외환차손익: 환율 급변동으로 인한 일회성 손익
- 소송 관련 손익: 승소/패소에 따른 손해배상금
- 구조조정 비용: 정리해고, 사업 철수, 희망퇴직

**2) 일회성 수익/비용 조정**
- 정부 보조금: R&D 지원금, 고용보조금 (지속성 없으면 제외)
- 대규모 광고비: 신제품 출시 캠페인 (일회성이면 제외)
- 특별 상여금: 창립기념 특별보너스

**3) 관계사 거래 조정**
- 특수관계자와의 거래가 시가와 다르면 공정가치로 조정
- 예: 계열사에 원가 이하로 제품 공급 → 시장가격으로 재계산

**정규화 실무 프로세스:**

**Step 1: 감사보고서 주석 사항 검토**
- 재무제표 주석에서 "영업외수익/비용" 상세 내역 확인
- 특수관계자거래 주석에서 거래 조건 확인

**Step 2: 조정 항목 식별 및 근거 기록**

| 계정과목 | 보고금액 | 조정액 | 조정 후 | 조정 근거 |
|----------|---------|-------|---------|----------|
| 영업이익 | 200억 | - | 200억 | - |
| 유형자산처분이익 | 50억 | -50억 | 0억 | 임대건물 매각 (비경상) |
| 정부보조금 | 10억 | -10억 | 0억 | R&D 일회성 지원 |
| 희망퇴직비용 | 15억 | +15억 | 0억 | 구조조정 비용 (비경상) |
| **정규화 EBIT** | | | **215억** | |

**Step 3: 정규화 EBIT 계산**
```
정규화 EBIT = 보고 EBIT + 비경상 비용 - 비경상 수익
```

**실무 예시: 제조업체 A사 2024년 정규화**

**손익계산서 (단위: 억원)**

| 항목 | 보고 금액 | 조정 | 정규화 금액 | 비고 |
|------|----------|------|------------|------|
| 매출액 | 1,000 | - | 1,000 | |
| 매출원가 | (600) | - | (600) | |
| **매출총이익** | **400** | | **400** | |
| 판매비관리비 | (200) | +20 | (180) | 신제품 런칭 마케팅비 20억 제외 |
| **영업이익 (EBIT)** | **200** | | **220** | |
| 이자수익 | 5 | - | 5 | |
| 이자비용 | (30) | - | (30) | |
| 외환차익 | 5 | -5 | 0 | 환율 급등에 따른 일회성 |
| 유형자산처분이익 | 50 | -50 | 0 | 유휴부동산 매각 |
| 희망퇴직비용 | (20) | +20 | 0 | 구조조정 비용 |
| **법인세비용차감전순이익** | **210** | | **195** | |
| 법인세비용 (25%) | (53) | | (49) | 정규화 이익 기준 재계산 |
| **정규화 당기순이익** | | | **146** | |
| **정규화 EBIT (DCF 사용)** | | | **220억** | |

**엑셀 작업 팁:**
1. 원본 재무제표 시트와 "Normalized" 시트 분리
2. 조정 내역 시트에 근거 문서 페이지 번호 기록
3. 정규화 EBIT를 별도 셀에 저장 → FCFF 계산 시트에 연결

**Step 4: 과거 추이 분석**

정규화된 재무제표로 과거 3~5개년 추이를 분석합니다.

**주요 분석 지표:**

**성장률:**
```
매출 CAGR = (최종년도 매출 / 기초년도 매출)^(1/연수) - 1
```

**수익성:**
```
매출총이익률 = 매출총이익 / 매출액 × 100%
영업이익률 = EBIT / 매출액 × 100%
EBITDA 마진 = (EBIT + 감가상각비) / 매출액 × 100%
```

**자산 효율성:**
```
총자산회전율 = 매출액 / 평균총자산
운전자본회전율 = 매출액 / 순운전자본
CapEx/매출액 비율 = 자본적지출 / 매출액 × 100%
```

**분석 예시 테이블:**

| 연도 | 2022 | 2023 | 2024 | CAGR/추이 |
|------|------|------|------|----------|
| 매출액 (억원) | 850 | 920 | 1,000 | 8.5% |
| 정규화 EBIT (억원) | 170 | 190 | 220 | 13.9% |
| 영업이익률 | 20.0% | 20.7% | 22.0% | ↑ 개선 |
| 감가상각비 (억원) | 40 | 45 | 50 | 11.8% |
| EBITDA (억원) | 210 | 235 | 270 | 13.4% |
| EBITDA 마진 | 24.7% | 25.5% | 27.0% | ↑ 개선 |
| CapEx (억원) | 80 | 85 | 90 | 6.1% |
| CapEx/매출액 | 9.4% | 9.2% | 9.0% | ↓ 안정화 |

**해석:**
- 매출은 연 8.5% 성장, 영업이익은 13.9% 성장 → 수익성 개선 중 (레버리지 효과)
- 영업이익률 상승 (20% → 22%) → 규모의 경제 또는 원가 절감 효과
- CapEx 비중 감소 → 성장기에서 성숙기로 전환 신호 (미래 FCF 증가 예상)
- EBITDA 마진 지속 개선 → 영업 효율성 향상

### 2.2 2단계: 미래 현금흐름 추정 (FCFF 계산)

#### 2.2.1 FCFF (Free Cash Flow to Firm) 개념

**FCFF 정의:**

기업이 영업활동으로 창출한 현금 중, 운영 유지에 필요한 투자를 하고 남은 "주주와 채권자 모두에게 배분 가능한 현금"입니다.

**FCFF vs FCFE:**
- **FCFF (Free Cash Flow to Firm)**: 기업 전체 가치 평가 시 사용 → 기업가치(EV) 산출
- **FCFE (Free Cash Flow to Equity)**: 자기자본 가치만 평가 시 사용 → 주식가치 직접 산출

**관계:**
```
FCFE = FCFF - 이자비용×(1-세율) + 순차입 증가
```

**본 교육자료는 FCFF 기준으로 설명합니다** (실무에서 가장 많이 사용됨).

#### 2.2.2 FCFF 계산 공식

**표준 공식 (NOPLAT 기준):**

```
FCFF = NOPLAT
       + 감가상각비 및 무형자산상각비
       - 자본적지출 (CapEx)
       - 순운전자본 증가액 (ΔNWC)
```

**각 항목 설명:**

**NOPLAT (Net Operating Profit Less Adjusted Taxes)**
= 세후영업이익 (금융비용 차감 전)
```
NOPLAT = EBIT × (1 - 실효세율)
```

**감가상각비 (+)**
= 유형자산 감가상각비 + 무형자산 상각비
- 손익계산서에서는 비용이지만 실제 현금 유출 없음 → 더해줌

**자본적지출 (CapEx, -)**
= 유형자산 취득 + 무형자산 취득
- 현금흐름표의 투자활동에서 확인

**순운전자본 증가액 (ΔNWC, -)**
= NWC(당기) - NWC(전기)
- 매출 증가 시 운전자본 증가 → 현금 묶임 → 차감

#### 2.2.3 FCFF 구성요소 상세 계산법

##### (1) NOPLAT (세후영업이익) 계산

**Step 1: 정규화 EBIT 확인**
- 섹션 2.1.2에서 계산한 정규화 EBIT 사용
- A사 2024년: 220억원

**Step 2: 실효세율 계산**

```
실효세율 = 법인세비용 / 법인세비용차감전순이익
```

**왜 이론세율(25%)이 아닌 실효세율을 사용하는가?**
- 세액공제(R&D 세액공제, 고용증대 세액공제, 투자세액공제)로 인해 실제 납부 세율은 낮아짐
- 이월결손금으로 법인세 납부 면제 기간 있을 수 있음
- 실무에서 실효세율은 보통 15~22%

**과거 3개년 실효세율 계산:**

| 연도 | 법인세비용차감전순이익 (억원) | 법인세비용 (억원) | 실효세율 |
|------|------------------------------|-----------------|---------|
| 2022 | 180 | 40 | 22.2% |
| 2023 | 200 | 45 | 22.5% |
| 2024 | 195 | 49 | 25.1% |
| **평균** | | | **23.3%** |

**실무 판단:**
- 2024년은 일회성 과세로 높음 → 과거 평균 23.3% 사용
- 또는 보수적으로 법정세율 25% 사용 가능

**Step 3: NOPLAT 계산**

```
NOPLAT = 정규화 EBIT × (1 - 실효세율)
        = 220억 × (1 - 0.233)
        = 220억 × 0.767
        = 168.7억원 ≈ 169억원
```

##### (2) 감가상각비 및 무형자산상각비

**데이터 출처 (우선순위):**

**1순위: 현금흐름표**
- 영업활동 현금흐름 부분
- "감가상각비" 항목 (가장 정확)

**2순위: 주석 사항**
- 유형자산 명세
  - 기초장부가액 + 취득 - 처분 - 감가상각비 = 기말장부가액
  - 감가상각비 확인
- 무형자산 명세
  - 무형자산상각비 확인

**3순위: 손익계산서 주석**
- 판매비와관리비 세부 내역
- "감가상각비" 항목

**A사 2024년 예시:**
- 유형자산 감가상각비: 45억원 (기계장치 30억, 건물 10억, 차량운반구 5억)
- 무형자산 상각비: 5억원 (소프트웨어 3억, 산업재산권 2억)
- **합계: 50억원**

**왜 더하는가?**
감가상각비는 비현금비용입니다. 손익계산서에서는 비용으로 차감되어 이익을 줄이지만, 실제로 현금이 나간 것은 아니므로 현금흐름 계산 시 다시 더해줍니다.

##### (3) 자본적지출 (CapEx) 추정

**CapEx 정의:**
유형자산/무형자산 취득에 사용된 현금 (토지, 건물, 기계장치, 차량, 소프트웨어 등)

**데이터 출처:**
현금흐름표 > 투자활동 현금흐름
```
CapEx = 유형자산의 취득 + 무형자산의 취득
```

**과거 CapEx 확인:**

| 연도 | 유형자산 취득 | 무형자산 취득 | CapEx 합계 | 매출액 | CapEx/매출 |
|------|-------------|-------------|-----------|--------|-----------|
| 2022 | 75 | 5 | 80 | 850 | 9.4% |
| 2023 | 80 | 5 | 85 | 920 | 9.2% |
| 2024 | 85 | 5 | 90 | 1,000 | 9.0% |
| **평균** | | | | | **9.2%** |

**미래 CapEx 추정 방법:**

**방법 A: 과거 평균 비율 적용 (가장 일반적)**
```
미래 CapEx = 추정 매출액 × (과거 평균 CapEx/매출액 비율)
```

**방법 B: 성장 단계별 차등 적용**
- **성장기**: 매출액의 10~15% (설비 확장, 생산능력 증대)
- **성숙기**: 매출액의 5~8% (유지보수, 대체투자)
- **감가상각비와 비교**: 성숙기는 CapEx ≈ 감가상각비 (Maintenance CapEx)

**방법 C: 경영진 사업계획 반영 (구체적 투자 계획이 있는 경우)**
- 대규모 공장 신설, 설비 교체 계획을 연도별로 반영
- 예: 2026년 신공장 건설 200억원 → 2026년 CapEx에 200억 추가

**실무 추정 예시: A사 미래 CapEx**

**가정:**
- 2025~2027: 성장기 지속 → 과거 평균 9.2% 유지
- 2028~2029: 성숙기 진입 → 점진적으로 8.5%로 하향 (유지보수 수준)

| 연도 | 추정 매출액 (억원) | CapEx 비율 | CapEx (억원) | 비고 |
|------|------------------|-----------|-------------|------|
| 2025E | 1,080 | 9.2% | 99 | 과거 평균 |
| 2026E | 1,155 | 9.2% | 106 | |
| 2027E | 1,220 | 9.0% | 110 | 성숙기 진입 시작 |
| 2028E | 1,280 | 8.8% | 113 | |
| 2029E | 1,320 | 8.5% | 112 | 유지보수 수준 |

**검증: CapEx vs 감가상각비 비교**

| 연도 | CapEx | 감가상각비 | CapEx/감가상각비 |
|------|-------|-----------|----------------|
| 2025E | 99 | 54 | 1.83배 (성장기) |
| 2026E | 106 | 58 | 1.83배 |

→ 점진적으로 1.5배 수준으로 수렴 (합리적 추세)

##### (4) 순운전자본 (NWC) 변동액 계산

**순운전자본 정의:**

```
순운전자본 (NWC) = 영업 관련 유동자산 - 영업 관련 유동부채
                  = (매출채권 + 재고자산 + 선급금) - (매입채무 + 선수금)
```

**상세 계산식 (정확한 방법):**
```
NWC = 유동자산 (현금 제외) - 무이자 유동부채
    = (매출채권 + 재고자산 + 선급금 + 기타유동자산)
      - (매입채무 + 선수금 + 미지급비용 + 기타유동부채)
```

**제외 항목:**
- 현금및현금성자산: 운전자본 아님 (잉여 현금)
- 단기차입금, 유동성장기부채: 재무활동 부채 (이자 발생)

**순운전자본 변동액:**
```
ΔNWC = NWC(t) - NWC(t-1)
```

**왜 차감하는가?**

운전자본이 증가하면 현금이 묶입니다.
- 매출채권 증가 = 외상매출 증가 = 현금 미회수 = 현금 유출
- 재고자산 증가 = 재고 매입 = 현금 유출
- 매입채무 증가 = 외상매입 증가 = 현금 유출 지연 = 플러스 효과 (부채 증가는 현금 절약)

**과거 NWC 계산 예시:**

**A사 2023~2024년 재무상태표 (단위: 억원)**

| 항목 | 2023년 | 2024년 | 변동 |
|------|--------|--------|------|
| **유동자산** | | | |
| 현금및현금성자산 | 100 | 120 | +20 (제외) |
| 매출채권 | 140 | 150 | +10 |
| 재고자산 | 95 | 100 | +5 |
| 선급금 | 10 | 12 | +2 |
| **유동부채** | | | |
| 매입채무 | (75) | (80) | -5 (채무 증가는 플러스) |
| 선수금 | (5) | (6) | -1 |
| 미지급비용 | (8) | (10) | -2 |
| 단기차입금 | (100) | (120) | (제외, 재무부채) |
| **NWC** | **157** | **166** | **+9억** |

**해석:**
- 2024년 NWC 증가 9억원 → FCFF에서 9억원 차감 (현금 유출)

**미래 NWC 증가액 추정 방법:**

**방법 1: 매출액 비례법 (간단)**
```
NWC/매출액 비율 = 과거 평균 유지
ΔNWC(t) = (NWC/매출액) × [매출액(t) - 매출액(t-1)]
```

**예시:**
- 과거 평균 NWC/매출액 = 166/1,000 = 16.6%
- 2025년 매출 증가: 1,080 - 1,000 = 80억원
- ΔNWC(2025) = 16.6% × 80 = 13.3억원 ≈ 13억원

**방법 2: 회전율 기준 (더 정확)**

**회전일수 계산:**
```
매출채권 회전일수 = (매출채권 / 매출액) × 365일
재고자산 회전일수 = (재고자산 / 매출원가) × 365일
매입채무 회전일수 = (매입채무 / 매출원가) × 365일
```

**A사 과거 회전일수:**

| 항목 | 2022 | 2023 | 2024 | 평균 |
|------|------|------|------|------|
| 매출채권 회전일수 | 58일 | 56일 | 55일 | 56일 |
| 재고자산 회전일수 | 52일 | 50일 | 49일 | 50일 |
| 매입채무 회전일수 | 42일 | 40일 | 41일 | 41일 |

**미래 NWC 역산:**
```
매출채권(2025) = 매출액(2025) × (56/365) = 1,080 × 0.153 = 165억원
재고자산(2025) = 매출원가(2025) × (50/365) = 650 × 0.137 = 89억원
매입채무(2025) = 매출원가(2025) × (41/365) = 650 × 0.112 = 73억원
```

**실무 추정 테이블: A사 미래 NWC 변동**

| 연도 | 매출액 | NWC/매출 | NWC | ΔNWC |
|------|--------|---------|-----|------|
| 2024 | 1,000 | 16.6% | 166 | - |
| 2025E | 1,080 | 16.6% | 179 | 13 |
| 2026E | 1,155 | 16.6% | 192 | 13 |
| 2027E | 1,220 | 16.6% | 203 | 11 |
| 2028E | 1,280 | 16.5% | 211 | 8 (성숙기, 효율 개선) |
| 2029E | 1,320 | 16.5% | 218 | 7 |

**실무 팁:**
- 성장기: NWC 증가 부담 큼 → FCF 감소
- 성숙기: 매출 둔화 + 회전율 개선 → NWC 증가 최소 → FCF 증가
- 회전율 개선 시나리오: 재고 관리 효율화, 매출채권 회수 단축 → ΔNWC 감소 → FCF 증가

#### 2.2.4 FCFF 통합 계산 및 5개년 추정

**FCFF 계산 프로세스 정리:**

```
① NOPLAT = EBIT × (1 - 실효세율)
② (+) 감가상각비
③ (-) CapEx
④ (-) ΔNWC
⑤ FCFF = ① + ② - ③ - ④
```

**A사 2025년 FCFF 계산 예시:**

| 단계 | 항목 | 금액 (억원) | 계산 근거 |
|------|------|-----------|----------|
| ① | EBIT | 238 | 매출 1,080억 × 영업이익률 22% |
| | 실효세율 | 23.3% | 과거 3년 평균 |
| | **NOPLAT** | **182** | 238 × (1 - 0.233) |
| ② | (+) 감가상각비 | 54 | 매출액의 5% |
| ③ | (-) CapEx | (99) | 매출액의 9.2% |
| ④ | (-) ΔNWC | (13) | NWC 16.6% × 매출 증가 80억 |
| ⑤ | **FCFF** | **124억원** | 182 + 54 - 99 - 13 |

**5개년 FCFF 추정 테이블 (단위: 억원)**

| 항목 | 2025E | 2026E | 2027E | 2028E | 2029E |
|------|-------|-------|-------|-------|-------|
| 매출액 | 1,080 | 1,155 | 1,220 | 1,280 | 1,320 |
| 매출 성장률 | 8.0% | 6.9% | 5.6% | 4.9% | 3.1% |
| 영업이익률 | 22.0% | 22.0% | 22.0% | 22.0% | 22.0% |
| EBIT | 238 | 254 | 268 | 282 | 290 |
| 실효세율 | 23.3% | 23.3% | 23.3% | 23.3% | 23.3% |
| **NOPLAT** | **182** | **195** | **206** | **216** | **223** |
| (+) 감가상각비 | 54 | 58 | 61 | 64 | 66 |
| (-) CapEx | (99) | (106) | (110) | (113) | (112) |
| (-) ΔNWC | (13) | (13) | (11) | (8) | (7) |
| **FCFF** | **124** | **134** | **146** | **159** | **170** |

**추세 분석:**
- FCFF가 연평균 8.2% 성장 (매출 성장률 5.7%보다 높음)
- 이유: 영업이익률 안정 + CapEx 비중 감소 + ΔNWC 감소
- 성숙기 진입으로 현금창출력 개선

### 2.3 3단계: 할인율 산정 (WACC 계산)

#### 2.3.1 WACC 개념과 구성

**WACC (Weighted Average Cost of Capital)**는 기업이 자금을 조달하는 데 드는 가중평균비용입니다. 주주(자기자본)와 채권자(타인자본) 모두에게 지급해야 하는 요구수익률의 가중평균입니다.

**기본 공식:**
```
WACC = (E / V) × Re + (D / V) × Rd × (1 - T)
```

**기호 설명:**
- E: 자기자본의 시장가치 (Market Value of Equity)
- D: 타인자본의 시장가치 (Market Value of Debt)
- V: 총자본 = E + D
- Re: 자기자본비용 (Cost of Equity) - CAPM으로 계산
- Rd: 타인자본비용 (Cost of Debt) - 차입금리
- T: 실효세율 (법인세 절감 효과)

**왜 타인자본비용에만 (1-T)를 곱하는가?**

이자비용은 법인세 차감 항목이므로, 실질 비용은 세후 비용입니다.
```
예: 이자율 5%, 법인세율 25%
실질 비용 = 5% × (1 - 0.25) = 3.75%
```

반면 배당금은 세금 차감 전 이익에서 지급하므로 세후 효과 없음.

#### 2.3.2 자기자본비용 (Re) 계산: CAPM 모형

**CAPM (Capital Asset Pricing Model)**을 사용합니다.

```
Re = Rf + β × ERP + (SRP)
```

**기호 설명:**
- Rf: 무위험이자율 (Risk-Free Rate)
- β (베타): 시장 대비 체계적 위험
- ERP: 시장위험프리미엄 (Equity Risk Premium) = Rm - Rf
- SRP: 규모위험프리미엄 (Size Risk Premium) - 선택적

##### (1) 무위험이자율 (Rf) 결정

**국내 실무 표준:**
- **국고채 3년물 수익률** (단기 프로젝트, 일반적)
- **국고채 10년물 수익률** (장기 프로젝트, DCF 권장)

**데이터 출처:**
- 한국은행 경제통계시스템 (ecos.bok.or.kr)
  - 금리 > 시장금리(일별) > 국고채(3년, 10년)
- 채권정보센터 (www.kofiabond.or.kr)
- 평가기준일 또는 최근 1개월 평균 사용

**2025년 1월 기준 예시:**
- 국고채 3년: 3.2%
- 국고채 10년: 3.5%
- **DCF 평가 시 Rf = 3.5% 권장** (장기 현금흐름이므로)

##### (2) 베타 (β) 산출: 5단계 프로세스

베타는 기업 주가 수익률과 시장 수익률 간의 민감도입니다.

**베타 해석:**
- β = 1.0: 시장과 동일한 변동성
- β > 1.0: 시장보다 변동성 높음 (높은 위험, 높은 수익)
- β < 1.0: 시장보다 안정적 (낮은 위험, 낮은 수익)

**비상장사 베타 산출 5단계 (핵심 프로세스)**

**Step 1: 유사 상장사 선정**

**선정 기준:**
1. 동종 업종 (KSIC 4자리 코드 동일 또는 유사)
2. 유사한 사업 모델 및 제품군
3. 유사한 매출 규모 (가능하면)
4. 최소 3~5개사 선정

**A사(자동차 부품 제조) 유사 상장사 선정 예시:**

| 회사명 | 업종 (KSIC) | 주요 제품 | 매출액 (억원) | 선정 여부 |
|--------|-----------|----------|-------------|---------|
| ㄱ전자 | C313 (자동차 부품) | 자동차 전장부품 | 8,000 | ⭕ |
| ㄴ산업 | C313 | 자동차 샤시 부품 | 6,500 | ⭕ |
| ㄷ테크 | C313 | 자동차 내장부품 | 12,000 | ⭕ |
| ㄹ부품 | C313 | 자동차 금형 및 부품 | 5,000 | ⭕ |
| ㅁ전자 | C264 (반도체) | 자동차용 반도체 | 20,000 | ❌ (업종 상이) |

**Step 2: 유사 상장사의 Levered Beta 수집**

**데이터 출처:**
- KIND (kind.krx.or.kr): 기업정보 > 기업개요 > Beta
- FnGuide, TS2000: 기업 개요 페이지
- Bloomberg: `Equity > DES > Beta`

**직접 계산 방법 (엑셀):**

1. **주가 및 시장지수 데이터 수집**
   - 기간: 최근 2년 주간(weekly) 또는 5년 월간(monthly)
   - 개별 주가: KIND, 네이버 금융
   - 시장지수: KOSPI 또는 KOSDAQ

2. **수익률 계산**
```
수익률(t) = [가격(t) - 가격(t-1)] / 가격(t-1)
```

3. **엑셀 회귀분석**
   - [데이터] 탭 > [데이터 분석] > [회귀분석]
   - Y 범위: 개별 주식 수익률
   - X 범위: 시장지수 수익률
   - **출력: X 1 계수 = Raw Beta**

4. **Adjusted Beta 계산 (Bloomberg 방식)**
```
Adjusted Beta = (2/3) × Raw Beta + (1/3) × 1.0
```

**유사 상장사 베타 수집 결과:**

| 회사명 | Raw Beta | Adjusted Beta | 시가총액 (억원) | D/E 비율 |
|--------|----------|--------------|--------------|---------|
| ㄱ전자 | 1.15 | 1.10 | 5,000 | 0.50 |
| ㄴ산업 | 1.35 | 1.23 | 3,500 | 0.80 |
| ㄷ테크 | 0.95 | 0.97 | 8,000 | 0.30 |
| ㄹ부품 | 1.25 | 1.17 | 4,000 | 0.60 |

**Step 3: Unlevered Beta (자산 베타) 계산**

부채의 영향을 제거한 순수한 영업 위험만 반영하는 베타입니다.

**Hamada 공식:**
```
Unlevered Beta = Levered Beta / [1 + (1 - T) × (D / E)]
```

**왜 Unlever하는가?**
각 기업의 부채 레버리지가 다르므로, 순수한 사업 위험만 비교하기 위해 부채 효과를 제거합니다.

**유사 상장사별 Unlevered Beta 계산:**

**예시: ㄱ전자**
```
Levered Beta = 1.10
D/E = 0.50 (부채 1,000억, 자기자본 2,000억)
T = 25% (법인세율)

Unlevered Beta = 1.10 / [1 + (1 - 0.25) × 0.50]
                = 1.10 / [1 + 0.375]
                = 1.10 / 1.375
                = 0.80
```

**전체 유사사 Unlevering 결과:**

| 회사명 | Levered β | D/E | T | Unlevered β |
|--------|----------|-----|---|------------|
| ㄱ전자 | 1.10 | 0.50 | 25% | 0.80 |
| ㄴ산업 | 1.23 | 0.80 | 25% | 0.76 |
| ㄷ테크 | 0.97 | 0.30 | 25% | 0.84 |
| ㄹ부품 | 1.17 | 0.60 | 25% | 0.81 |
| **평균** | | | | **0.80** |
| **중위값** | | | | **0.81** |

**Step 4: 평균 또는 중위값 선택**

**실무 판단:**
- 표본이 5개 이상이고 분포가 정규분포에 가까우면 **평균** 사용
- 표본이 적거나 이상치(outlier)가 있으면 **중위값** 사용

**A사: 평균 0.80과 중위값 0.81이 유사 → 평균 0.80 사용**

**Step 5: 평가대상사의 자본구조로 Re-lever**

평가대상 비상장사 A사의 자본구조를 반영하여 베타를 재계산합니다.

**Re-levering 공식:**
```
Levered Beta (A사) = Unlevered Beta × [1 + (1 - T) × (D / E)_A사]
```

**A사 자본구조:**
- 부채 (D): 800억원 (단기차입금 200억 + 장기차입금 600억)
- 자기자본 (E): 2,000억원 (장부가치, 비상장이므로)
- D/E = 800 / 2,000 = 0.40
- 실효세율 (T): 23.3%

**계산:**
```
Levered Beta (A사) = 0.80 × [1 + (1 - 0.233) × 0.40]
                    = 0.80 × [1 + 0.767 × 0.40]
                    = 0.80 × [1 + 0.307]
                    = 0.80 × 1.307
                    = 1.05
```

**A사 베타 = 1.05**

**실무 주의사항:**
- 유사 상장사가 3개 미만: 산업 평균 베타 사용 (한국은행, 업종 리서치 참조)
- 이상치 제거: 베타가 2.0 이상 또는 0.3 이하인 기업은 제외 검토
- 비상장사 자본구조: 장부가치 사용 (시장가치 불명)

##### (3) 시장위험프리미엄 (ERP) 결정

**ERP (Equity Risk Premium)**는 주식시장에 투자할 때 무위험자산 대비 추가로 요구하는 수익률입니다.

```
ERP = E(Rm) - Rf
```
- E(Rm): 시장 기대수익률
- Rf: 무위험이자율

**한국 시장 ERP 실무 표준:**

| 출처 | ERP | 비고 |
|------|-----|------|
| Damodaran (NYU) | 8.5% | 2025년 한국 추정치 |
| 한국은행 연구 | 7.5~9.5% | 역사적 평균 |
| 국내 투자은행 | 8~9% | 실무 통용 |
| **권장값** | **8.5%** | 국제 표준 |

**ERP 산출 방법론 (참고):**

**방법 1: Historical Premium (과거 데이터 기반)**
```
ERP = 과거 30년 KOSPI 평균 수익률 - 국고채 평균 수익률
```
- 장점: 객관적 데이터
- 단점: 과거가 미래를 대표하지 못할 수 있음

**방법 2: Implied ERP (내재 프리미엄)**
```
Gordon 모형에서 역산:
ERP = (배당수익률 + 배당성장률) - Rf
```
- 장점: 현재 시장 기대 반영
- 단점: 계산 복잡, 배당정책 영향

**글로벌 비교:**
- 미국 ERP: 약 5.5%
- 한국 Country Risk Premium: 약 3.0%
- 한국 Total ERP = 5.5% + 3.0% = 8.5%

##### (4) 규모위험프리미엄 (SRP) - 선택적

중소기업/중견기업은 대기업보다 위험이 높으므로 추가 프리미엄을 더할 수 있습니다.

**Ibbotson 방법론 (미국 데이터 기반):**

| 시가총액 | SRP |
|---------|-----|
| < $200M | +3~5% |
| $200M ~ $500M | +2~3% |
| > $500M | 0% |

**한국 실무 적용 (매출액 기준):**

| 매출 규모 | SRP | 적용 대상 |
|----------|-----|----------|
| 500억 미만 | +2~3% | 중소기업 |
| 500~1,000억 | +1~2% | 중견기업 |
| 1,000억 이상 | 0% | 대기업 |

**A사: 매출 1,000억 이상 → SRP = 0% (미적용)**

**주의사항:**
- SRP는 선택적이며, 보수적 평가 시 적용
- VC/PE 투자, 초기 스타트업 평가에서 주로 사용
- 상장사 인수 평가에서는 일반적으로 미적용

##### (5) 자기자본비용 (Re) 최종 계산

**A사 자기자본비용 계산:**

```
Re = Rf + β × ERP + SRP
   = 3.5% + 1.05 × 8.5% + 0%
   = 3.5% + 8.93% + 0%
   = 12.43% ≈ 12.4%
```

**A사 자기자본비용 (Re) = 12.4%**

**만약 SRP 2% 적용 시 (중소기업 가정):**
```
Re = 3.5% + 1.05 × 8.5% + 2.0% = 14.43%
```

#### 2.3.3 타인자본비용 (Rd) 계산

**타인자본비용**은 기업이 차입에 대해 지급하는 평균 이자율입니다.

**방법 1: 재무제표 기준 (가장 간단)**

```
Rd = 이자비용 / 평균 총차입금
```

**A사 2024년:**
- 이자비용: 30억원 (손익계산서)
- 기초 총차입금 (2023년 말): 750억원
- 기말 총차입금 (2024년 말): 800억원
- 평균 총차입금 = (750 + 800) / 2 = 775억원

```
Rd = 30 / 775 = 3.87% ≈ 3.9%
```

**방법 2: 신규 차입금리 기준 (더 정확)**

미래 차입 조건을 반영하는 것이 더 합리적입니다.

- 최근 대출 계약서의 이자율 확인
- 금융기관 제시 금리 (예: CD금리 + 가산금리)
- 상장사의 경우 사채 발행 수익률 참조

**A사 최근 대출 조건:**
- CD금리 (3개월): 3.5%
- 가산금리: 1.5%p (신용등급 A 기준)
- Rd = 3.5% + 1.5% = 5.0%

**방법 3: 신용등급 기반 추정**

| 신용등급 | 차입금리 (예시, 2025년 기준) |
|----------|---------------------------|
| AAA | CD금리 + 0.5% = 4.0% |
| AA | CD금리 + 1.0% = 4.5% |
| A | CD금리 + 1.5~2.0% = 5.0~5.5% |
| BBB | CD금리 + 2.5~3.0% = 6.0~6.5% |
| BB 이하 | CD금리 + 4.0% 이상 |

**A사: 신용등급 A → Rd = 5.0% 적용**

**실무 판단:**
- 재무제표 이자비용은 과거 금리 반영
- 신규 차입금리는 미래 조달 비용 반영
- **DCF는 미래 평가이므로 신규 차입금리(5.0%) 사용 권장**

#### 2.3.4 자본구조 (E/V, D/V) 결정

**목표 자본구조 원칙:**

DCF는 미래 현금흐름 평가이므로, "미래의 자본구조"를 반영해야 합니다.

**방법 1: 현재 시장가치 기준 (상장사)**
```
E = 시가총액 (주가 × 발행주식수)
D = 부채 장부가치 (재무상태표)
```

**방법 2: 유사 상장사 평균 자본구조 (비상장사 권장)**

**Step 1: 유사 상장사 자본구조 조사**

| 회사명 | 시가총액 (E) | 부채 (D) | 총자본 (V) | E/V | D/V |
|--------|------------|----------|-----------|-----|-----|
| ㄱ전자 | 5,000 | 1,000 | 6,000 | 83.3% | 16.7% |
| ㄴ산업 | 3,500 | 1,200 | 4,700 | 74.5% | 25.5% |
| ㄷ테크 | 8,000 | 900 | 8,900 | 89.9% | 10.1% |
| ㄹ부품 | 4,000 | 1,080 | 5,080 | 78.7% | 21.3% |
| **평균** | | | | **81.6%** | **18.4%** |

**Step 2: A사 목표 자본구조 설정**
- 유사사 평균 적용: E/V = 82%, D/V = 18%
- 또는 경영진이 목표로 하는 구조 사용

**순환참조(Circular Reference) 문제:**

WACC 계산에 자기자본 시장가치(E)가 필요한데, E는 DCF 결과로 나옵니다.

**해결법 A: Iteration (반복 계산)**
1. 초기값: 장부가치 자본구조로 WACC₁ 계산
2. WACC₁로 DCF → 기업가치₁ 산출
3. 기업가치₁로 자본구조 업데이트 → WACC₂ 재계산
4. 2~3회 반복하면 수렴

**해결법 B: 목표 자본구조 고정 (실무 일반적)**
- 유사사 평균 또는 경영진 목표 자본구조를 고정
- 순환참조 없이 WACC 직접 계산

**A사: 해결법 B 적용 → E/V = 82%, D/V = 18%**

#### 2.3.5 WACC 최종 계산

**A사 WACC 계산 종합:**

**입력값 정리:**
- Re (자기자본비용): 12.4%
- Rd (타인자본비용): 5.0%
- E/V (자기자본 비중): 82%
- D/V (타인자본 비중): 18%
- T (실효세율): 23.3%

**계산:**
```
WACC = (E / V) × Re + (D / V) × Rd × (1 - T)
     = 0.82 × 12.4% + 0.18 × 5.0% × (1 - 0.233)
     = 10.17% + 0.90% × 0.767
     = 10.17% + 0.69%
     = 10.86% ≈ 10.9%
```

**A사 WACC = 10.9% (DCF 할인율로 사용)**

**검증: 합리성 체크**
- 일반 제조업 WACC 범위: 8~12% → 10.9%는 합리적
- Re > WACC > Rd(1-T) 확인: 12.4% > 10.9% > 3.8% ✓

**민감도 분석: 베타 변동 시 WACC 영향**

| 베타 (β) | Re | WACC | 비고 |
|---------|----|----|------|
| 0.9 | 11.2% | 9.9% | -1.0%p |
| 1.0 | 12.0% | 10.5% | -0.4%p |
| **1.05** | **12.4%** | **10.9%** | **Base** |
| 1.1 | 12.9% | 11.3% | +0.4%p |
| 1.2 | 13.7% | 12.0% | +1.1%p |

→ 베타 추정이 ±0.1 변하면 WACC가 약 ±0.4%p 변동

### 2.4 4단계: Terminal Value (영구가치) 계산

#### 2.4.1 Terminal Value의 개념과 중요성

**Terminal Value (TV)**는 명시적 추정기간 (5~10년) 이후 기업이 창출하는 모든 미래 현금흐름의 현재가치입니다.

**TV가 중요한 이유:**
- DCF 기업가치의 **60~80%를 차지**
- TV 가정 1% 변화 → 전체 가치 10~20% 변동
- M&A 협상에서 가장 논쟁이 되는 부분

#### 2.4.2 TV 계산 방법 1: 영구성장모형 (Gordon Growth Model)

**가장 많이 사용되는 방법입니다.**

**공식:**
```
TV = FCF(n+1) / (WACC - g)
```

**기호:**
- FCF(n+1): 추정 마지막 해 다음 년도의 현금흐름
- g: 영구성장률 (perpetuity growth rate)
- n: 추정 마지막 연도

**전제 조건:**
- WACC > g (만약 g ≥ WACC이면 분모가 0 또는 음수 → 계산 불가)
- 기업이 영속 존속 (going concern)

**A사 TV 계산 (추정기간 5년, 2025~2029년):**

**Step 1: 2029년 (5년차) FCFF 확인**
- FCFF(2029) = 170억원

**Step 2: 영구성장률 (g) 설정**
- g = 2.0% (후술하는 영구성장률 가이드 참조)

**Step 3: 2030년 (6년차) FCFF 계산**
```
FCF(2030) = FCF(2029) × (1 + g)
           = 170 × 1.02
           = 173.4억원
```

**Step 4: Terminal Value 계산**
```
TV = FCF(2030) / (WACC - g)
   = 173.4 / (0.109 - 0.02)
   = 173.4 / 0.089
   = 1,949억원
```

**Step 5: TV의 현재가치 계산**
```
TV의 PV = TV / (1 + WACC)^n
         = 1,949 / (1.109)^5
         = 1,949 / 1.653
         = 1,179억원
```

#### 2.4.3 영구성장률 (g) 설정 가이드

**실무 원칙: 영구성장률은 보수적으로 설정**

**1) 이론적 상한:**
```
g ≤ 명목 GDP 성장률
```

**논리:**
- 한 기업이 경제 전체보다 빠르게 영원히 성장할 수는 없음
- 한국 장기 명목 GDP 성장률: 약 3~4%
- **실무 권장: g = 0~2%**

**2) 산업별 가이드라인:**

| 산업 | 영구성장률 (g) | 근거 |
|------|-------------|------|
| 성숙 제조업 | 0~1% | 저성장, 경쟁 심화 |
| 필수소비재 | 1~2% | 인구성장률 + 인플레이션 |
| IT/소프트웨어 | 1.5~2.5% | 기술 변화 빠르지만 규모 확대 |
| 헬스케어/제약 | 2~3% | 고령화 수혜 (예외적으로 높음) |
| 인프라/유틸리티 | 1~2% | 규제산업, 안정적 |
| 자동차 부품 (A사) | 1~2% | 전기차 전환기, 보수적 |

**3) 영구성장률 vs 추정기간 성장률**

| 구분 | 추정기간 (1~5년차) | 영구성장 (6년차~) |
|------|------------------|-----------------|
| 매출 성장률 | 5~8% | 0~2% |
| 의미 | 성장 모멘텀 반영 | 성숙기 안정 성장 |

**흔한 실수:**
- ❌ "우리는 매년 10% 성장한다" → g = 10% 적용
- ⭕ 추정기간에만 10% 성장, 이후 g = 2% (GDP 성장률 수준)

**4) 인플레이션 고려**

영구성장률은 명목 성장률입니다 (실질성장률 + 인플레이션).

- 한국 장기 인플레이션 목표: 2%
- 실질 성장률 0% 가정 → g = 2% (인플레이션만큼)
- 실질 성장률 1% 가정 → g = 3% (하지만 보수적으로 2% 권장)

**5) g = 0% 시나리오 (매우 보수적)**

일부 보수적 평가에서는 g = 0%를 적용합니다.

- 의미: 영원히 현금흐름이 일정 (성장 없음)
- 적용 경우: 쇠퇴 산업, 단기 수명 기술, M&A 최저가 산정

**A사 시나리오별 TV 비교:**

| 시나리오 | g | FCF(2030) | TV (억원) | TV의 PV | 비고 |
|---------|---|----------|----------|---------|------|
| 보수적 | 0% | 170 | 1,560 | 944 | Base 대비 -20% |
| 중립 | 1% | 172 | 1,715 | 1,038 | -12% |
| **표준** | **2%** | **173** | **1,949** | **1,179** | **Base** |
| 공격적 | 3% | 175 | 2,215 | 1,340 | +14% |

→ g가 1%p 변할 때 TV가 약 13% 변동

**실무 권장: g = 2.0%**
- 인플레이션 수준의 명목 성장 (실질 성장 0%)
- 국제 실무 표준
- 자동차 부품 산업 성숙기 고려

#### 2.4.4 TV 계산 방법 2: Exit Multiple 방식

**언제 사용하는가?**
- VC/PE 투자 (Exit 시점 가치 추정)
- 경기순환 산업 (영구성장 가정 부적절)
- 상장 예정 기업 (IPO Valuation 참고)

**공식:**
```
TV = EBITDA(n) × Exit Multiple
또는
TV = EBIT(n) × Exit Multiple
```

**A사 Exit Multiple 방식 예시:**

**전제:**
- 5년 후 (2029년) PE 펀드 Exit 가정
- 2029년 EBITDA = EBIT 290억 + 감가상각비 66억 = 356억원
- 유사 상장사 평균 EV/EBITDA = 7.5배

**계산:**
```
TV (Exit Value) = 356억 × 7.5 = 2,670억원
```

**Gordon Model과 비교:**
- Gordon Model (g=2%): TV = 1,949억원
- Exit Multiple: TV = 2,670억원
- 차이: +37% (Exit Multiple이 더 높음)

**왜 차이가 나는가?**
- Exit Multiple은 시장 평균 배수 반영 (시장이 프리미엄 지급)
- Gordon은 보수적 성장률 반영

**어느 것을 사용할까?**
- 계속기업 가정 (M&A, 일반 평가) → Gordon Model
- 단기 투자 회수 (VC/PE) → Exit Multiple
- **실무: 두 방법 병행하여 범위 제시**

#### 2.4.5 TV 합리성 검증 (Sanity Check)

**검증 1: TV가 전체 가치의 몇 %인가?**

```
TV 비중 = TV의 PV / (추정기간 FCF PV + TV의 PV) × 100%
```

**정상 범위: 60~80%**
- < 60%: 추정기간이 너무 김 또는 g가 너무 낮음
- > 80%: 추정기간이 너무 짧음 또는 g가 너무 높음

**A사 계산:**
```
추정기간 FCF 현재가치 합계: 450억원 (후술)
TV 현재가치: 1,179억원
전체 기업가치: 1,629억원

TV 비중 = 1,179 / 1,629 = 72.4%
```
→ 정상 범위 내 (60~80%) ✓

**검증 2: Implied Exit Multiple 계산**

TV를 EBITDA 배수로 역산하여 합리성을 검증합니다.

```
Implied Multiple = TV / EBITDA(n)
                 = 1,949 / 356
                 = 5.5배
```

**유사 상장사 EV/EBITDA 평균 7.5배와 비교:**
- 5.5배는 평균보다 낮음 → 보수적 평가 ✓
- 만약 10배 이상이면 g가 과도하게 높을 가능성 검토

### 2.5 5단계: 현재가치 할인 및 기업가치/주식가치 도출

#### 2.5.1 현재가치 할인 (Discounting to Present Value)

각 연도의 FCFF와 TV를 WACC로 할인하여 현재가치를 구합니다.

**할인계수 (Discount Factor) 계산:**
```
DF(t) = 1 / (1 + WACC)^t
```

**A사 DCF 평가 통합 테이블 (단위: 억원)**

| 연도 | t | FCFF | DF (10.9%) | PV |
|------|---|------|-----------|-----|
| 2025 | 1 | 124 | 0.902 | 112 |
| 2026 | 2 | 134 | 0.813 | 109 |
| 2027 | 3 | 146 | 0.733 | 107 |
| 2028 | 4 | 159 | 0.661 | 105 |
| 2029 | 5 | 170 | 0.596 | 101 |
| **추정기간 합계** | | | | **534** |
| **TV** | 5 | 1,949 | 0.596 | **1,162** |
| **기업가치 (EV)** | | | | **1,696억원** |

**계산 설명:**
- DF(1) = 1 / (1.109)^1 = 0.902
- DF(5) = 1 / (1.109)^5 = 0.596
- PV(2025) = 124 × 0.902 = 112억원
- TV의 PV = 1,949 × 0.596 = 1,162억원

**기업가치 (Enterprise Value) = 1,696억원**

#### 2.5.2 기업가치에서 자기자본가치로 전환

**공식:**
```
자기자본가치 (Equity Value) = 기업가치 (EV)
                            - 순부채 (Net Debt)
                            + 비영업자산 (Non-Operating Assets)
```

**각 항목 설명:**

##### (1) 기업가치 (Enterprise Value, EV)
- DCF 계산 결과: 1,696억원
- 영업활동으로 창출하는 가치
- 주주와 채권자 모두의 청구권

##### (2) 순부채 (Net Debt)

```
순부채 = 총차입금 - 현금및현금성자산 - 단기금융상품
```

**A사 2024년 말 재무상태표 기준:**

| 항목 | 금액 (억원) |
|------|-----------|
| **부채** | |
| 단기차입금 | 200 |
| 유동성장기부채 | 100 |
| 장기차입금 | 500 |
| **총차입금** | **800** |
| **자산** | |
| 현금및현금성자산 | 120 |
| 단기금융상품 | 30 |
| **차감 현금** | **150** |
| **순부채** | **650억원** |

**주의사항:**
- **리스부채**: 제외 (또는 별도 판단, IFRS16 적용 시 영업리스도 부채)
- **전환사채(CB)**: 부채로 분류하되, 전환권 가치는 Black-Scholes 모형으로 별도 평가
- **금융업**: 차입금이 영업의 일부이므로 순부채 개념 다름

##### (3) 비영업자산 (Non-Operating Assets)

영업활동과 무관한 자산으로 별도 평가하여 기업가치에 더합니다.

**비영업자산 예시:**
- 투자부동산 (임대용 건물, 유휴 토지)
- 상장주식 투자 (장기금융상품)
- 관계기업/종속기업 투자 (지분법적용투자주식)
- 매각 예정 자산

**평가 방법:**
- 부동산: 감정평가액 (또는 공시지가 × 할증률)
- 상장주식: 평가기준일 종가 × 주식수
- 비상장주식: 별도 DCF 또는 장부가치
- 기타: 공정가치 (장부가치 참조)

**A사 비영업자산:**

| 항목 | 장부가액 | 공정가치 | 비고 |
|------|---------|---------|------|
| 투자부동산 (임대 건물) | 60 | 80 | 감정평가 |
| 상장주식 (㉮전자 1%) | 15 | 20 | 시가 |
| 유휴 토지 | 15 | 20 | 공시지가 × 1.3 |
| **합계** | **90** | **120억원** | |

→ 비영업자산 공정가치 120억원 가산

#### 2.5.3 자기자본가치 및 주당가치 계산

**A사 자기자본가치 계산:**

```
자기자본가치 = 기업가치 - 순부채 + 비영업자산
             = 1,696 - 650 + 120
             = 1,166억원
```

**주당가치 계산:**

```
주당가치 = 자기자본가치 / 발행주식수
```

**A사:**
- 발행주식수: 200만주 (보통주 200만주)
- 주당가치 = 1,166억원 / 200만주 = **58,300원/주**

**희석주식 반영 (만약 스톡옵션이 있다면):**

```
완전희석주식수 = 발행주식 + 잠재주식 (옵션, CB, BW 전환 가정)
희석주당가치 = 자기자본가치 / 완전희석주식수
```

예: 스톡옵션 20만주 → 완전희석 220만주
- 희석주당가치 = 1,166억 / 220만주 = 53,000원/주

#### 2.5.4 가치 범위 (Valuation Range) 제시

실무에서는 단일 값이 아닌 범위를 제시합니다.

**민감도 분석 결과:**

| 시나리오 | WACC | g | 자기자본가치 (억원) | 주당가치 (원) |
|---------|------|---|-----------------|-----------|
| Downside | 12.0% | 0% | 950 | 47,500 |
| Base | 10.9% | 2% | 1,166 | 58,300 |
| Upside | 10.0% | 3% | 1,420 | 71,000 |

**확률 가중 평균:**
- Downside 25%, Base 50%, Upside 25%
```
기대가치 = 950×0.25 + 1,166×0.5 + 1,420×0.25
         = 238 + 583 + 355
         = 1,176억원
```

**최종 평가 범위 제시:**

**"A사 자기자본가치는 1,050억원 ~ 1,300억원 범위로 평가되며, 중심값은 1,166억원입니다."**

**주당가치: 52,500원 ~ 65,000원, 중심값 58,300원**

### 2.6 DCF 평가 체크리스트 및 실무 가이드

#### 2.6.1 단계별 검증 체크리스트

**□ 재무제표 분석 단계**
- [ ] 과거 3~5개년 재무제표 확보 (감사받은 재무제표 권장)
- [ ] 비경상적 손익 제거 완료
- [ ] 정규화 EBIT 계산 및 근거 문서화
- [ ] 감가상각비 정확히 추출

**□ FCFF 추정 단계**
- [ ] NOPLAT 계산 (실효세율 적용)
- [ ] CapEx 추정 (과거 평균 또는 사업계획 반영)
- [ ] 순운전자본 변동액 계산 (매출 증가에 비례)
- [ ] 5개년 FCFF 일관성 검토

**□ WACC 계산 단계**
- [ ] 무위험이자율 확인 (국고채 3년 또는 10년)
- [ ] 베타 산출 완료 (상장사: Historical, 비상장사: Comparable 5단계)
- [ ] ERP = 8.5% 적용
- [ ] 타인자본비용 계산 (이자비용/총차입금 또는 신규 금리)
- [ ] 목표 자본구조 설정 (유사사 평균)
- [ ] WACC 최종값 합리성 검토 (8~12% 범위)

**□ Terminal Value 단계**
- [ ] 영구성장률 설정 (0~2% 권장)
- [ ] TV = FCF(n+1) / (WACC - g) 계산
- [ ] TV 비중 확인 (60~80% 적정)
- [ ] Implied Exit Multiple 계산 및 유사사 비교

**□ 최종 가치 산출 단계**
- [ ] 각 연도 FCFF 현재가치 할인
- [ ] 기업가치 (EV) 합산
- [ ] 순부채 차감 (총차입금 - 현금)
- [ ] 비영업자산 가산
- [ ] 자기자본가치 도출
- [ ] 주당가치 계산

**□ 민감도 분석 및 보고**
- [ ] WACC 민감도 (±0.5%p, ±1.0%p)
- [ ] 영구성장률 민감도 (0%, 1%, 2%, 3%)
- [ ] 시나리오 분석 (Base, Upside, Downside)
- [ ] 가치 범위 제시

#### 2.6.2 흔한 실수와 해결법

**실수 1: 영구성장률 과대 설정**
- 문제: g = 5% → GDP 성장률(3%) 초과
- 해결: g ≤ 2% 원칙, Implied Multiple로 검증

**실수 2: CapEx와 감가상각비 불일치**
- 문제: 성장기인데 CapEx = 감가상각비
- 해결: 성장기 CapEx > 감가상각비 (1.5~2배)

**실수 3: 운전자본 변동 무시**
- 문제: 매출 증가하는데 ΔNWC = 0
- 해결: NWC/매출액 비율 과거 평균 유지

## 섹션 3: DCF 평가법의 실전 활용

### 3.1 실무 적용 사례

#### 사례 1: M&A 가격 결정

**배경:**
- 유통기업 D사가 식품제조사 E사 (상장사, 시가총액 3,000억) 인수 검토
- 시너지 반영 필요

**DCF 적용:**
1. Stand-alone 가치: DCF → 3,200억원
2. 시너지 가치: 유통망 활용으로 연 EBIT 40억 증가 → 영구가치 400억
3. 인수가격: 3,200억 + 400억 × 50% = 3,400억원 (프리미엄 13%)

#### 사례 2: 스타트업 투자 (Series A)

**배경:**
- SaaS 스타트업 B사, ARR 30억, 아직 흑자 아님

**VC Method + DCF:**
1. Exit (5년 후 IPO): ARR 900억 × EV/ARR 10배 = 9,000억
2. VC 요구수익률 40% 할인: 9,000 / (1.4)^5 = 1,669억
3. 생존확률 70% 반영: 1,669 × 0.7 = 1,168억원

#### 사례 3: 손상평가 (Impairment)

**배경:**
- 신약 개발 자회사 (장부 1,000억), 임상 실패

**DCF 재평가:**
1. 사업계획 하향: FCFF 50억/년 수준
2. 할인율 상향: WACC 14% (위험 증가)
3. 사용가치: 50억 / 0.14 = 357억원
4. 손상차손: 643억원 인식

|------|------------------|------|---------|

| 상대가치 (PER) | 1,100 | 30% | 330 |

### 3.3 주의사항

**DCF의 한계:**
- 미래 예측 불확실성
- 가정 의존성 (WACC, g 민감)
