## 1차 계획: 평가 서비스 플랫폼 (최우선 목표)

### 1차 목표
**"5대 평가법 엔진 + AI/ML + 판매 사이트 + 시뮬레이터"**

고객이 웹사이트에서:
1. 무료 시뮬레이터로 간단히 체험
2. 정식 평가 의뢰 (자료 업로드)
3. 결제
4. AI/ML 기반 자동 평가
5. 전문가 검토
6. 보고서 다운로드

---

### Phase 1: 5대 평가엔진 + AI/ML 기능

#### 1.1 기본 계산 엔진 (이전 문서 참조)
- DCF, 상대가치, 자산가치, 자본시장법, 상증세법
- 보고서 자동 생성 (PDF, Excel)

#### 1.2 AI/ML 기능 추가

**A. 재무제표 자동 추출 (OCR + NLP)**
```
고객 업로드: 재무제표 PDF/이미지
    ↓
Tesseract OCR / AWS Textract
    ↓
숫자/표 인식 → 재무제표 JSON 변환
    ↓
LLM으로 계정과목 자동 매핑
    ↓
Database 저장
```

**구현:**
- Tesseract OCR (오픈소스)
- OpenAI GPT-4 Vision API (표 인식)
- 정확도 95% 이상 목표

**B. 비교기업 자동 추천 (ML 모델)**
```
입력: 대상기업 (산업, 매출, 수익성)
    ↓
유사도 알고리즘 (코사인 유사도, KNN)
    ↓
후보 100개 → 최적 10개 추천
    ↓
배수 자동 계산
```

**구현:**
- scikit-learn (KNN, 클러스터링)
- 산업 코드 + 재무비율 벡터화
- 실시간 추천 (< 2초)

**C. 미래 실적 예측 (시계열 분석)**
```
입력: 과거 5년 재무 데이터
    ↓
ARIMA / LSTM 시계열 모델
    ↓
향후 3~5년 매출/이익 예측
    ↓
신뢰구간 함께 제공
```

**구현:**
- statsmodels (ARIMA)
- TensorFlow/PyTorch (LSTM)
- 경제지표 반영 (GDP, 금리, 환율)

**D. 리스크 자동 평가 (NLP 감성분석)**
```
입력: 뉴스, 공시, 소송 정보
    ↓
NLP 감성분석 (긍정/부정/중립)
    ↓
리스크 점수 (1~100)
    ↓
보고서에 자동 반영
```

**구현:**
- KoBERT (한국어 NLP)
- 네이버 뉴스 크롤링
- DART 공시 API 연동

**E. 보고서 자동 작성 (Claude API)**
```
입력: 평가 결과 JSON + 재무데이터 + 산업분석
    ↓
Anthropic Claude API (200K 토큰 컨텍스트)
    ↓
"1. 회사 개요 (2,000자)
 2. 산업 및 경쟁 환경 분석 (3,000자)
 3. 재무 현황 분석 (5,000자)
 4. 평가 방법론 및 가정 (4,000자)
 5. 평가 결과 (3,000자)
 6. 민감도 분석 (2,000자)
 7. 결론 및 유의사항 (2,000자)"
 → 총 20,000자 이상 자동 생성
    ↓
템플릿에 삽입 → PDF (50~100페이지)
```

**구현:**
- Anthropic Claude 3.5 Sonnet API (우선)
  - 이유 1: 200K 토큰 컨텍스트 (GPT-4의 13배)
  - 이유 2: 긴 문서 작성에 최적화
  - 이유 3: 정확한 재무 분석
  - 이유 4: 한국어 품질 우수
- 프롬프트 엔지니어링 (체계적 구조)
- 전문가 최종 검토 필수

### Phase 2: 판매 사이트 구축

#### 2.1 사이트 구조

```
홈페이지
├── 메인 (서비스 소개, 가격표)
├── 시뮬레이터 (무료 간편 계산)
├── 평가 의뢰 (로그인 필요)
│   ├── 1단계: 평가 목적 선택 (M&A, 투자유치, 상속증여, 소송, 내부관리)
│   ├── 2단계: 기업 정보 입력
│   ├── 3단계: 자료 업로드 (재무제표, 정관, 주주명부 등)
│   ├── 4단계: 평가법 선택 (단일/복수)
│   ├── 5단계: 결제
│   └── 6단계: 진행 현황 조회
├── 내 평가 (과거 의뢰 내역, 보고서 다운로드)
├── 고객센터 (FAQ, 문의)
└── 로그인/회원가입
```

#### 2.2 회원 시스템

**회원 등급:**
| 등급 | 가입 조건 | 혜택 |
|------|---------|------|
| 일반 | 이메일 인증 | 시뮬레이터 무료 |
| 기업 | 사업자등록번호 인증 | 할인 쿠폰 |
| VIP | 연간 10건 이상 | 20% 할인 |

**회원 정보:**
- 이메일, 비밀번호 (암호화)
- 이름, 연락처
- 사업자등록번호 (선택)
- 과거 의뢰 내역

#### 2.3 가격 체계

**기본 요금제:**
| 평가법 | 일반 기업 | 상장사 | 소요시간 | 페이지 |
|-------|---------|-------|---------|-------|
| **단일 평가법** | 200만원 | 300만원 | 3영업일 | 30~50p |
| DCF | 250만원 | 400만원 | 5영업일 | 50~80p |
| 상대가치 | 150만원 | 250만원 | 2영업일 | 20~40p |
| 자산가치 | 200만원 | 300만원 | 3영업일 | 30~50p |
| 자본시장법 | 180만원 | - | 3영업일 | 30~50p |

**할인:**
- 첫 의뢰: 20% 할인
- 연간 계약 (10건 이상): 30% 할인
- 추천인 코드: 각 10% 할인

#### 2.4 자료 접수 프로세스

**필수 제출 자료:**

| 평가 목적 | 필수 자료 | 선택 자료 |
|---------|---------|---------|
| **M&A** | • 3개년 재무제표<br>• 정관<br>• 주주명부 | • 사업계획서<br>• 실사자료<br>• 계약서(LOI) |
| **투자유치** | • 3개년 재무제표<br>• 사업계획서<br>• 정관 | • 특허/IP 자료<br>• 고객 리스트 |
| **상속/증여** | • 3개년 재무제표<br>• 법인세 신고서<br>• 부동산 등기부 | • 감정평가서<br>• 특수관계자 거래 내역 |
| **소송** | • 5개년 재무제표<br>• 소송 관련 자료 | • 감정평가서<br>• 전문가 의견서 |

**자료 업로드:**
- 파일 형식: PDF, Excel, 이미지 (JPG, PNG)
- 최대 용량: 100MB
- 암호화 전송 (TLS 1.3)
- 자동 OCR 처리

#### 2.5 결제 시스템

**결제 수단:**
- 신용카드 (국내/해외)
- 계좌이체
- 가상계좌
- 법인카드
- 나중에 결제 (법인 고객, 여신 승인 시)

**결제 프로세스:**
```
평가 의뢰서 작성 완료
    ↓
견적서 자동 생성 (PDF)
    ↓
결제 페이지 (PG사 연동)
    ↓
결제 완료 → 이메일/SMS 알림
    ↓
평가 작업 시작
```

**PG사 연동:**
- 토스페이먼츠 (국내)
- Stripe (해외)
- 수수료: 3.3%

**환불 정책:**
- 작업 시작 전: 100% 환불
- 작업 시작 후 (초안 전): 50% 환불
- 초안 전달 후: 환불 불가

#### 2.6 평가 진행 프로세스

**고객 View:**
```
의뢰 → [접수 완료] → [자료 검토] → [평가 진행 중 40%]
    → [평가 진행 중 70%] → [초안 완성] → [최종 검토]
    → [보고서 완료] → 다운로드
```

**실시간 진행률 표시:**
- 대시보드에서 현재 진행 상태 확인
- 예상 완료일 표시
- 담당자 연락처

**알림:**
- 이메일: 단계별 알림
- SMS: 최종 완료 시
- 카카오톡: 선택 사항

#### 2.7 보고서 다운로드

**다운로드 방식:**
- 웹사이트 로그인 → 내 평가 → 다운로드 버튼
- 이메일 첨부 (보안 링크)
- 비밀번호 보호 (고객 지정)

**보고서 형식:**
- PDF (기본)
- Excel (재무모델 포함)
- PPT (발표자료, 옵션)

**보관 기간:**
- 3년간 재다운로드 가능
- 이후 별도 요청 시 복원 (수수료 5만원)

**워터마크:**
- "본 보고서는 [회사명]에 의뢰하여 작성되었습니다"
- 날짜, 평가 ID

### Phase 3: 시뮬레이터 (무료 체험)

#### 3.1 시뮬레이터 목적

**목표:**
- 잠재 고객 유입 (Funnel Top)
- 서비스 가치 체험
- 리드 생성 (이메일 수집)

**컨셉:**
"30초 만에 우리 회사 가치 알아보기"

#### 3.2 5대 평가법 간편 시뮬레이터

**A. DCF 간편 계산기**

**입력 (5개 항목):**
1. 올해 매출액 (억원)
2. 올해 영업이익률 (%)
3. 향후 3년 평균 성장률 (%)
4. 업종 선택 (드롭다운)
5. 순부채 (억원)

**자동 설정 (사용자 보이지 않음):**
- WACC: 업종별 평균 (10~15%)
- 영구성장률: 3%
- FCF 전환율: 업종별 평균
- 평가 기간: 5년

**출력:**
```
┌─────────────────────────────────┐
│  예상 기업가치: 500억원          │
│  예상 주당가치: 50,000원         │
│                                 │
│  [상세 보고서 받기 →]           │
│  (이메일 입력 → 10% 할인쿠폰)   │
└─────────────────────────────────┘
```

**B. 상대가치 간편 계산기**

**입력:**
1. 올해 매출액
2. 올해 순이익
3. 업종 선택

**자동 계산:**
- 해당 업종 평균 PER, PSR 적용
- 비교기업 3개 자동 표시

**출력:**
```
┌─────────────────────────────────┐
│  PER 방식: 400억원              │
│  PSR 방식: 450억원              │
│  평균: 425억원                  │
│                                 │
│  비교기업: 삼성전자, LG전자, SK │
└─────────────────────────────────┘
```

**C. 자산가치 간편 계산기**

**입력:**
1. 총 자산 (재무상태표)
2. 총 부채
3. 부동산 보유 여부 (Y/N)
4. 부동산 장부가액 vs 시가 (배수)

**출력:**
```
┌─────────────────────────────────┐
│  장부 순자산: 300억원            │
│  조정 순자산: 500억원            │
│  (부동산 재평가 +200억)         │
└─────────────────────────────────┘
```

**D. 상증세법 간편 계산기**

**입력:**
1. 최근 3년 평균 순이익
2. 순자산 (재무상태표)
3. 발행주식수
4. 주주 유형 (최대주주/소액주주)

**출력:**
```
┌─────────────────────────────────┐
│  1주당 평가액: 15,000원          │
│  할증 후: 18,000원 (최대주주)    │
│                                 │
│  100만주 증여 시                │
│  증여세 예상액: 35억원           │
└─────────────────────────────────┘
```

**E. 종합 시뮬레이터**

**입력:**
- 위 4가지 평가법 최소 입력값
- 10개 항목

**출력 (Football Field Chart):**
```
DCF         [████████████████    ] 500억
상대가치    [██████████████      ] 450억
자산가치    [██████████          ] 350억
상증세법    [███████████████     ] 480억
           ─┼────┼────┼────┼────┼─
          300   400   500   600  700

추천 평가 범위: 450~520억원
```

#### 3.3 시뮬레이터 기능

**인터랙티브 요소:**
- 슬라이더 (숫자 입력 대신)
- 실시간 계산 (입력 즉시 결과 업데이트)
- 차트 애니메이션
- 민감도 분석 (성장률 ±5% 시 가치 변화)

**공유 기능:**
- 결과 이미지 저장
- SNS 공유 (카카오톡, 페이스북)
- URL 복사 (결과 저장)

**리드 생성:**
- 계산 후 "상세 보고서 받기" 버튼
- 이메일 입력 → 10% 할인 쿠폰 발송
- 영업팀 자동 알림

#### 3.4 시뮬레이터 UI/UX

**디자인 원칙:**
- 모바일 최적화 (반응형)
- 3분 이내 완료
- 전문 용어 최소화 (툴팁 설명)

**예시 플로우:**
```
[시뮬레이터 시작]
    ↓
Step 1: 평가법 선택 (5개 중 1개)
    ↓
Step 2: 간단한 정보 입력 (5개 항목)
    ↓

[상세 보고서 받기] or [다시 계산]
```

## 2차 계획: 데이터 플랫폼 (평가 DB 구축)

### 2차 목표
**"평가 데이터 축적 → 기업 랭킹 → 벤치마크 서비스"**

### 주요 기능

#### 2.1 평가 데이터 축적
- 1차에서 수행한 모든 평가 결과 DB화
- 익명화된 데이터 (기업명 제외)
- 산업별, 규모별, 평가법별 분류

**축적 데이터:**
- 재무비율 (ROE, ROA, 부채비율 등)
- 평가 배수 (PER, PBR, EV/EBITDA)
- WACC, 베타
- 성장률, 이익률

#### 2.2 기업 랭킹 시스템

**종합 점수 (100점 만점):**
- 재무건전성: 30점
- 수익성: 25점
- 성장성: 25점
- 안정성: 20점

**산업별 랭킹:**
- 제조업 TOP 100
- IT/소프트웨어 TOP 100
- 바이오/제약 TOP 100

**공개 방식:**
- 무료: 업종별 TOP 10
- 유료: 전체 랭킹, 상세 분석

#### 2.3 벤치마크 서비스

**기업 회원 대상:**
- "우리 회사 vs 동종 업계 비교"
- 재무비율 벤치마크
- 평가 배수 벤치마크
- 월 9.9만원 구독

**투자자 대상:**
- 산업 리포트 (분기별)
- 밸류에이션 트렌드
- 월 29만원 구독

## 3차 계획: 투자 매칭 플랫폼

### 3차 목표
**"자금 필요 기업 ↔ 투자자 매칭"**

### 주요 기능

#### 3.1 기업 측 (자금 수요)

**프로필 등록:**
- 기업 소개
- 사업 계획서
- 평가 보고서 (1차에서 작성)
- 투자 조건 (금액, 지분율, 용도)

**공개 범위:**
- 공개: 모든 투자자 열람 가능
- 비공개: 승인된 투자자만 열람

#### 3.2 투자자 측 (자금 공급)

**투자자 유형:**
- 엔젤 투자자
- 벤처캐피탈
- 기업 전략적 투자자 (CVC)
- 사모펀드 (PEF)

**검색 기능:**
- 산업, 지역, 투자 단계 필터
- 밸류에이션 범위
- 랭킹 점수

#### 3.3 매칭 알고리즘

**AI 기반 추천:**
- 투자자 과거 투자 이력 분석
- 기업 특성과 투자자 선호 매칭
- 성사 확률 예측

**매칭 프로세스:**
```
기업 프로필 등록
    ↓
AI 추천 (TOP 10 투자자)
    ↓
기업 → 투자자에게 제안서 발송
    ↓
투자자 관심 표명
    ↓
플랫폼 내 메시지 / 미팅 주선
    ↓
투자 계약 (오프라인)
    ↓
플랫폼 수수료 (성공 보수 3~5%)
```

#### 3.4 수익 모델

- 기업 프로필 등록: 무료
- 투자자 멤버십: 월 50만원
- 투자 성사 시 수수료: 투자금의 3~5%
- 추가 서비스: 법률 자문, 실사 대행
