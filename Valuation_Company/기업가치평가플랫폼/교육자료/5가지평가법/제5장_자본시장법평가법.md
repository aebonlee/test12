# 제5장 | 자본시장법 평가법 (본질가치)

## 섹션 1: 자본시장법 평가법이란 무엇인가

### 1.1 정의와 핵심 개념

**본질가치(Intrinsic Value)**는 자본시장법에서 정한 법적 평가 방법입니다. 기업의 자산가치와 수익가치를 정해진 비율로 가중평균하여 계산합니다. 법적 근거는 자본시장과 금융투자업에 관한 법률 시행령 **제176조의5**(주식매수청구권 행사가액의 산정)와 **제176조의6**(합병 등의 경우 주식매수청구권 행사가액의 특례)에 있습니다.

본질가치의 핵심은 **자산가치 40% + 수익가치 60%**의 가중구조입니다. 이는 기업의 현재 자산 상태와 미래 수익창출능력을 균형있게 반영하되, 계속기업(going concern) 관점에서 수익가치에 더 높은 가중치를 부여하는 방식입니다. 이러한 복합평가법은 단일 평가법의 한계를 보완하고, 법적 분쟁 시 객관성과 공정성을 확보하기 위해 고안되었습니다.

구체적인 산식은 다음과 같습니다:
```
본질가치 = (자산가치 × 1 + 수익가치 × 1.5) ÷ 2.5
```

이는 자산가치와 수익가치에 1:1.5의 가중치를 부여한 후 그 합을 2.5로 나누어 평균한 것으로, 결과적으로 자산가치 40%, 수익가치 60%의 가중평균과 동일합니다. 이 비율은 법령으로 고정되어 있어, 평가자의 자의적 판단을 배제합니다.

### 1.2 왜 자본시장법 평가법이 필요한가

자본시장법 평가법은 단순한 이론적 평가법이 아니라 **법적 의무사항**입니다. 다음의 경우 반드시 본질가치 평가를 수행해야 합니다:

첫째, **주식매수청구권 행사 시** 비상장주식의 매수가격을 결정할 때 사용됩니다. 합병, 분할, 영업양도 등 중요한 구조조정 상황에서 반대주주가 주식매수청구권을 행사하면, 회사는 본질가치에 기초하여 공정한 매수가격을 제시해야 합니다.

둘째, **M&A 거래**에서 외부평가의견서 작성이 요구될 때 적용됩니다. 특히 상장사 간 합병, 주식교환, 자산양수도 등의 거래에서 금융감독원과 증권선물위원회는 본질가치 평가를 포함한 외부평가기관의 공정성 의견서를 요구합니다.

셋째, **Pre-IPO 단계**에서 기업가치 산정 및 제3자 배정 증자 시 발행가액의 적정성을 입증하는 용도로 활용됩니다. 이는 기존 주주와 신규 투자자 간 이해상충을 방지하고, 불공정거래 시비를 예방합니다.

본질가치 평가가 중요한 이유는 **이해관계자 보호**와 **시장 신뢰성 제고**에 있습니다. 법적으로 표준화된 방법론을 사용함으로써 대주주와 소액주주, 매도자와 매수자 간 정보 비대칭과 협상력 차이로 인한 불공정을 완화합니다. 또한 법원이나 감독기관이 거래가격의 적정성을 판단할 때 명확한 기준을 제공하여, 분쟁 발생 시 객관적 해결 근거가 됩니다.

### 1.3 다른 평가법과의 차이

본질가치 평가법은 다른 평가법과 명확한 차이점을 갖습니다.

**DCF(현금흐름할인법)와의 비교**: DCF는 미래 현금흐름을 추정하고 적정 할인율로 현재가치화하는 순수한 수익접근법입니다. 반면 본질가치는 자산가치와 수익가치를 법정 비율로 혼합합니다. DCF가 이론적으로 가장 정교하지만 가정에 따라 결과가 크게 달라지는 반면, 본질가치는 표준화된 계산으로 객관성을 확보합니다.

**상대가치평가(Multiples)와의 비교**: PER, PBR, EV/EBITDA 등 상대가치평가는 비교기업의 시장배수를 활용합니다. 이는 시장 상황을 즉각 반영하지만, 비교기업 선정의 자의성과 시장 왜곡의 영향을 받습니다. 본질가치는 개별기업의 재무제표를 기초로 하므로 외부 시장상황에 덜 민감하며, 비상장회사 평가에 더 적합합니다.

**상증세법 평가법과의 비교**: 상속세 및 증여세법 시행령 제54조의 비상장주식 평가와 구조가 유사하지만, 적용 범위와 세부 계산이 다릅니다. 상증세법은 세금 과세를 위한 평가로서 1주당 순자산가치와 1주당 순손익가치를 산술평균하거나 가중평균합니다. 반면 자본시장법의 본질가치는 거래 당사자 간 공정한 가격 결정을 목적으로 하며, 수익가치에 더 높은 가중치(60%)를 부여합니다.

**본질가치평가의 특징**은 세 가지로 요약됩니다: **법적 강제성**(자본시장법 적용 상황에서 필수), **표준화된 방법론**(자산가치 40% + 수익가치 60% 고정), **분쟁 해결 기준**(법원과 감독기관이 인정하는 객관적 기준). 이는 실무에서 다른 평가법과 함께 사용되며, 교차 검증(Cross-check) 용도로 활용됩니다.

## 섹션 2: 자본시장법 평가법의 계산 방법

### 2.1 계산 공식과 단계별 프로세스

본질가치 계산은 법령에서 정한 명확한 6단계 프로세스를 따릅니다. 각 단계별 상세 계산 방법을 숙지하면 실제 평가보고서 작성이 가능합니다.

#### 전체 공식

기본 공식은 다음과 같습니다:

```
본질가치 = (자산가치 × 1 + 수익가치 × 1.5) ÷ 2.5
```

이를 백분율로 표현하면:
```
본질가치 = 자산가치 × 40% + 수익가치 × 60%
```

#### 1단계: 재무제표 수집 및 기초 분석

**1-1. 재무제표 3개년 확보**
- 재무상태표(대차대조표)
- 손익계산서
- 현금흐름표
- 자본변동표
- 주석사항 및 감사보고서

**1-2. 재무제표 정규화(Normalization)**
일회성 항목 식별 및 조정:
```
조정 대상 항목:
+ 일회성 이익: 자산처분이익, 투자자산처분이익, 보험차익
- 일회성 손실: 재해손실, 소송배상금, 구조조정비용
± 특수관계자 거래 정상가격 조정
± 계열사 매출/매입의 시가 조정
```

**1-3. 기준일 설정**
평가기준일: 통상 합병 이사회 결의일 또는 주주총회 소집 결정일의 직전 월말

#### 2단계: 자산가치 계산 (순자산가치법)

**2-1. 장부상 순자산 파악**
```
장부상 순자산(자기자본) = 총자산 - 총부채
```

**2-2. 자산 항목별 시장가치 재평가**

| 자산 항목 | 평가 방법 | 조정 산식 |
|---------|---------|---------|
| **토지** | 감정평가 또는 개별공시지가 | 시장가치 = 개별공시지가 ÷ 현실화율(0.7~0.8) × 위치계수 |
| **건물** | 재조달원가법 | 시장가치 = (재조달원가 - 감가상각) × 건물상태계수 |
| **기계장치** | 기술평가 또는 중고시장가 | 시장가치 = 신품가격 × (1 - 경제적 내용연수/총내용연수) |
| **재고자산** | 순실현가능가치 | 시장가치 = 판매예상가격 - 판매비용 - 정상이익 |
| **매출채권** | 회수가능액 | 시장가치 = 장부가액 - 대손충당금(연체율 기준 재계산) |
| **투자자산** | 공정가치 | 상장주식: 평가기준일 종가, 비상장: 별도 평가 |
| **무형자산** | 개별 평가 | 특허권/상표권: Relief-from-Royalty법, 영업권: Excess Return법 |

**2-3. 부채 항목별 시장가치 재평가**

```
장기차입금 시장가치 = Σ(미래 원리금 상환액 ÷ (1 + 시장이자율)^n)
전환사채 시장가치 = 부채요소 + 전환권 가치(Black-Scholes 모형)
우발부채 = 소송금액 × 패소확률 × 현재가치할인계수
```

**2-4. 비계상 항목 반영**

```
+ 비계상 자산: 자체 개발 무형자산, 우발자산(소송 승소 가능성)
- 비계상 부채: 퇴직금 추가 부담액, 환경복구 의무, 보증채무
± 리스부채(IFRS 16 적용 시 재계산)
```

**2-5. 최종 자산가치 산출**

```
조정 순자산가치 = 장부상 순자산
                + 자산 재평가 증가액
                - 자산 재평가 감소액
                - 부채 재평가 증가액
                + 부채 재평가 감소액
                + 비계상 자산
                - 비계상 부채

1주당 자산가치 = 조정 순자산가치 ÷ 발행주식총수
```

**계산 예시: (주)테크밸리 자산가치**

| 항목 | 장부가액 | 시장가치 조정 | 조정 후 가액 | 비고 |
|-----|---------|-------------|-----------|------|
| 총자산 | 150억원 | | | |
| - 토지 | 30억원 | +15억원 | 45억원 | 감정평가 |
| - 건물 | 20억원 | +5억원 | 25억원 | 재조달원가 |
| - 기계 | 15억원 | -3억원 | 12억원 | 중고시장가 |
| - 재고 | 10억원 | -1억원 | 9억원 | 순실현가능가치 |
| - 매출채권 | 20억원 | -2억원 | 18억원 | 대손율 상향 |
| - 무형자산(영업권) | 0억원 | +8억원 | 8억원 | Excess Return법 |
| - 기타자산 | 55억원 | 0억원 | 55억원 | |
| 총부채 | 70억원 | | | |
| - 차입금 | 50억원 | +2억원 | 52억원 | 시가평가 |
| - 우발부채 | 0억원 | +3억원 | 3억원 | 소송 패소 가능성 |
| - 기타부채 | 20억원 | 0억원 | 20억원 | |
| **장부상 순자산** | **80억원** | | | |
| **조정 순자산** | | | **97억원** | 80 + 22(자산증) - 5(부채증) |
| **발행주식수** | | | 100만주 | |

#### 3단계: 수익가치 계산 (수익환원법)

**3-1. 정상수익 산정**

수익가치는 수익환원법(Income Capitalization Method)을 사용합니다:

```
수익가치 = 정상수익 ÷ 자본환원율
```

**정상수익 계산 공식: 3년 가중평균**

```
정상수익 = (N-2년 순이익 × 1 + N-1년 순이익 × 2 + N년 순이익 × 3) ÷ 6
```

또는 단순평균(3년 실적 변동성이 낮을 경우):
```
정상수익 = (N-2년 순이익 + N-1년 순이익 + N년 순이익) ÷ 3
```

**일회성 손익 조정**

```
조정 순이익 = 당기순이익
            - 일회성 이익(자산처분이익, 보험차익, 법인세 환급)
            + 일회성 손실(재해손실, 소송비용, 구조조정비용)
            ± 특수관계자 거래 정상가격 조정
```

**계산 예시: (주)테크밸리 정상수익**

| 연도 | 보고 순이익 | 일회성 조정 | 조정 순이익 | 가중치 | 가중 이익 |
|-----|----------|----------|----------|-------|---------|
| N-2년(2022) | 8억원 | -1억원(토지처분) | 7억원 | 1 | 7억원 |
| N-1년(2023) | 10억원 | +2억원(소송손실) | 12억원 | 2 | 24억원 |
| N년(2024) | 12억원 | -1억원(보험차익) | 11억원 | 3 | 33억원 |
| **합계** | | | | 6 | 64억원 |
| **가중평균 정상수익** | | | | | **10.67억원** |

업종 평균 수익률과 비교 검증:
```
제조업 평균 ROE: 8%
테크밸리 ROE: 10.67억원 ÷ 97억원(순자산) = 11.0% → 업종 평균 상회(합리적)
```

**3-2. 자본환원율(Capitalization Rate) 산정**

자본환원율은 수익가치 계산의 핵심 변수로, 다음 3가지 방법 중 하나를 선택합니다.

**방법 1: 무위험수익률 + 위험프리미엄**

```
자본환원율 = 무위험수익률 + 위험프리미엄 + 규모프리미엄
```

구성요소:
- **무위험수익률**: 3년 만기 국고채 수익률(예: 3.5%)
- **위험프리미엄**: 업종별 차이
  - 제조업: 6~8%
  - IT/소프트웨어: 8~12%
  - 바이오/제약: 12~15%
  - 부동산/건설: 7~10%
- **규모프리미엄**: 중소기업 추가 리스크
  - 대기업: 0%
  - 중견기업: 1~2%
  - 중소기업: 2~3%

**계산 예시:**
```
자본환원율 = 3.5%(국고채) + 7%(제조업 위험) + 2%(중소기업) = 12.5%
```

**방법 2: CAPM(자본자산가격결정모형) 활용**

```
자본환원율 = 무위험수익률 + β × 시장위험프리미엄
```

- **무위험수익률**: 3년 국고채 수익률 3.5%
- **β(베타)**: 업종별 평균 베타
  - 한국거래소 발표 업종별 베타 사용
  - 제조업: 0.9~1.1
  - IT: 1.2~1.5
  - 바이오: 1.5~2.0
- **시장위험프리미엄**: 한국 주식시장 역사적 평균 7~8%

**계산 예시:**
```
자본환원율 = 3.5% + 1.0(제조업 β) × 7.5%(시장프리미엄) = 11.0%
```

**방법 3: 업종별 평균 자본비용 적용**

한국은행 또는 신용평가사가 발표하는 업종별 평균 자본비용을 직접 사용:

|-----|-------------|
| 제조업 | 10~12% |
| IT/소프트웨어 | 12~15% |
| 바이오/제약 | 15~18% |
| 유통/서비스 | 11~13% |
| 건설/부동산 | 9~11% |

**실무 권고사항:**
- 3가지 방법을 모두 계산하여 범위 확인
- 중간값 또는 보수적 값(높은 환원율) 선택
- 평가보고서에 계산 근거 명시

**3-3. 수익가치 최종 산출**

```
수익가치 = 정상수익 ÷ 자본환원율
```

**계산 예시: (주)테크밸리**

방법별 자본환원율 비교:
- 방법 1(무위험+프리미엄): 12.5%
- 방법 2(CAPM): 11.0%
- 방법 3(업종평균): 11.0%

```
1주당 수익가치 = 93억원 ÷ 100만주 = 9,300원
```


또는

### 2.2 필요한 데이터

본질가치 평가를 위해서는 다음 자료가 필수적입니다:

**1. 재무제표 3개년**
- 재무상태표(대차대조표): 자산, 부채, 자본의 구성
- 손익계산서: 매출, 비용, 당기순이익
- 현금흐름표: 영업, 투자, 재무활동 현금흐름
- 자본변동표: 자본금, 이익잉여금 변동
- 주석사항: 회계정책, 우발부채, 특수관계자 거래

최근 3년치 재무제표가 필요한 이유는 정상수익을 산정하고 일회성 요인을 제거하기 위함입니다. 외부감사를 받은 재무제표를 사용해야 신뢰성이 확보됩니다.

**2. 자산 감정평가 자료**
- 토지, 건물의 감정평가서(공인 감정평가법인 작성)
- 기계장치, 설비의 기술평가서
- 재고자산의 시장가치 조사
- 무형자산(특허, 상표권, 영업권) 평가보고서

부동산 등 중요 자산은 장부가액이 시장가치를 반영하지 못하는 경우가 많으므로, 전문 감정평가기관의 평가가 필수입니다.

**3. 업종별 할인율 또는 자본환원율**
- 한국은행 발표 업종별 자본수익률
- 금융감독원 또는 신용평가사 제공 할인율 자료
- 유사상장기업의 자기자본비용(CAPM 적용)
- 업종별 위험프리미엄(보통 무위험수익률 + 6~12%)

자본환원율은 수익가치 계산의 핵심이므로, 합리적이고 객관적인 자료에 기초해야 합니다. 실무에서는 보수적 접근을 위해 범위(예: 10~15%)를 제시하고 중간값을 사용하기도 합니다.

**4. 정상수익 추정 자료**
- 3년간 조정 손익계산서(일회성 항목 제거)
- 향후 사업계획서 및 매출 예측
- 업종 평균 수익률 자료
- 경영진 인터뷰 및 실사 자료

**5. 외부 평가기관 의견서**
- 자본시장법상 적격 외부평가기관이 작성한 공정성 의견서
- 평가 방법론의 적정성 검토 의견
- 평가 가정의 합리성 검증

**6. 법적 서류**
- 이사회 결의서(합병, 분할 등 주요 안건 결의)
- 주주총회 소집통지서 및 결의서
- 공시자료(전자공시시스템 제출 서류)
- 주식매수청구권 행사 관련 통지서

이러한 자료는 평가 기준일 기준으로 작성되어야 하며, 평가보고서에 첨부하여 객관성과 검증가능성을 확보해야 합니다.

## 섹션 3: 자본시장법 평가법의 실전 활용

### 3.1 실무 적용 사례

**사례 1: 상장사 간 합병 - 삼성물산-제일모직 유형**

2015년 삼성물산과 제일모직의 합병은 한국 자본시장법상 본질가치 평가가 쟁점이 된 대표적 사례입니다. 두 회사 모두 상장사였지만, 시장가격만으로 합병비율을 정할 경우 소액주주 이익이 침해될 우려가 있었습니다.

**평가 프로세스**:
- 삼성물산과 제일모직 각각에 대해 적격 외부평가기관(회계법인, 투자은행) 3곳 이상 선정
- DCF, 상대가치평가(PER, PBR, EV/EBITDA), 자산가치평가를 모두 수행
- 비상장 기준으로 본질가치 산정: 자산가치(순자산 재평가) 40% + 수익가치(정상수익 환원) 60%
- 상장사 시장가격과 본질가치를 종합하여 최종 합병비율 결정
- 외부평가의견서를 이사회 및 주주총회에 제출하고, 금융감독원에 공시

**쟁점**:
- 삼성물산의 자산가치 평가 시 삼성전자 지분의 공정가치 산정
- 제일모직의 수익가치 계산 시 패션사업과 전자소재사업의 정상수익 추정
- 합병 시너지를 합병비율에 어느 정도 반영할 것인가

이 사례는 외부평가기관의 독립성, 평가 가정의 합리성, 소액주주 보호 절차의 중요성을 보여줍니다. 실무에서는 법원이나 증권선물위원회의 사후 검증 가능성을 고려하여 보수적이고 객관적인 평가를 수행해야 합니다.

**사례 2: Pre-IPO 기업가치 산정**

(주)바이오텍은 바이오 신약 개발 회사로, 상장 예비심사 전 시리즈 C 투자를 유치하려고 합니다. 기존 주주와 신규 투자자 간 발행가액을 둘러싼 분쟁을 방지하기 위해 본질가치 평가를 수행합니다.

**평가 과정**:
- **자산가치**: 순자산 50억원 + 파이프라인(특허권) 평가액 100억원 = 150억원
- **수익가치**: 향후 신약 출시 후 정상수익 예상 30억원, 자본환원율 15%, 수익가치 200억원
- **본질가치**: (150억원 × 1 + 200억원 × 1.5) ÷ 2.5 = 180억원

시리즈 C 투자금 50억원을 유치할 때, 투자 후 기업가치(Post-money)를 180억원 기준으로 설정하면 신규 투자자는 27.8% 지분을 취득하게 됩니다. 이는 DCF 평가(예: 250억원)보다 보수적이지만, 법적 분쟁 시 합리적 근거로 인정받을 수 있습니다.

Pre-IPO 단계에서는 증권신고서 제출 전 외부평가기관의 공정성 의견서를 첨부하여 발행가액의 적정성을 입증해야 하며, 본질가치는 이 과정에서 핵심 자료가 됩니다.

**사례 3: 주식교환을 통한 M&A**

(주)플랫폼A는 (주)플랫폼B를 주식교환 방식으로 인수하려 합니다. (주)플랫폼B의 주주들은 자사 주식을 (주)플랫폼A의 신주와 교환하게 되는데, 교환비율 산정을 위해 본질가치 평가를 활용합니다.

**평가 결과**:
- (주)플랫폼A 본질가치: 1,000억원(1주당 50,000원)
- (주)플랫폼B 본질가치: 200억원(1주당 20,000원)
- 주식교환비율: 1 : 0.4(플랫폼B 주식 1주당 플랫폼A 신주 0.4주 교부)

외부평가기관은 양사의 재무제표를 실사하고, 자산 감정평가와 수익가치를 계산한 후 본질가치를 산출합니다. 이 과정에서 다음을 검토합니다:
- 양사의 회계정책 차이 조정
- 특수관계자 거래의 정상가격 조정
- 일회성 손익 제거 및 정상수익 산정
- 업종별 자본환원율의 합리성

주식교환이 완료되면 (주)플랫폼B는 (주)플랫폼A의 100% 자회사가 되며, 반대주주는 주식매수청구권을 행사하여 본질가치에 기초한 가격으로 주식을 매도할 수 있습니다.

### 3.2 주의사항과 팁

**흔한 실수 1: 자의적 가정**
수익가치 계산 시 과도하게 낙관적인 정상수익을 추정하거나, 자본환원율을 지나치게 낮게 설정하면 평가결과가 왜곡됩니다. 예를 들어 최근 1년의 호실적만을 근거로 정상수익을 산정하면, 일회성 요인이 제거되지 않아 수익가치가 과대평가될 수 있습니다. 실무 팁: 최소 3년 이상의 실적을 가중평균하고, 업종 평균 수익률과 비교하여 합리성을 검증해야 합니다.

**흔한 실수 2: 비교가능성 부족**
자산가치 계산 시 유사거래 사례나 시장가치 자료 없이 장부가액을 그대로 사용하면 자산가치가 과소 또는 과대평가됩니다. 특히 부동산을 보유한 기업의 경우 장부가액과 시가의 괴리가 클 수 있습니다. 실무 팁: 중요 자산은 반드시 공인 감정평가법인의 감정평가를 받아야 하며, 무형자산도 전문 평가기관의 의견을 받는 것이 좋습니다.

**흔한 실수 3: 법적 요건 미준수**
자본시장법상 본질가치 평가는 단순히 계산만 하면 끝나는 것이 아닙니다. 적격 외부평가기관(금융감독원 등록 회계법인, 신용평가사, 자산평가법인)의 평가보고서가 필수이며, 이사회 결의와 주주총회 승인 절차를 거쳐야 법적 효력이 있습니다. 실무 팁: M&A나 합병 추진 초기에 법무법인과 회계법인을 동시에 선임하여 법적 절차와 평가 일정을 통합 관리해야 합니다.

**실무 팁 1: 적격 외부평가기관 선정**
자본시장법 시행령은 외부평가기관의 자격을 엄격히 규정합니다. Big 4 회계법인(삼일, 삼정, 안진, 한영)이나 주요 신용평가사(한국신용평가, NICE신용평가, 한국기업평가)를 선정하면 평가보고서의 신뢰성과 법원 인정 가능성이 높아집니다. 비용은 기업 규모에 따라 3천만원~2억원 수준이지만, 법적 분쟁 시 입증책임을 다했다는 증거가 됩니다.

**실무 팁 2: 자산가치와 수익가치의 격차 관리**
자산가치와 수익가치의 차이가 2배 이상 벌어지면, 본질가치의 대표성에 의문이 제기될 수 있습니다. 예를 들어 부동산이 많은 자산보유회사는 자산가치 200억원, 수익가치 50억원으로 격차가 큽니다. 이 경우 본질가치 110억원[(200×1+50×1.5)÷2.5]이 과연 적정한가에 대한 논란이 생길 수 있습니다. 실무 팁: DCF나 상대가치평가 등 다른 평가법을 병행하여 교차 검증하고, 평가보고서에 격차 발생 원인을 상세히 설명해야 합니다.

**실무 팁 3: 이사회 및 주주총회 승인 절차**
합병, 분할, 주요 자산양수도는 이사회 결의(이사 3분의 2 이상 찬성)와 주주총회 특별결의(출석주주 의결권의 3분의 2 이상, 발행주식총수의 3분의 1 이상 찬성)가 필요합니다. 본질가치 평가보고서는 이사회 결의 전에 완료되어야 하며, 주주총회 소집통지서에 첨부하여 주주들이 충분히 검토할 수 있도록 최소 2주 전에 발송해야 합니다.

**실무 팁 4: 공시 의무 이행**
상장사는 외부평가의견서를 전자공시시스템(DART)에 주요사항보고서로 제출해야 하며, 비상장사도 일정 규모 이상 거래는 공시 의무가 있습니다. 공시 내용에는 평가기관명, 평가 방법론, 주요 가정, 최종 평가액이 포함되어야 하며, 허위 공시나 중요사항 누락 시 형사처벌 및 손해배상 책임이 발생할 수 있습니다.

**언제 자본시장법 평가법을 사용하는가**
다음의 법적 의무 상황에서 반드시 본질가치 평가를 수행해야 합니다:
- 주식매수청구권 행사 시 비상장주식 매수가격 산정
- 상장사 간 합병 시 합병비율 산정 및 외부평가의견서 작성
- 비상장 자회사 간 합병 시 소액주주 보호를 위한 가격 결정
- 주식교환, 주식이전, 분할합병 등 조직재편 시 교환비율 산정
- Pre-IPO 제3자 배정 증자 시 불공정거래 시비 방지

**다른 평가법과의 교차 검증**
본질가치는 법적 의무 사항이지만, 실무에서는 DCF, 상대가치평가, 거래사례 비교법 등을 병행하여 타당성을 검증합니다. 만약 본질가치가 100억원인데 DCF가 200억원, 유사거래 배수로 계산하면 150억원이라면, 본질가치가 과소평가되었을 가능성이 있습니다. 이 경우 자산가치 평가 시 무형자산이 누락되었는지, 수익가치 계산 시 자본환원율이 너무 높게 설정되었는지 재검토해야 합니다. 최종적으로는 여러 평가법의 결과를 종합하여 합리적인 가치 범위를 제시하는 것이 바람직합니다.
