# 조세심판원 결정례: 조심2022서5914

## 사례번호
**조심2022서5914** (2022.12.15. 결정)

---

## 쟁점 내용

### 핵심 쟁점
청구법인이 특수관계법인에게 비상장주식을 **현금흐름할인법(DCF법)**으로 평가한 가액에 양도한 경우, 상증세법상 보충적 평가방법과의 차이를 부당행위계산부인으로 과세할 수 있는가?

### 배경
- 청구법인은 2011년 6월 AAA 주식회사 발행주식(비상장)을 취득하여 보유
- 2016년 4월 특수관계인인 BBB 주식회사에 DCF법으로 평가한 가액에 양도
- 조사청은 상증세법상 보충적 평가방법에 따른 평가액이 양도가액보다 높다고 판단
- 법인세법상 부당행위계산부인 규정을 적용하여 과세

---

## 평가 공식

### 1. 상증세법상 보충적 평가방법

```
주당평가액 = (순손익가치 × 3 + 순자산가치 × 2) / 5
```

**구성요소:**
- 순손익가치 = 1주당 순손익액 × (1 / 환원율)
- 순자산가치 = (자산총액 - 부채총액) / 발행주식총수
- 가중치: 순손익가치 3, 순자산가치 2

### 2. 현금흐름할인법 (DCF: Discounted Cash Flow)

```
기업가치 = Σ [미래현금흐름 / (1 + 할인율)^n] + 영구가치
```

**구성요소:**
- 미래현금흐름 추정 (통상 5~10년)
- 할인율 (WACC: 가중평균자본비용)
- 영구가치 (Terminal Value)
- 순부채 차감 후 주주가치 산정

### 3. 본 사례의 평가 차이

```
[평가방법별 결과]
• DCF법 평가액:         청구법인이 적용한 가액
• 상증세법 평가액:      조사청이 적용한 가액 (더 높음)
• 차이:                 부당행위계산부인 대상액
```

## 당사자 주장

### 청구법인 주장

1. **DCF법의 합리성**
   - 현금흐름할인법에 따른 평가가액이 합리적이고 공정한 **시가**에 해당
   - 미래현금흐름을 반영한 객관적 평가방법
   - 2개 평가기관(CCC, DDD)의 평가 결과 적용

2. **보충적 평가방법의 부당성**
   - 상증세법상 보충적 평가방법 적용이 불합리
   - 과거 실적 중심 평가는 미래가치 반영 불충분
   - DCF법이 더 객관적이고 합리적

3. **시가 해당 여부**
   - 감정평가법인의 평가액은 **시가**로 인정되어야 함
   - 법인세법상 부당행위계산부인 대상이 아님

4. **자산평가 이의**
   - 쟁점건물의 임대차보증금을 감정평가액으로 평가해야 함
   - 순자산가치 산정에 오류 존재

### 처분청(조사청) 주장

1. **보충적 평가방법 우선 적용**
   - 비상장주식의 시가는 상증세법상 보충적 평가방법으로 산정
   - DCF법은 법적 근거가 불충분
   - 특수관계 거래에서 보충적 평가액이 기준

2. **저가양도 인정**
   - 보충적 평가액 > DCF 평가액
   - 특수관계법인 간 거래로 부당행위계산부인 해당

3. **부당성 인정**
   - 법인세 부담을 부당하게 감소시킨 거래
   - 법인세법 제52조(부당행위계산부인) 적용 정당

## 결정 내용

### 심판원 결정: **기각** (처분청 승)

### 결정 요지

#### 1. DCF법의 시가 인정 불가

**심판원 판단:**
```
비상장주식의 거래에서 DCF법에 따른 평가액을
무조건 시가로 인정할 수 없음

특히, 특수관계인 간 거래에서는
상증세법상 보충적 평가방법이 우선 적용됨
```

#### 2. 보충적 평가방법의 적정성

- 비상장주식의 경우 **상증세법 시행령 제54조**의 보충적 평가방법이 기본
- DCF법은 개별 사안의 특수성이 있을 때 예외적으로 인정
- 본 건은 일반적인 비상장주식 거래로 특수성 없음

#### 3. 부당행위계산부인 적용 타당

**법인세법 제52조 적용요건 충족:**
```
☑ 특수관계인 간 거래
☑ 저가양도 (보충적 평가액 > 양도가액)
☑ 법인의 이익을 계열사에 이전
☑ 법인세 부담 부당 감소
```

#### 4. 과세처분 적법

- 조사청의 부당행위계산부인 과세처분에 잘못이 없음
- 보충적 평가액과 DCF 평가액의 차액만큼 익금산입 정당

### 법적 근거
- 법인세법 제52조 (부당행위계산의 부인)
- 상속세및증여세법 제63조 (재산의 평가)
- 상속세및증여세법 시행령 제54조 (비상장주식의 평가)
- 법인세법 시행령 제88조 (부당행위계산의 유형)

## 시사점

### 1. DCF법의 제한적 인정

**중요 원칙:**
```
DCF법 ≠ 자동적 시가 인정

[시가 인정 요건]
① 객관적이고 합리적인 평가
② 특수관계 아닌 제3자 간 거래
③ 또는 특수한 사정이 있는 경우
```

### 2. 평가방법의 우선순위

**비상장주식 평가 우선순위:**
```
1순위: 실제 거래가액 (시가)
        ↓ (시가 없는 경우)
2순위: 상증세법 보충적 평가방법
        - (순손익가치 × 3 + 순자산가치 × 2) / 5
        ↓ (보충적 평가 불합리한 경우)
3순위: 합리적 평가방법 (DCF 등)
        - 특수한 사정 입증 필요
```

### 3. 특수관계 거래의 엄격성

**특수관계법인 간 거래 시:**
- 보충적 평가방법이 **원칙적 기준**
- DCF법 적용 시 **특수한 사정** 입증 필요
- 단순히 "합리적"이라는 주장만으로는 부족
- 명확한 증빙과 사유 필요

### 4. 부당행위계산부인 리스크

**저가양도 판단기준:**
```
보충적 평가액 > 실제 거래가액
        ↓
특수관계 거래 + 합리적 사유 없음
        ↓
부당행위계산부인 적용
        ↓
차액 만큼 익금산입 + 법인세 추징
```

### 5. 실무 적용 가이드

#### 비상장주식 양도 시 체크리스트

```
[평가방법 선택]
□ 특수관계 여부 확인
□ 보충적 평가액 사전 산정
□ DCF 평가액과 비교
□ 차이가 크면 리스크 높음

[DCF법 적용 시]
□ 특수한 사정 존재 여부 검토
□ 객관적 증빙자료 확보
□ 복수 평가기관 평가
□ 유권해석 또는 사전 심사 고려
```

#### 안전한 거래가액 설정

```
거래가액 ≥ 상증세법 보충적 평가액

또는

DCF 평가액 사용 시:
- 세무당국 사전 승인
- 명확한 특수사정 입증
- 충분한 근거자료 확보
```

### 6. 판례 변화 추이 참고

**과거 유사 판례:**
- 조심 2018중333 (2018.6.7.): DCF법 인정 사례
- 특수한 사정이 명확한 경우 DCF법 시가 인정 가능
- 본 건은 특수사정 인정 안 됨

**핵심 차이:**
```
[DCF 인정 ○]
• 업종 특수성
• 미래가치 중심 산업
• 과거실적 무의미
• 명확한 사업계획

[DCF 인정 ×]
• 일반 제조/서비스업
• 안정적 수익구조
• 과거실적 의미 있음
• 특수사정 불명확
```

### 7. 세무리스크 회피 방안

**방법 1: 보충적 평가액 기준 거래**
- 가장 안전한 방법
- 세무리스크 최소화

**방법 2: DCF법 사용 + 안전장치**
- 사전 세무당국 확인
- 특수사정 명확히 입증
- 충분한 증빙자료
- 복수 전문기관 평가

**방법 3: 혼합 적용**
- 보충적 평가액과 DCF 평가액 비교
- 높은 금액으로 거래
- 추가 세금 부담 감수

### 평가 실무 포인트

**이 사례의 핵심:**
- DCF법은 만능 평가방법이 아님
- 특수관계 거래에서는 보충적 평가방법이 **원칙**
- DCF법 적용 시 **특수한 사정** 입증 필수
- 사전 세무검토 없이 DCF법만 적용하면 **과세 리스크**

**실무 권고사항:**
```
비상장주식 M&A 또는 특수관계 거래 시
→ 반드시 상증세법 보충적 평가액 산정
→ DCF 평가액과 비교
→ 차이가 크면 세무사 또는 회계사와 검토
→ 필요시 세무당국 사전질의
→ 안전한 거래가액 결정
```
