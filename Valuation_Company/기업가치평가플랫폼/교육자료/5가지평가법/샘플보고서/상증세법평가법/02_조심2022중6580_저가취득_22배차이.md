# 조세심판원 결정례: 조심2022중6580

## 사례번호
**조심2022중6580** (2022.10.21. 결정)

---

## 쟁점 내용

### 핵심 쟁점
청구인이 특수관계인으로부터 비상장주식을 **액면가로 취득**했으나, **보충적 평가액이 액면가의 22배**를 초과하는 경우 저가취득에 따른 증여세 과세가 적법한가?

### 배경
- 청구인은 ㈜AAA의 대표이사
- 2019년 10월 25일 회사 사용인(양도인)으로부터 주식 4,500주를 액면가에 양수
- 발행회사가 지속적으로 이익을 창출하여 보충적 평가액이 액면가의 22배 초과
- 처분청은 저가거래로 판단하여 증여세 부과

---

## 평가 공식

### 상증세법상 비상장주식 평가 공식

```
주당평가액 = (순손익가치 × 3 + 순자산가치 × 2) / 5
```

**본 사례에서:**
- 액면가: 1주당 5,000원
- 보충적 평가액: 1주당 약 110,000원 (액면가의 **22배**)
- 취득 주식수: 4,500주
- 증여이익: (110,000원 - 5,000원) × 4,500주 = 약 4억 7천만원

### 저가취득 증여의제 규정
상속세및증여세법 제35조 및 제42조에 따라:
```
증여재산가액 = 시가 - 대가
              = 보충적 평가액 - 액면가
```

## 당사자 주장

### 청구인 주장
1. **특수관계 부존재**
   - 양도인이 거래일 이전인 2019년 9월 30일에 퇴사
   - 퇴사자이므로 상증세법상 특수관계인에 해당하지 않음

2. **실질대가 포함**
   - 양도인에게 지급한 퇴직금이 실질적인 양수대가에 포함되어야 함
   - 따라서 저가거래가 아님

3. **과세요건 미충족**
   - 처분청이 저가거래의 구체적 근거를 제시하지 않음
   - 정당한 사유가 있는 거래임

### 처분청 주장
1. **특수관계 인정**
   - 근로소득지급명세서상 양도인이 **2019년 12월 11일**까지 근무
   - 거래 당시(2019.10.25.) 회사 사용인으로서 특수관계 성립

2. **현저한 저가거래**
   - 보충적 평가액이 액면가의 **22배 초과**
   - 시가와 대가의 차액이 3억원 이상이므로 증여세 과세대상

3. **정당한 사유 없음**
   - 퇴직금은 별개의 근로대가
   - 주식양수대가에 포함될 수 없음

## 결정 내용

### 심판원 결정: **기각** (처분청 승)

### 결정 요지

1. **특수관계 인정**
   ```
   근로소득지급명세서상 양도인은 2019.12.11.까지 근무
   → 거래일(2019.10.25.) 당시 회사의 사용인
   → 상증세법 시행령 제2조의2 제1항 제7호에 따른 특수관계인
   ```

2. **저가거래 인정**
   - 보충적 평가액과 액면가의 차이가 **22배**로 현저함
   - 정당한 사유 없이 액면가로 거래
   - 상증세법 제35조 제1항의 저가양수 증여의제 해당

3. **퇴직금 주장 배척**
   - 퇴직금은 근로의 대가로 별도 지급되는 것
   - 주식양수 대가와는 법적 성격이 다름
   - 따라서 주식양수대가에 포함될 수 없음

4. **증여세 과세 적법**
   - 처분청의 증여세 부과처분에 잘못이 없음

### 법적 근거
- 상속세및증여세법 제35조 (증여추정)
- 상속세및증여세법 제42조 (저가·고가 양도·양수에 따른 이익의 증여)
- 상속세및증여세법 시행령 제2조의2 (특수관계인의 범위)
- 상속세및증여세법 시행령 제54조 (비상장주식의 평가)

## 시사점

### 1. 특수관계 판단의 엄격성
- 형식적 퇴사일이 아닌 **실질적 근무기간** 기준
- 근로소득지급명세서 등 객관적 증빙자료로 판단
- 거래 당시 특수관계 여부가 핵심

### 2. 저가거래 인정 기준
**22배 차이**와 같이 시가와 대가의 차이가 현저한 경우:
```
[저가거래 인정 요건]
① 특수관계인 간 거래
② 시가 > 대가 (현저한 차이)
③ 정당한 사유 부존재
```

### 3. 정당한 사유의 입증 책임
- 납세자가 정당한 사유를 **구체적으로 입증**해야 함
- 단순한 주장만으로는 부족
- 퇴직금 등 별개 명목의 금액은 대가로 인정 곤란

### 4. 평가 실무 주의사항

#### 사용인으로부터 주식 취득 시
```
[체크리스트]
☑ 근로관계 종료 시점 확인
☑ 근로소득지급명세서 확인
☑ 4대보험 상실일자 확인
☑ 실질적 근무 종료일 확인
```

#### 저가거래 리스크 평가
```
차이비율 = (보충적 평가액 - 취득가액) / 취득가액 × 100

• 차이비율 > 30% → 증여세 리스크 높음
• 본 사례: 2,100% (22배) → 명백한 저가거래
```

### 5. 실무 대응 방안

**거래 전 검토사항:**
1. 특수관계 여부 명확히 확인
2. 시가 또는 보충적 평가액 사전 산정
3. 적정한 거래가액 결정
4. 정당한 사유 존재 시 명확한 증빙자료 확보

**거래가액 결정:**
- 비상장주식의 경우 보충적 평가액 이상으로 거래
- 또는 시가감정평가 실시하여 적정가액 산정
- 증여세 과세 리스크 사전 제거

### 6. 유사 사례 참고기준

이 판례는 다음과 같은 경우에 참고:
- 임직원으로부터 주식 취득
- 액면가 또는 저가로 주식 거래
- 지속적 흑자기업의 주식 거래
- 특수관계 여부가 불명확한 경우

### 평가 실무 포인트
이 사례는 **현저한 저가거래**에 대한 과세당국의 엄격한 입장을 보여주는 중요한 판례로, 비상장주식 거래 시 반드시 보충적 평가액을 고려해야 함을 시사함

### 증여세 회피 방지
보충적 평가액의 **22배 차이**가 나는 거래에서 증여세 과세는 조세회피 방지 차원에서 당연한 결과이며, 향후 유사 거래에 대한 명확한 기준이 됨
