2. SK이노베이션-SK E&S 합병 (2024년)
================================================================

■ 공시 정보
- 공시일: 2024년 7월 17일
- 공시 유형: 회사합병 결정
- 접수번호: 20240717000575

■ 접근 방법

[방법 1] KIND 공시뷰어 직접 링크
URL: https://kind.krx.co.kr/common/disclsviewer.do?method=search&acptno=20240717000575

[방법 2] KIND 검색
1. https://kind.krx.co.kr 접속
2. 검색창에 "SK이노베이션" 입력
3. 2024년 7월 17일자 "회사합병 결정" 클릭
4. 접수번호 확인: 20240717000575

[방법 3] DART 시스템 (전자공시시스템)
1. https://dart.fss.or.kr 접속
2. 회사명 검색: "SK이노베이션"
3. 2024년 7월 17일 주요사항보고서 확인

■ 주요 내용
- 합병비율: 1:1.1917417
- 자산규모: 약 100~105조원
- 평가방법:
  * 상장사(SK이노베이션): 시가총액 기준
  * 비상장사(SK E&S): 자산가치 + 수익가치 (1:1.5 가중)

■ 교육 포인트
- 상장사 vs 비상장사 평가방법 차이
- 자산가치와 수익가치의 가중평균
- 평가방법 선택의 공정성 논란
- 시가 vs NAV 적용 시 합병비율 차이

3. 공시 문서 다운로드 방법
================================================================

■ KIND 시스템 활용

1) 웹 브라우저로 접근
   - 위 URL을 브라우저 주소창에 직접 입력
   - 공시 내용 확인 후 브라우저 인쇄 기능 (Ctrl+P) 사용
   - "PDF로 저장" 선택하여 로컬에 저장

2) 첨부파일 다운로드
   - 공시 페이지 내 "첨부" 또는 "다운로드" 버튼 클릭
   - 평가의견서, 합병계약서, 이사회의사록 등 첨부문서 다운로드

■ DART 시스템 활용

1) 접근: https://dart.fss.or.kr
2) 회사명으로 검색
3) 문서 유형 필터: "주요사항보고서"
4) 날짜 범위: 2024.07.01 ~ 2024.07.31
5) 해당 문서 클릭 후 PDF 다운로드
