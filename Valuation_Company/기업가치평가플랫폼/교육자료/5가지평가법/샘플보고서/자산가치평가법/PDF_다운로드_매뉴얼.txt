자산가치평가 보고서 PDF 다운로드 매뉴얼
================================================================

작성일: 2025년 10월 17일
대상: 교육자 및 학습자
목적: KIND/DART 시스템에서 실제 공시 PDF 다운로드 방법 안내

================================================================
방법 1: KIND 시스템에서 직접 다운로드
================================================================

■ 클래시스-이루다 합병 보고서

1단계: 웹 브라우저 열기
   - Chrome, Edge, Firefox 등 최신 브라우저 사용 권장

2단계: URL 접속
   https://kind.krx.co.kr/external/2024/07/18/000369/20240718000929/11344.htm

3단계: 페이지 로딩 대기
   - 공시 내용이 화면에 완전히 표시될 때까지 대기

4단계: PDF로 저장
   방법 A) 브라우저 인쇄 기능 사용
   - Ctrl+P (또는 File > Print)
   - 프린터 선택: "PDF로 저장" 또는 "Microsoft Print to PDF"
   - 파일명: 01_클래시스_이루다_합병_NAV.pdf
   - 저장 위치: 본 폴더

   방법 B) 브라우저 확장 프로그램 사용
   - "Save as PDF" 등의 확장 프로그램 설치
   - 전체 페이지를 PDF로 저장

5단계: 첨부파일 다운로드 (있는 경우)
   - 페이지 내 "첨부파일" 또는 "다운로드" 링크 클릭
   - 외부평가의견서, 합병계약서 등 별도 다운로드

■ SK이노베이션-SK E&S 합병 보고서

1단계: 웹 브라우저로 접속
   https://kind.krx.co.kr/common/disclsviewer.do?method=search&acptno=20240717000575

2단계: 공시뷰어 화면 대기
   - iframe 형태로 문서가 로딩됨
   - 문서가 완전히 표시될 때까지 대기

3단계: 다운로드 옵션 찾기
   방법 A) 뷰어 내 다운로드 버튼
   - 화면 상단 또는 우측에 "다운로드", "출력", "PDF" 버튼 확인
   - 클릭하여 PDF 다운로드

   방법 B) 브라우저 인쇄 기능
   - Ctrl+P
   - "PDF로 저장" 선택
   - 파일명: 02_SK이노베이션_SKE&S_합병.pdf

4단계: 첨부문서 확인
   - "첨부" 탭 또는 섹션 확인
   - 외부평가의견서 (자산가치평가 상세 내용 포함)
   - 합병계약서
   - 이사회의사록
   - 각 문서 개별 다운로드

================================================================
방법 2: DART 시스템 활용
================================================================

■ 접근 단계

1단계: DART 홈페이지 접속
   https://dart.fss.or.kr

2단계: 회사명 검색
   - 상단 검색창에 "SK이노베이션" 입력
   - 또는 "공시서류검색" 메뉴 선택

3단계: 검색 필터 설정
   - 회사명: SK이노베이션
   - 보고서명: "주요사항보고서" 선택
   - 접수일자: 2024.07.01 ~ 2024.07.31
   - "검색" 버튼 클릭

4단계: 해당 문서 선택
   - 2024년 7월 17일자 "주요사항보고서(회사합병결정)" 클릭
   - 또는 접수번호로 검색: 20240717000575

5단계: PDF 다운로드
   - 문서 뷰어 화면에서 "다운로드" 아이콘 클릭
   - 또는 첨부파일 목록에서 개별 다운로드
   - 파일명 수정: 02_SK이노베이션_SKE&S_합병.pdf

5단계: 원문 보기
   - 공시 제목 클릭 시 KIND 또는 DART로 연결
   - 위 방법 1 또는 2 진행

트러블슈팅
================================================================

■ 문제: 페이지가 로딩되지 않음
해결방법:
- 다른 브라우저 시도 (Chrome, Edge, Firefox)
- 브라우저 캐시 삭제 후 재시도
- VPN 연결 확인 (해외 접속 시)
- 한국 시간 기준 업무시간(09:00-18:00) 내 접속

■ 문제: PDF 저장 옵션이 보이지 않음
해결방법:
- Ctrl+P 단축키 사용
- 브라우저 설정에서 "인쇄" 기능 확인
- Adobe Acrobat Reader 또는 PDF 리더 설치
- "Microsoft Print to PDF" 프린터 드라이버 설치

■ 문제: 첨부파일 다운로드 실패
해결방법:
- 팝업 차단 해제
- 브라우저 다운로드 설정 확인
- ActiveX 또는 보안 프로그램 설치 (IE 사용 시)
- 크롬 브라우저 사용 권장 (ActiveX 불필요)

■ 문제: 문서 내용이 깨져 보임
해결방법:
- 한글 폰트 설치 확인
- PDF 리더 업데이트
- 다른 PDF 리더 프로그램 사용 (Adobe, 알PDF, Sumatra 등)
