## 즉시 실행 가능한 방법

### 방법 1: DART 웹사이트 (5분, 추천)

1. **접속**: https://dart.fss.or.kr
2. **검색**:
   - 회사명: `다음커뮤니케이션`
   - 기간: `2014.05.01` ~ `2014.08.31`
   - 보고서: `주요사항보고서` 체크
3. **찾기**: "회사합병결정" 보고서
4. **다운로드**: 첨부파일 > "외부평가보고서.pdf"

**예상 파일명**: 삼정회계법인 평가보고서

---

### 방법 2: Python 스크립트 (10분)

#### 사전 준비
```bash
# 1. API 키 발급 (https://opendart.fss.or.kr)
#    - 회원가입 > 개발가이드 > 인증키 신청

# 2. 라이브러리 설치
pip install opendartreader
```

#### 스크립트 실행
```bash
# 1. dart_downloader.py 열기
# 2. API_KEY = 'YOUR_API_KEY_HERE' 부분에 발급받은 키 입력
# 3. 실행
python dart_downloader.py
```

**출력**:
- `03_카카오다음_DART검색결과.csv` - rcpNo 목록
- `03_카카오다음_DART검색결과.txt` - 텍스트 요약

**다음 단계**: CSV에서 rcpNo 확인 → DART 접속 → PDF 다운로드
