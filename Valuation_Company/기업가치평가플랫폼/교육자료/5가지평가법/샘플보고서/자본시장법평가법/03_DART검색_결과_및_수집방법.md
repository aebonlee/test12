## 2. DART 검색 방법론

### 방법 1: DART 웹사이트 직접 검색 (권장)

#### 접속 URL
https://dart.fss.or.kr

#### 검색 절차
1. **공시통합검색** 메뉴 선택
2. **회사명**: "다음커뮤니케이션" 또는 "하이크코리아" 입력
3. **검색기간**: 2014.05.01 ~ 2014.08.31 (다음 경우)
4. **보고서명**: "주요사항보고서" 선택 또는 "합병" 키워드 입력
5. 검색 결과에서 "회사합병결정" 보고서 찾기
6. **첨부파일** 확인 - "평가보고서" 또는 "외부평가" PDF 다운로드

#### 주요 검색 키워드
- 주요사항보고서(회사합병결정)
- 임시주주총회소집공고
- 합병계약서
- 외부평가보고서
- 자본시장법 제176조의5

### 방법 2: OpenDART API 사용

#### API 인증키 발급
1. OpenDART 접속: https://opendart.fss.or.kr
2. "개발가이드" > "인증키 신청" 메뉴
3. 회원가입 후 API 인증키 발급 (무료)

#### API 활용 예시

```python
import OpenDartReader

# API 키 설정
api_key = 'YOUR_API_KEY_HERE'
dart = OpenDartReader(api_key)

# 회사 검색
companies = dart.list('다음커뮤니케이션')
corp_code = companies.iloc[0]['corp_code']

# 공시 검색
disclosures = dart.list_date_ex(
    start='2014-05-01',
    end='2014-08-31',
    corp_code=corp_code,
    report_nm='주요사항보고서'
)

# rcpNo 확인
for idx, row in disclosures.iterrows():
    print(f"날짜: {row['rcept_dt']}")
    print(f"보고서: {row['report_nm']}")
    print(f"rcpNo: {row['rcept_no']}")
    print(f"URL: https://dart.fss.or.kr/dsaf001/main.do?rcpNo={row['rcept_no']}")
    print("-" * 80)
```

## 3. 자본시장법 제176조의5 평가 공식 정리

### 일반 원칙 (비상장법인 평가)

```
본질가치 = (자산가치 × 1 + 수익가치 × 1.5) ÷ 2.5
        = 자산가치 × 40% + 수익가치 × 60%
```

**다양한 표현 방식**:
1. (자산×1 + 수익×1.5) / 2.5
2. (자산×2 + 수익×3) / 5
3. 자산×0.4 + 수익×0.6

### SPAC 특례 규정 (시행령 제176조의5 제3항 제2호 나목)

```
본질가치 = (자산가치 × 1 + 수익가치 × 3) ÷ 4
        = 자산가치 × 25% + 수익가치 × 75%
```

**특징**:
- 수익가치 가중치 더 높음 (75%)
- 성장성 높은 기업에 유리
- RF시스템즈-교보SPAC 사례에서 적용

### 상장법인 평가

**원칙**: 기준주가법
```
기준주가 = (1개월 가중평균 × 2 + 1주일 가중평균 × 1 + 최근일 종가 × 3) ÷ 6
```

**예외**: 기준주가 < 자산가치인 경우
→ 자산가치 적용 가능
