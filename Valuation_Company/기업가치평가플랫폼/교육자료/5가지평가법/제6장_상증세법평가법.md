# 제6장 | 상증세법 평가법

## 섹션 1: 상증세법 평가법이란 무엇인가

### 1.1 정의와 핵심 개념

**상증세법 평가법**은 상속세 및 증여세법에 따라 비상장주식의 가치를 평가하는 법정 방법입니다. 상속세 및 증여세법 시행령 제54조에 근거하여, 세법상 과세 목적으로 주식 가치를 객관적으로 산정하기 위해 마련된 제도입니다.

핵심 계산 구조는 **순손익가치 60%와 순자산가치 40%를 가중평균**하는 방식입니다. 구체적으로 순손익가치에 3의 가중치를, 순자산가치에 2의 가중치를 부여하여 평균하면 결과적으로 60:40 비율이 됩니다. 이는 기업의 수익창출능력(순손익가치)과 순수한 자산 상태(순자산가치)를 모두 반영하되, 수익창출능력을 더 중요하게 평가하는 방식입니다.

비상장주식 평가의 목적은 명확합니다. 시장가격이 형성되지 않는 비상장기업의 주식을 상속하거나 증여할 때, 과세당국과 납세자가 합의할 수 있는 공정한 기준을 제공하는 것입니다. 이를 통해 자의적인 평가를 방지하고 세무 분쟁을 최소화합니다.

### 1.2 왜 상증세법 평가법이 필요한가

상증세법 평가법은 무엇보다 **세무 목적**에서 필수적입니다. 상속세와 증여세의 과세표준을 산정할 때, 비상장주식의 가치를 어떻게 정할 것인가가 핵심 쟁점입니다. 상장주식은 시장가격이 있지만, 비상장주식은 거래가 없어 객관적 가격을 찾기 어렵습니다. 이때 상증세법 평가법이 표준화된 계산식을 제공하여 공정한 과세를 가능하게 합니다.

특히 **경영권 승계 계획**을 세울 때 이 평가법은 핵심 도구가 됩니다. 창업주가 자녀에게 가업을 물려줄 때, 주식 증여 시점의 세금 부담을 사전에 계산할 수 있습니다. 가업승계 지원 제도를 활용하면 세액 공제나 감면 혜택을 받을 수 있는데, 이때도 상증세법 평가액이 기준이 됩니다.

국세청 입장에서는 과소평가를 방지하여 세수를 확보하고, 납세자 입장에서는 합법적 절세 범위 내에서 평가액을 최소화하려 합니다. 양측의 이해관계가 맞물린 지점에서 상증세법 평가법은 투명한 기준선 역할을 합니다. 법적 근거가 명확하므로 세무조사 시에도 정당한 평가 방법으로 인정받습니다.

### 1.3 다른 평가법과의 차이

상증세법 평가법은 **자본시장법 평가법**과 외형상 유사합니다. 둘 다 자산가치와 수익가치를 혼합하며, 두 방법 모두 **자산가치 40% + 수익가치 60%**의 가중구조를 사용합니다. 다만 자본시장법은 (자산×1 + 수익×1.5)÷2.5로 계산하고, 상증세법은 (순자산×2 + 순손익×3)÷5로 계산하는 등 세부 계산 방식과 용어가 다릅니다. 또한 자본시장법은 합병, 분할 등 자본시장 거래 목적이고, 상증세법은 순수하게 세무 신고 목적입니다.

**DCF나 상대가치평가**와 비교하면 근본적인 철학이 다릅니다. DCF는 미래 현금흐름을 추정하여 현재가치로 환산하는 미래지향적 방법이며, 상대가치평가는 시장 배수를 활용한 시장 중심 접근입니다. 반면 상증세법 평가법은 **과거 3년 실적 평균**을 기반으로 하여 보수적이고 객관적인 평가를 추구합니다.

상증세법 평가의 핵심 특징은 세 가지입니다. 첫째, **보수적 접근**으로 과도한 가치 부풀리기를 방지합니다. 둘째, **과거 실적 중심**이어서 검증 가능한 데이터만 사용합니다. 셋째, 법에서 정한 **할인/할증 규정**이 있어 대주주는 최대 20% 할증, 소액주주는 최대 20% 할인이 적용됩니다. 이러한 요소들이 결합되어 세법 특유의 평가 체계를 형성합니다.

## 섹션 2: 상증세법 평가법의 계산 방법

### 2.1 계산 공식과 단계별 프로세스

상증세법 평가는 명확한 6단계 프로세스를 따릅니다. 각 단계별 상세 계산 방법을 숙지하면 실제 세무 신고용 평가서 작성이 가능합니다.

#### 전체 공식

**기본 공식:**
```
평가액 = (순손익가치 × 3 + 순자산가치 × 2) ÷ 5
```

백분율로 표현:
```
평가액 = 순손익가치 × 60% + 순자산가치 × 40%
```

이는 기업의 수익창출능력(60%)을 더 중요하게 보되, 자산 상태(40%)도 반영하는 균형적 접근입니다.

#### 1단계: 재무제표 수집 및 기준일 설정

**1-1. 평가기준일 결정**
- 상속: 피상속인 사망일
- 증여: 증여계약일 또는 명의변경일 중 빠른 날
- 일반 거래: 매매계약일

**1-2. 재무제표 3개년 확보**
평가기준일이 속하는 사업연도의 직전 3개 사업연도:
- 손익계산서 (각 사업연도)
- 재무상태표 (평가기준일 현재)
- 법인세 신고서 (세무조정계산서 포함)

**예시:** 평가기준일이 2024.12.31인 경우
- 순손익 계산: 2022, 2023, 2024년 사업연도 손익
- 순자산 계산: 2024.12.31 재무상태표

#### 2단계: 순손익가치 계산 (세무조정 포함)

**2-1. 각 사업연도 순손익액 확인**

순손익액은 **회계상 당기순이익이 아닌 세무조정 후 각 사업연도 소득금액**입니다.

**세무조정 프로세스:**

```
각 사업연도 순손익 = 당기순이익
                   + 손금불산입 (세법상 비용 인정 안 됨)
                   - 익금불산입 (세법상 수익 인정 안 됨)
                   + 손금산입 (세법상 추가 비용 인정)
                   - 익금산입 (세법상 추가 수익 인정)
```

**주요 세무조정 항목:**

| 조정 항목 | 구분 | 예시 | 조정 방향 |
|---------|------|------|---------|
| **손금불산입** | 비용 부인 | 접대비 한도초과, 기부금 한도초과, 업무무관 비용 | + (이익 증가) |
| **익금불산입** | 수익 부인 | 배당금 익금불산입, 자산수증익 | - (이익 감소) |
| **손금산입** | 비용 추가 인정 | 세법상 감가상각비(회계 초과분) | + (장부이익이 세법이익보다 큼) |
| **익금산입** | 수익 추가 인정 | 재고자산 평가차익 | - (희귀) |

**비정상 손익 조정:**

일회성·비정상적 손익은 제거하여 정상 수익력 반영:
```
조정 순손익 = 세무조정 후 소득금액
           - 일회성 이익(자산처분이익, 보험차익, 채무면제이익)
           + 일회성 손실(재해손실, 대규모 소송비용)
           ± 특수관계자 거래 정상가격 조정
```

#### 업종별 세무조정 필수 체크리스트

상증세법 평가 시 세무조정은 필수입니다. 법인세 신고서(세무조정계산서)를 확인하면서 다음 항목을 점검하세요.

##### ✅ 제조업 필수 점검 항목

**손금불산입 (이익 증가 방향)**
- [ ] 접대비 한도초과액 (중소기업 2,400만원 + 수입금액×0.3%)
- [ ] 기부금 한도초과액 (소득금액×10%)
- [ ] 업무무관 자산 감가상각비 (임원 전용 차량, 골프회원권 등)
- [ ] 퇴직급여충당금 한도초과
- [ ] 대손충당금 한도초과
- [ ] 특수관계자 고가 매입 (시가 대비 20% 이상)

**익금불산입 (이익 감소 방향)**
- [ ] 수입배당금 익금불산입 (계열사 배당)
- [ ] 자산수증익 (무상 증여받은 자산)
- [ ] 채무면제익 (특수관계자 채무 탕감)

**일회성 손익 제거**
- [ ] 유형자산 처분손익 (토지, 건물, 기계 매각)
- [ ] 보험차익 (화재보험금 등)
- [ ] 재해손실
- [ ] 대규모 소송 배상금

##### ✅ 서비스업/IT 필수 점검 항목

**손금불산입**
- [ ] 인건비 중 업무무관 지출 (과다한 임원 급여)
- [ ] 광고선전비 한도초과
- [ ] 복리후생비 중 특정인 편익
- [ ] 접대비 한도초과
- [ ] 연구개발비 부인 (요건 미충족)

**익금불산입**
- [ ] 수입배당금
- [ ] 기타 영업외수익 중 일회성 항목

**일회성 손익 제거**
- [ ] 투자자산 처분손익 (주식, 채권 매각)
- [ ] 법인세 환급금
- [ ] 정부 보조금 (일회성)

##### ✅ 부동산업 필수 점검 항목

**손금불산입**
- [ ] 토지 관련 차입금 이자 (손금불산입 대상)
- [ ] 건설자금 이자 중 자본화 대상
- [ ] 취득세, 등록세 중 자산가액 산입 대상

**일회성 손익 제거**
- [ ] 분양수익 중 대규모 일시 인식분
- [ ] 재개발/재건축 관련 일회성 손익

##### 📌 실무 활용 팁

1. **법인세 신고서 확인**
   - 세무조정계산서 (별지 제3호서식)
   - 소득처분명세서

2. **3개년 일관성 검토**
   - 동일 항목이 매년 발생하는지 확인
   - 특정 연도에만 급증한 항목 → 일회성 가능성

3. **특수관계자 거래 주의**
   - 계열사 간 거래는 시가 대비 정상가격으로 조정
   - 임원 대여금 이자 (정기예금이자율 적용 필수)

4. **세무사 협업**
   - 위 체크리스트로 1차 자체 점검
   - 의문 항목은 세무사에게 문의
   - 조정 근거 문서 준비 (계약서, 영수증 등)

**2-2. 3년 평균 순손익 계산**

```
3년 평균 순손익 = (N-2년 순손익 + N-1년 순손익 + N년 순손익) ÷ 3
```

**주의:** 상증세법은 **가중평균이 아닌 단순산술평균** 사용 (자본시장법과 차이)

**2-3. 1주당 순손익액 산출**

```
1주당 순손익액 = 3년 평균 순손익 ÷ 발행주식총수
```

발행주식총수:
- 보통주 + 우선주 (종류별로 별도 평가 가능)
- 자기주식 제외
- 평가기준일 현재 주식수

**2-4. 순손익가치 산출 (자본환원)**

```
1주당 순손익가치 = 1주당 순손익액 ÷ 할인율
```

**할인율:** 현행 **10%** (상속세 및 증여세법 시행령 제54조)
- 국세청 고시로 변경 가능 (정기예금이자율 등 고려)
- 2024년 현재 10% 적용 중

**계산 예시: A제조(주)**

| 사업연도 | 당기순이익 | 세무조정 | 일회성 조정 | 최종 순손익 |
|---------|----------|---------|-----------|----------|
| 2022 | 12억원 | +2억원(손불) | -3억원(토지처분) | 11억원 |
| 2023 | 18억원 | +3억원(손불) | +1억원(소송손실) | 22억원 |
| 2024 | 25억원 | +2억원(손불) | -2억원(보험차익) | 25억원 |
| **합계** | 55억원 | +7억원 | -4억원 | **58억원** |
| **3년 평균** | | | | **19.33억원** |

```
발행주식총수: 100만주
1주당 순손익액 = 19.33억원 ÷ 100만주 = 1,933원/주
1주당 순손익가치 = 1,933원 ÷ 0.10 = 19,330원/주
```

#### 3단계: 순자산가치 계산 (시가 재평가)

**3-1. 자산 항목별 시가 평가**

| 자산 항목 | 평가 방법 | 법적 근거 |
|---------|---------|---------|
| **토지** | 개별공시지가 또는 감정평가액 | 부동산 가격공시법 |
| **건물** | 국세청 기준시가 또는 감정평가액 | 상증세법 시행령 |
| **상장주식** | 평가기준일 전후 2개월 평균 종가 | 상증세법 시행령 제49조 |
| **비상장주식** | 상증세법 평가 재귀적 적용 | 상증세법 시행령 제54조 |
| **채권** | 액면가 × 잔존기간별 할인율 | 상증세법 시행령 |
| **매출채권** | 장부가액 - 대손추산액 | 회수가능액 |
| **재고자산** | 순실현가능가치 | 장부가액 또는 시가 |
| **기계·설비** | 장부가액 (특별한 경우 감정평가) | 취득원가 - 감가상각 |

**3-2. 부채 항목별 시가 평가**

```
장기차입금 시가 = 잔액 (만기 일시상환 시)
               = Σ(원리금 상환액 ÷ (1+시장이자율)^n) (분할상환 시)

우발부채 = 소송금액 × 패소확률 (재무제표 주석 확인)
```

**3-3. 순자산 산출**

```
순자산 시가 = Σ(자산 항목별 시가) - Σ(부채 항목별 시가) - 우발부채
```

**3-4. 1주당 순자산가치 산출**

```
1주당 순자산가치 = 순자산 시가 ÷ 발행주식총수
```

**계산 예시: A제조(주)**

|-----|---------|---------|------|

| - 건물 | 30억원 | 기준시가 | 35억원 |
| - 상장주식 | 20억원 | 2개월 평균가 | 25억원 |
| - 매출채권 | 15억원 | 회수가능액 | 14억원 |
| - 재고자산 | 10억원 | 시가 | 10억원 |
| - 기타자산 | 25억원 | 장부가 | 25억원 |
| **자산 합계** | **150억원** | | **189억원** |
| **부채** | | | |
| - 차입금 | 40억원 | 시가 | 40억원 |

```
1주당 순자산가치 = 134억원 ÷ 100만주 = 13,400원/주
```

#### 4단계: 1주당 평가액 산출 (가중평균)

**4-1. 기본 평가액 계산**

```
1주당 평가액 = (순손익가치 × 3 + 순자산가치 × 2) ÷ 5
```

또는
```
1주당 평가액 = 순손익가치 × 60% + 순자산가치 × 40%
```

**A제조(주) 사례:**
```
1주당 평가액 = (19,330원 × 3 + 13,400원 × 2) ÷ 5
            = (57,990원 + 26,800원) ÷ 5
            = 84,790원 ÷ 5
            = 16,958원/주
```

검증(백분율):
```
1주당 평가액 = 19,330원 × 60% + 13,400원 × 40%
            = 11,598원 + 5,360원
            = 16,958원/주 ✓
```

#### 5단계: 할인·할증 적용

**5-1. 할증 사유 (최대주주 등)**

**대상:** 최대주주 및 그 특수관계인이 보유한 주식
- 최대주주: 단독으로 가장 많은 주식 보유
- 특수관계인: 배우자, 직계존비속, 6촌 이내 혈족, 4촌 이내 인척 등

**할증률:** **20% 할증 (경영권 프리미엄)**

```
할증 후 평가액 = 1주당 평가액 × 1.20
```

**적용 조건:**
- 최대주주 + 특수관계인 지분율 합계가 50% 초과 또는
- 경영권 행사가 명백한 경우

**5-2. 할인 사유 (소액주주)**

**대상:** 다음 중 하나를 충족하는 소액주주
1. **지분율 기준**: 해당 주주 및 특수관계인 지분율 합계가 **3% 미만**
2. **금액 기준**: 보유 주식 시가총액이 **15억원 미만** (2024년 기준)

**할인률:** **20% 할인 (유동성 디스카운트)**

```
할인 후 평가액 = 1주당 평가액 × 0.80
```

**할인 배제 사유:**
- 증여 후 특수관계인 지분율 합계가 3% 이상이 되는 경우 할인 불가
- 조세 회피 목적이 명백한 경우 할인 부인

**5-3. 할인·할증 판단 흐름도**

```
시작
  ↓
최대주주 또는 특수관계인? → 예 → 20% 할증
  ↓ 아니오
지분율 3% 미만? → 예 → 20% 할인
  ↓ 아니오
시가총액 15억 미만? → 예 → 20% 할인
  ↓ 아니오
할인·할증 없음 (원평가액)
```

**계산 예시:**

| 구분 | 기본 평가액 | 할증/할인 | 최종 평가액 |
|-----|----------|---------|----------|
| 최대주주 (지분 40%) | 16,958원 | 20% 할증 | 20,350원 |
| 소액주주 (지분 2%) | 16,958원 | 20% 할인 | 13,566원 |
| 일반주주 (지분 10%) | 16,958원 | 없음 | 16,958원 |

#### 6단계: 최종 평가액 산출 및 신고

**6-1. 총 평가액 계산**

```
총 평가액 = 1주당 최종 평가액 × 보유주식수
```

**6-2. 증여세 과세표준 산정**

```
과세표준 = 총 평가액 - 증여재산공제
```

증여재산공제:
- 배우자: 6억원
- 직계존속: 5천만원 (미성년자 2천만원)
- 직계비속: 5천만원
- 기타 친족: 1천만원

**6-3. 세액 계산**

| 과세표준 | 세율 | 누진공제 |
|---------|------|---------|
| 1억원 이하 | 10% | 0 |
| 5억원 이하 | 20% | 1천만원 |
| 10억원 이하 | 30% | 6천만원 |
| 30억원 이하 | 40% | 1.6억원 |
| 30억원 초과 | 50% | 4.6억원 |

**6-4. 신고 기한 및 납부**

- 증여세: 증여일이 속하는 달의 말일로부터 3개월 이내
- 상속세: 상속개시일이 속하는 달의 말일로부터 6개월 이내
- 신고·납부 지연 시 가산세 부과

**종합 계산 예시:**

최대주주가 자녀에게 10만주 증여 (A제조(주))
```
1주당 할증 평가액: 20,350원
총 평가액: 20,350원 × 10만주 = 20억 3,500만원
증여재산공제: 5천만원
```

### 2.2 필요한 데이터

상증세법 평가를 위해서는 체계적인 데이터 수집이 필수입니다.

**재무제표 자료**로는 최근 3개 사업연도의 손익계산서와 재무상태표가 필요합니다. 여기서 주의할 점은 단순히 회계상 당기순이익이 아니라 **세무조정 후 순손익**을 사용해야 한다는 것입니다. 법인세 신고서상 각 사업연도 소득금액을 확인해야 합니다. 일회성 손익이나 비정상적인 항목은 제거하여 정상적인 수익력을 반영합니다.

**자산 재평가 자료**는 순자산가치 산정에 필수입니다. 토지와 건물은 감정평가서가 있으면 그 가액을, 없으면 공시지가를 기준으로 합니다. 상장주식이나 채권은 평가기준일 전후 2개월 평균 시장가격을 적용합니다. 비상장주식은 다시 상증세법 평가 방식으로 계산해야 하므로 복잡도가 증가합니다.

**발행주식총수**는 정확히 파악해야 합니다. 자기주식은 제외하고, 우선주가 있다면 별도로 평가합니다. **할인율**은 현행 법령상 연 10%가 적용되지만, 정기예금이자율 등을 고려하여 변경될 수 있으므로 최신 국세청 고시를 확인해야 합니다.

**할인/할증 사유**도 사전에 검토해야 합니다. 최대주주와 특수관계인이 보유한 주식은 경영권 프리미엄으로 최대 20% 할증됩니다. 반대로 소액주주가 보유한 주식(지분율 3% 미만 또는 시가 총액 15억원 미만)은 유동성 디스카운트로 최대 20% 할인됩니다. 이러한 조정이 최종 세액에 큰 영향을 미치므로 정확한 판단이 필요합니다.

## 섹션 3: 상증세법 평가법의 실전 활용

### 3.1 실무 적용 사례

**사례 1: 가업승계를 위한 주식 증여(부모 → 자녀)**

B전자(주)의 창업주 김사장(70세)은 장남에게 경영권을 물려주기로 결정했습니다. 보유 중인 주식 30%(300만주 중 90만주)를 증여하려 합니다.

상증세법 평가 결과:
- 1주당 순손익가치: 25,000원
- 1주당 순자산가치: 20,000원
- 1주당 평가액: (25,000×3 + 20,000×2)÷5 = 23,000원
- 대주주 할증 20% 적용: 23,000 × 1.20 = 27,600원
- 90만주 평가액: 248억 4,000만원

증여세 산정:
- 과세표준: 248억원(10억원 공제 후)
- 세율: 50%(누진공제 4.6억원)
- 납부세액: 약 119억원

여기서 가업상속공제 제도를 활용하면 최대 500억원까지 공제받을 수 있어 세부담이 대폭 줄어듭니다. 단, 업종 요건, 고용 유지 등 까다로운 조건을 충족해야 하므로 사전 세무 설계가 필수입니다.

**사례 2: 경영자 사망 시 상속세 산정**

C물류(주)의 대표이사가 급서하여 배우자와 자녀 2명이 상속인이 됩니다. 고인은 발행주식 60%(120만주)를 보유했습니다.

상증세법 평가 결과:
- 1주당 평가액: 15,000원
- 대주주 할증 20% 적용: 18,000원
- 120만주 평가액: 216억원

상속세 산정:
- 상속재산: 주식 216억원 + 기타 재산 50억원 = 266억원
- 일괄공제 5억원 차감
- 과세표준: 261억원
- 납부세액: 약 126억원

이 경우 상속인들은 막대한 상속세 부담으로 주식을 처분해야 할 수도 있습니다. 따라서 생전에 증여를 나눠서 하거나, 가업상속공제를 받을 수 있도록 준비하는 것이 절세 전략이 됩니다.

**사례 3: 소액주주 할인 활용 전략**

D바이오(주)의 창업주는 자녀 3명에게 각각 2%씩 주식을 증여하려 합니다. 각 자녀가 받는 지분이 3% 미만이므로 소액주주 20% 할인이 적용됩니다.

할인 전 평가액(1주): 30,000원
할인 후 평가액(1주): 24,000원
절감 효과: 주당 6,000원 = 20%

100만주 기준으로 60억원의 평가액 차이가 발생합니다. 이를 활용하면 같은 주식을 증여하더라도 증여세를 크게 줄일 수 있습니다. 다만, 세법은 실질과세 원칙을 적용하므로 명의만 나눠 놓고 실제로는 한 사람이 지배하는 경우 할인이 부인될 수 있습니다.

### 3.2 주의사항과 팁

**흔한 실수 1: 세무조정 누락**
회계상 당기순이익을 그대로 사용하면 안 됩니다. 법인세 신고서상 세무조정 후 각 사업연도 소득금액을 확인해야 합니다. 접대비 한도초과액, 감가상각비 시인부인액 등이 조정되므로 회계이익과 차이가 발생합니다.

**흔한 실수 2: 비정상 손익 미제거**
특정 연도에 일회성 자산처분이익 50억원이 발생했다면, 이를 그대로 포함하면 평가액이 과대계상됩니다. 비정상적 손익은 제거하거나 평준화하여 기업의 정상적 수익력을 반영해야 합니다.

**흔한 실수 3: 우발부채 미반영**
계속 중인 소송, 보증채무, 환경오염 복구 의무 등 재무제표에 명시되지 않은 우발부채가 있다면 순자산가치에서 차감해야 합니다. 이를 누락하면 세무조사 시 불이익을 받을 수 있습니다.

**실무 팁 1: 세무사·회계사 협업 필수**
상증세법 평가는 단순해 보여도 세무조정, 자산 재평가, 할인/할증 판단 등 전문적 영역이 많습니다. 반드시 경험 많은 세무사나 회계사와 협업하여 정확한 평가를 받아야 합니다.

**실무 팁 2: 국세청 유권해석 확인**
특수한 상황(합병 직후, 자본 구조 변경, 신규 사업 진출 등)에서는 평가 방법이 불명확할 수 있습니다. 세무 신고 전에 국세청에 유권해석을 신청하면 법적 안정성을 확보할 수 있습니다.

**실무 팁 3: 할인/할증 규정 적극 활용**
소액주주 할인은 최대 20%까지 가능합니다. 증여 시점에 지분율을 조정하거나, 증여 순서를 조율하여 합법적으로 할인 혜택을 최대화할 수 있습니다. 반대로 대주주 할증이 적용되지 않도록 지분을 분산하는 전략도 고려할 수 있습니다.

**실무 팁 4: 사전 증여 vs 사후 상속 시뮬레이션**
생전 증여와 사후 상속의 세부담을 비교 시뮬레이션하여 최적 시나리오를 선택해야 합니다. 일반적으로 기업 가치가 낮을 때 증여하고, 가업상속공제를 최대한 활용하는 것이 유리합니다.

**실무 팁 5: 가업상속공제 요건 확인**
상속세 및 증여세법 제18조에 따라 가업상속공제를 받으려면 업종(제조업, 서비스업 등), 매출액, 업력(10년 이상), 고용 유지 등 까다로운 요건을 충족해야 합니다. 사전에 요건을 검토하고 미비한 부분을 보완해야 공제 혜택을 받을 수 있습니다.

**언제 상증세법 평가법을 사용하는가?**
상속세 신고, 증여세 신고, 합병 시 세무 신고가 대표적입니다. 또한 주주간 주식 매매 시 시가 인정 범위를 판단하거나, 특수관계인 간 거래 시 정상가액 산정에도 활용됩니다.

**시가 인정 범위와 국세청 조사 대응**
상증세법 평가액의 ±30% 범위 내에서 거래하면 시가로 인정받습니다. 이 범위를 벗어나면 세무조사 대상이 될 수 있으므로, 평가 근거를 명확히 문서화하고 외부 전문가 의견서를 준비하는 것이 안전합니다.
