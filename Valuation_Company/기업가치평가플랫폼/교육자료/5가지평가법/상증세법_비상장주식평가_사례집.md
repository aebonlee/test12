### 사례 1: 조심2023서0142 (2023.6.29.)
**쟁점**: 자기주식 보유법인의 비상장주식 평가 - 순자산가치 80% 하한 적용

**사실관계**:
- 부동산보유비율: 54.1%
- 순손익가치와 순자산가치 2:3 가중평균액이 순자산가치의 80% 미만인 경우

**평가방법**:
- 청구인 주장: 가중평균액 적용 후 자기주식 반영하여 재계산
- 처분청 입장: 순자산가치의 80% 하한 적용

**결정**:
- **기각** - 처분청 입장 인정
- 상증법 시행령 제54조 제1항 단서규정에 따라 1주당 순자산가치의 80%로 평가

**평가공식**:
```
1주당 가중평균액 = (순손익가치 × 2 + 순자산가치 × 3) / 5
※ 부동산과다보유법인(자산 중 부동산 50% 이상) 적용

단, 가중평균액 < 순자산가치 × 80% 인 경우
→ 최종 평가액 = 순자산가치 × 80%
```

### 사례 2: 조심2022중6580 (2022.10.21.)
**쟁점**: 비상장주식 저가취득에 따른 증여세 과세

**사실관계**:
- 청구인이 특수관계인(회사 사용인)으로부터 비상장주식 4,500주를 액면가로 취득
- 거래가액이 보충적평가액의 약 4.5% 수준 (22배 이상 차이)

**평가방법**:
- 상증세법상 보충적 평가방법 적용
- 순손익가치와 순자산가치를 3:2 가중평균
- 가중평균액과 순자산가치 80% 중 큰 금액 선택

**결정**:
- **기각** - 증여세 부과 정당
- 특수관계 인정 (양도인이 거래일 이후까지 근무)
- 거래가액과 평가액의 현저한 차이로 저가거래 인정
- 퇴직금을 양수대가에 포함시킬 객관적 증빙 부재

**평가공식**:
```
일반법인의 경우:
1주당 평가액 = (순손익가치 × 3 + 순자산가치 × 2) / 5

단, 가중평균액 < 순자산가치 × 80% 인 경우
→ 최종 평가액 = 순자산가치 × 80%
```

**출처**: https://casenote.kr/조세심판원/조심2022중6580

### 사례 3: 조심2021서0588 (2022.1.27.)
**쟁점**: 순자산가치 계산 시 장부가액의 정의

**사실관계**:
- 순자산가액 계산 시 개별자산 평가액과 비교하는 "장부가액"의 해석 문제
- 청구인: 기업회계기준상 장부가액 주장
- 처분청: 세무상 장부가액(취득가액 - 감가상각비) 적용

**평가방법**:
- 상증법 시행령 제55조 제1항: 장부가액 = 취득가액 - 감가상각비
- 개별자산 평가 시 평가액 < 장부가액인 경우 → 장부가액 적용

**결정**:
- **기각** - 세무상 장부가액 해석 타당
- 법문의 명확한 규정, 대법원 판례, 재정경제부 입법 의도 종합 고려
- "최소한 취득원가로 평가" 원칙 확인

**핵심원칙**:
```
순자산가치 계산 시 개별자산 평가
1) 원칙: 시가 평가
2) 보충적 평가액 < 장부가액(세무상) → 장부가액 적용
3) 단, 장부가액보다 적은 정당한 사유가 있는 경우 예외
```

**출처**: https://casenote.kr/조세심판원/조심2021서0588

### 사례 4: 조심2021광1438 (2021.10.20.)
**쟁점**: 상속재산 비상장주식 평가액의 적정성

**사실관계**:
- 쟁점①주식(AAA): 일반법인으로 3:2 가중평균 적용
- 쟁점②주식(BBB): 폐업법인으로 순자산가치만 적용

**평가방법**:
1. **AAA 주식**: 상증세법 시행령 제54조 제1항
   - 순손익가치 × 3 + 순자산가치 × 2 가중평균

2. **BBB 주식**: 시행령 제54조 제4항 제2호
   - 폐업법인으로 분류 → 순자산가치 평가만 적용

**결정**:
- **기각** - 처분청의 평가 산정 타당
- 청구인이 대체 평가 자료 미제시
- 법령에 따른 평가방법 적용의 정당성 인정

**특례 적용**:
```
순자산가치만으로 평가하는 경우 (시행령 제54조 제4항):
1) 청산절차 진행 중인 법인
2) 사업의 계속이 곤란한 법인
3) 사업개시 전 법인
4) 사업개시 후 3년 미만 법인
5) 휴폐업 중인 법인 ← 이 사례 해당
6) 자산 중 부동산 등 비율 80% 이상 법인
```

### 사례 5: 조심2023서8015 (2023.12.12.)
**쟁점**: 비상장주식 고가양도에 따른 증여세 부과

**사실관계**:
- 청구인이 비상장법인 주식을 특수관계인 및 비특수관계인에게 고가 양도
- 매매사례가액: 발행주식 액면가 총액의 1% 미달로 시가 인정 불가

**평가방법**:
- 보충적 평가방법 적용 (순손익가치와 순자산가치 가중평균)
- 전환사채 전환가격 검토
- 매매사례가액의 시가 인정 요건 미충족

**결정**:
- **기각** - 고가양도 과세 정당
- 거래경위, 가격결정이유 등 확인 불가
- 다른 매매사례가액도 시가로 보기 어려움

**시가 인정 요건**:
```
매매사례가액이 시가로 인정되기 위한 요건:
1) 평가기준일 전후 6개월 이내 거래
2) 특수관계 없는 자 간 거래
3) 발행주식 액면가 총액의 1% 이상 거래
4) 거래의 관행상 정당한 사유 존재
```

### 사례 6: 조심2022서1786 (2022.9.26.)
**쟁점**: 비상장주식 저가양수에 따른 증여세 부과

**사실관계**:
- 청구인이 비특수관계자로부터 액면가로 비상장주식 취득
- 거래가액이 보충적 평가액의 1.11%에 불과 (약 90배 차이)

**평가방법**:
- 상증세법 제63조 보충적 평가방법
- 순손익가치와 순자산가치 3:2 가중평균
- 가중평균액 < 순자산가치 80% → 순자산가치 80% 적용

**결정**:
- **기각** - 증여세 부과 정당
- 거래가액과 평가액의 현저한 차이
- 미처분이익잉여금 등 고려 시 정당한 사유 없음
- 청구인이 합리적 평가근거 미제시

**저가양수 판단기준**:
```
특수관계 없는 자 간 거래에서도:
1) 거래가액 < 시가(평가액) × 30%인 경우
2) 거래의 관행상 정당한 사유가 없는 경우
→ 저가양수로 증여세 과세
```

### 사례 7: 조심2022서5914 (2022.12.15.)
**쟁점**: 현금흐름할인법(DCF) vs 보충적 평가방법

**사실관계**:
- 청구법인이 특수관계법인에게 비상장주식을 DCF법으로 평가하여 양도
- 처분청이 보충적 평가방법 적용하여 저가양도로 판단

**평가방법**:
- 청구인: DCF법에 따른 평가가액 주장
- 처분청: 순손익가치 × 3 + 순자산가치 × 2 가중평균

**결정**:
- **기각** - 보충적 평가방법 적용 타당
- 시가 불분명 시 상증세법상 보충적 평가방법에 따른 가액을 시가로 봄
- 부당행위계산부인 규정 적용 정당
- 부정과소신고가산세도 인정

**핵심원칙**:

시가 인정 순서:
1순위: 매매사례가액 (요건 충족 시)
2순위: 감정평가액 (2개 이상 평균)
3순위: 보충적 평가방법 (순손익가치 + 순자산가치)

### 사례 8: 조심2021서3854 (2022.8.11.)
**쟁점**: 소액거래 판정 기준 (양수인별 vs 전체)

**사실관계**:
- 한 양도인이 두 명의 양수인에게 동일 조건으로 주식 양도
- 각 거래는 소액이지만, 전체로는 발행주식 총액의 1% 초과

**평가방법**:
- 외부 회계법인의 DCF 평가를 기초로 비특수관계자 간 협상
- 동일 계약서상 거래의 실질 판단

**결정**:
- **인용** (청구인 승소) - 경정청구 거부처분 취소
- 동일 계약서상 거래를 실질적으로 하나의 거래로 판단
- 쟁점거래가액을 매매사례가액으로 인정
- 대법원 판례(2012.4.26, 2010두26988) 인용

**소액거래 판정**:
```
발행주식 액면가 총액 1% 기준 판정 시:
- 형식: 양수인별 거래가액으로 판단 → 소액 제외
- 실질: 전체 거래의 실질 고려 → 일괄거래 인정 가능

법원 판단: 실질과세 원칙에 따라 거래의 실질 중시
```

**출처**: https://casenote.kr/조세심판원/조심2021서3854

**쟁점**: 소액 매매사례가액의 시가 인정 (재산평가심의위원회 심의)

**사실관계**:

**평가방법**:
- 평가기간: 증여일 전 6개월 ~ 후 3개월
- 평가기간 내 거래가액의 평균액 적용
- 상증세법 시행령 제49조의2: 재산평가심의위원회 심의

- **기각** - 처분청의 시가 인정 타당
- 평가기간 이내 거래
- 특수관계 없는 자 간 거래
- 재산평가심의위원회 심의 거쳐 정당성 인정

**재산평가심의위원회 역할**:
```
소액 비상장주식 거래의 시가 인정:
1) 원칙: 발행주식 액면가 총액의 1% 미만 → 시가 불인정
2) 예외: 재산평가심의위원회 심의로 정당한 사유 인정 시
   → 소액거래도 시가로 인정 가능

심의사항: 거래의 정당성, 가격의 합리성, 시가 인정 타당성 등
```

## 국세청 유권해석

### 해석 1: 상속증여세과-231 (2014.7.3.)
**주제**: 보통주와 우선주를 발행한 비상장법인의 주식 평가방법

**쟁점**:
1. 순자산가치 계산 시 우선주(전환우선주) 처리
2. 순손익가치 계산 시 유상증자 효과 반영

**유권해석**:
- **전환우선주는 발행주식총수에 포함**
- 순자산가치 및 순손익가치 계산 시 모두 포함

**법적 근거**:
- 상속세 및 증여세법 제63조 제1항 제1호 다목
- 시행령 제54조 제1항

**선례**:
- 서면4팀-1894 (2004.11.23.): 상환우선주도 발행주식총수에 포함
- 서면4팀-1179 (2004.7.27.): 전환우선주 포함 원칙

**평가방법**:
```
순자산가치 = (자산가액 - 부채가액) / 발행주식총수
※ 발행주식총수에 보통주, 전환우선주, 상환우선주 모두 포함

순손익가치 = 최근 3년 순손익액 가중평균 / 환원율
※ 주식수 계산 시 우선주도 포함하여 환산
```

**출처**: https://casenote.kr/국세청/상속증여세과-231-66e763

### 해석 2: 재산세과-526 (2011.11.7.)
**주제**: 비상장법인이 보유한 다른 비상장법인 주식의 평가방법

**쟁점**:
- A회사의 순자산가치 계산 시 보유 중인 B회사 주식(10% 초과) 평가방법

**유권해석**:
- B회사 주식을 보충적 평가방법으로 평가
- **평가액 < 장부가액인 경우 → 장부가액 적용**
- 단, 장부가액보다 적은 정당한 사유가 있는 경우 예외

**핵심원칙**:
```
비상장법인의 순자산가액 계산:
1) 개별자산은 시가로 평가 (원칙)
2) 보유 비상장주식: 보충적 평가방법 적용
3) 보충적 평가액 vs 장부가액 비교
   - 보충적 평가액 ≥ 장부가액 → 보충적 평가액
   - 보충적 평가액 < 장부가액 → 장부가액
   - 단, 정당한 사유 있으면 보충적 평가액
```

**참고**: 재산세과-4209 (2008.12.12.) 동일 원칙

**출처**: https://casenote.kr/국세청/재산세과-526-269f8c

### 해석 3: 서면-2015-법령해석재산-2404 (2016.11.16.)
**주제**: 출자전환 방식으로 채무조정 중인 법인의 비상장주식 평가

**쟁점**:
- 회생계획에 따라 향후 출자전환 예정인 채무를 부채로 처리할 수 있는지

**사실관계**:
- A법인이 2014년 회생인가 받음
- 10년 후 80억원 부채를 출자전환하기로 계획
- 결산 시 자본조정 계정으로 처리

**유권해석**:
- **출자전환 예정 채무는 부채에 해당**
- 회생계획인가 내역대로 출자전환채무로 회계처리한 경우 부채 인정

**평가방법**:
```
순자산가치 계산:
순자산가액 = 자산 - 부채
※ 출자전환 예정 채무도 부채에 포함

1주당 순자산가치 = 순자산가액 / 발행주식총수
※ 발행주식총수는 현재 발행된 주식만 포함
   (향후 출자전환으로 발행될 주식은 제외)
```

**법적 근거**: 상증세법 시행령 제54조·제55조

**출처**: https://casenote.kr/국세청/서면-2015-법령해석재산-2404-709536

### 해석 4: 서면인터넷방문상담4팀-1683 (2006.6.12.)
**주제**: 장기대여금의 평가 기준 (소송 진행 중인 채권)

**쟁점**:
- 비상장법인 순자산가치 계산 시 소송 중인 장기대여금 평가
- 기업회계기준 장부가액 vs 세무상 유보금액 반영 가액

**유권해석**:
- 장기대여금 = **원본 + 평가기준일까지 미수이자**
- 단, 회수불가능한 부분은 제외

**회수불가능 판단기준**:
- 채무자의 재산상황
- 소송 진행과정
- 기타 구체적 사실관계 종합 검토
- **소송 진행 중이라는 사유만으로는 자동 제외 안 됨**

**평가원칙**:
```
채권 평가:
1) 원칙: 원본 + 미수이자
2) 회수불가능 채권 제외
   - 객관적으로 회수 불가능 확정된 경우
   - 입증책임: 납세의무자

소송 중인 채권:
- 단순 소송 중이라는 이유만으로 제외 불가
- 채무자 재산상황, 소송 경과 등 실질 판단
```

**출처**: https://casenote.kr/국세청/서면인터넷방문상담4팀-1683-1d7d39

## 평가방법 요약

### 1. 기본 평가공식

#### (1) 일반법인
```
1주당 평가액 = (순손익가치 × 3 + 순자산가치 × 2) / 5

단, 가중평균액 < 순자산가치 × 80% 인 경우
→ 최종 평가액 = 순자산가치 × 80%
```

#### (2) 부동산과다보유법인 (자산 중 부동산 50% 이상)
```
1주당 평가액 = (순손익가치 × 2 + 순자산가치 × 3) / 5

단, 가중평균액 < 순자산가치 × 80% 인 경우
→ 최종 평가액 = 순자산가치 × 80%
```

#### (3) 특례: 순자산가치만으로 평가하는 경우
```
1주당 평가액 = 순자산가치

적용 대상:
- 청산절차 진행 중인 법인
- 사업 계속이 곤란한 법인
- 사업개시 전 법인
- 사업개시 후 3년 미만 법인
- 휴폐업 중인 법인
- 자산 중 부동산 등 비율 80% 이상 법인
- 계속 결손법인(평가기준일 속하는 사업연도 전 3년 내 계속 결손)
```

---

### 2. 순손익가치 계산

#### (1) 최근 3년 순손익액 가중평균
```
1주당 최근 3년 순손익액 가중평균 =
  [(1년 전 순손익액 × 3) + (2년 전 순손익액 × 2) + (3년 전 순손익액 × 1)] / 6
```

#### (2) 순손익가치 환산
```
1주당 순손익가치 = 1주당 최근 3년 순손익액 가중평균 / 순손익가치 환원율

순손익가치 환원율 = 기획재정부령이 정하는 이자율 (보통 10%)
```

#### (3) 1주당 순손익액 계산
```
1주당 순손익액 = 순손익액 / 환산주식수

환산주식수 = 기초주식수 + (기중 유상증자주식수 × 개월수/12)
※ 무상증자, 주식배당은 기초주식수에 가산
```

---

### 3. 순자산가치 계산

#### (1) 기본 공식
```
1주당 순자산가치 = 순자산가액 / 발행주식총수

순자산가액 = 자산가액 - 부채가액
```

#### (2) 자산 평가
```
개별자산 평가:
1순위: 시가
2순위: 보충적 평가방법
3순위: 장부가액 (보충적 평가액 < 장부가액인 경우)

장부가액 = 취득가액 - 감가상각비 (세무상 기준)
```

#### (3) 보유 비상장주식 평가
```
비상장법인이 다른 비상장주식을 보유한 경우:
- 보충적 평가방법으로 평가
- 보충적 평가액 < 장부가액 → 장부가액 적용
- 단, 정당한 사유 있으면 보충적 평가액 적용 가능
```

#### (4) 부채 평가
```
부채 인정:
- 확정된 부채: 장부가액
- 출자전환 예정 채무: 부채로 인정
- 우발채무: 확정 시 부채 인정

채권 평가:
- 원본 + 평가기준일까지 미수이자
- 회수불가능 채권: 제외 (입증책임: 납세자)
```

#### (5) 발행주식총수
```
발행주식총수 포함 범위:
- 보통주
- 전환우선주
- 상환우선주
- 기타 모든 발행주식

자기주식:
- 발행주식총수에서 차감하지 않음
- 자기주식도 평가 대상
```

---

### 4. 시가 인정 기준

#### (1) 시가 인정 순서
```
1순위: 매매사례가액
2순위: 감정평가액 (2개 이상 평균)
3순위: 보충적 평가방법
```

#### (2) 매매사례가액 요건
```
다음 요건을 모두 충족해야 시가로 인정:

1) 시기: 평가기준일 전 6개월 ~ 후 3개월 이내
2) 당사자: 특수관계 없는 자 간 거래
3) 규모: 발행주식 액면가 총액의 1% 이상
4) 정당성: 거래의 관행상 정당한 사유 존재

※ 소액거래(1% 미만)도 재산평가심의위원회 심의로 시가 인정 가능
```

#### (3) 시가 불인정 사례
```
시가로 인정되지 않는 평가방법:
- 현금흐름할인법(DCF)
- 기타 감정평가 요건 미충족 평가
- 일방적 자체 평가
```

---

### 5. 저가·고가 거래 판단

#### (1) 저가양수 (증여세 과세)
```
양수가액 < 시가(평가액) × 30%

특수관계자 간: 무조건 과세
비특수관계자 간: 정당한 사유 없으면 과세
```

#### (2) 고가양도 (증여세 과세)
```
양도가액 > 시가(평가액) × 130%

특수관계자 간: 무조건 과세
비특수관계자 간: 정당한 사유 없으면 과세
```

#### (3) 정당한 사유 판단
```
다음 사항을 종합 고려:
- 거래 경위 및 목적
- 가격 결정 과정
- 객관적 평가자료 존재 여부
- 재무상태 및 수익성
- 업종 특성
- 기타 합리적 사정
```

---

### 6. 입증책임

#### (1) 과세관청
```
입증 사항:
- 시가 산정이 어렵다는 점
- 특수관계 존재
- 부당행위계산 부인 요건
```

#### (2) 납세의무자
```
입증 사항:
- 거래의 정당한 사유
- 회수불가능 채권
- 보충적 평가액보다 적은 정당한 사유
- 장부가액의 합리성
```
