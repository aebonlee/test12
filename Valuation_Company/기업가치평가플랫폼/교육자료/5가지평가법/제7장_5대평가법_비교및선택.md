## 섹션 2: 상황별 평가법 선택

### 2.1 거래 목적별 선택 가이드

#### M&A 거래

**주 평가법: DCF + 상대가치평가 + 자산가치평가**

M&A는 가장 복잡하고 중요한 의사결정이므로 복수 평가법을 반드시 적용해야 합니다.

**매수자 관점:**
```
1차: DCF로 독립적 가치(Standalone Value) 산정
2차: 시너지 가치 추가 분석
3차: 상대가치평가로 시장성 검증
4차: 자산가치평가로 하한선(Floor) 확인
최종: 가중평균 (DCF 50% + 배수법 30% + NAV 20%)
```

**매도자 관점:**
```
1차: 상대가치평가로 최근 거래 배수 확인 (경영권 프리미엄 포함)
2차: DCF로 잠재 가치 입증
3차: 자산가치평가로 최소 가치 제시
최종: 최대값 또는 가중평균의 상한선 주장
```

**실무 팁:**
- 매수자는 보수적 가정, 매도자는 공격적 가정 사용
- 협상 과정에서 점진적으로 격차 축소
- 양측 평가 결과의 중간값이 최종 거래가 되는 경우 많음

#### IPO (기업공개)

**주 평가법: 상대가치평가 + DCF**

**공모가 산정 프로세스:**
```
1단계: 유사 상장기업 PER, PBR, EV/EBITDA 산출
2단계: 대상기업의 지표 × 업종 평균 배수 = 기준가
3단계: DCF로 내재가치 검증
4단계: 희망공모가 밴드 설정 (기준가 ±20%)
5단계: 수요예측(Book Building)으로 최종 공모가 결정
```

**배수 선택 기준:**
- 흑자 기업: PER 주로 사용
- 적자 성장기업: PSR 또는 EV/Sales
- 자산 집약 기업: PBR
- 안정 성숙 기업: PER + 배당수익률

**실무 팁:**
- 코스닥은 PER 15~25배 범위가 일반적
- 코스피는 PER 10~20배 범위
- 성장주는 프리미엄, 가치주는 할인 적용

#### 지분 투자 (VC/PE)

**주 평가법: DCF + 상대가치평가 + VC Method**

**투자 단계별 차이:**

**시드/초기 (Series A):**
- 주로 PSR (매출 배수) 사용
- 경쟁 스타트업 비교
- 창업자 경력, 시장 규모 정성 평가 비중 높음

**성장기 (Series B/C):**
- DCF 시작 가능 (3~5년 예측)
- 유사 상장사 PER/PSR 참고
- Forward Multiple (미래 실적 기준) 사용

**후기 (Pre-IPO):**
- DCF 주 평가법
- IPO 후 예상 시가총액 역산 (VC Method)
- 상장 유사기업 할인율 적용 (유동성 할인 20~30%)

**VC Method 계산 예시:**
```
목표 수익률: 10배
투자금: 100억원
예상 Exit: 5년 후 IPO, 시가총액 2,000억원
역산 현재가치: 2,000억 ÷ 10 = 200억원
투자지분: 100억 ÷ 200억 = 50%
```

### 2.2 산업별 선택 가이드

#### 제조업

**추천 조합: DCF + EV/EBITDA**

제조업은 자산이 명확하고 현금흐름이 안정적이어서 DCF 적용이 가장 적합합니다.

**평가 프로세스:**
```
1차: DCF (FCFF 방식)
  - 과거 5년 실적 분석
  - 매출 성장률 3~5% (GDP 연동)
  - EBITDA 마진 안정적 가정
  - WACC 8~12% 적용

2차: EV/EBITDA 멀티플
  - 업종 평균 6~12배
  - 성장성/수익성 조정

3차: NAV (선택)
  - 공장 토지/건물 재평가
  - 기계설비 시장가치 조정
```

**주의사항:**
- 설비 CAPEX 정확히 추정
- 운전자본 변동 반영
- 경기 사이클 고려

#### IT/소프트웨어

**추천 조합: PSR + DCF + DAU/MAU 지표**

소프트웨어 기업은 초기 적자가 흔하고 무형자산 비중이 높아 전통적 평가법 적용이 어렵습니다.

**평가 프로세스:**
```
1차: PSR (매출 배수)
  - SaaS: PSR 5~15배
  - B2B 소프트웨어: PSR 3~8배
  - 게임: PSR 1~3배

2차: 사용자 지표 기반
  - DAU(일간 활성 사용자) × ARPU
  - LTV(고객생애가치) / CAC(고객획득비용) 비율
  - Churn Rate(이탈률) 분석

3차: DCF (매출 가시화 후)
  - ARR(연간 반복 매출) 기반 예측
  - Rule of 40 (성장률 + 이익률 > 40%)
```

**주의사항:**
- 구독 모델은 MRR, ARR 중시
- 네트워크 효과 정성평가
- 플랫폼은 GMV(거래액) 기준 평가도 고려

#### 부동산/건설

**추천 조합: NAV + DCF**

부동산은 자산가치가 기업가치의 핵심이므로 NAV가 주 평가법입니다.

**평가 프로세스:**
```
1차: NAV (순자산가치)
  - 토지: 감정평가 또는 공시지가 ÷ 현실화율
  - 건물: 재조달원가 - 감가상각
  - 개발사업: 사업성 분석 (DCF)
  - 부채: 장부가 (단기 부채 多)

2차: 개발 프로젝트 DCF
  - 분양 매출 추정
  - 공사비 및 기간
  - 할인율 10~15%

3차: PBR 검증
  - 부동산 개발사 PBR 0.5~1.5배
```

**주의사항:**
- 담보 대출 비율 확인
- 인허가 리스크
- 미분양 위험

#### 바이오/제약

**추천 조합: rNPV + 상대가치평가**

바이오는 파이프라인(신약 개발 단계)이 가치의 핵심이며, 성공 확률을 반영한 rNPV(risk-adjusted NPV)를 사용합니다.

**평가 프로세스:**
```
1차: rNPV (파이프라인별)
  파이프라인 가치 = Peak Sales × Net Margin × 시장 잔여 기간
                   × 성공 확률 ÷ (1+할인율)^출시 연도

  성공 확률 (평균):
  - 전임상: 10%
  - 임상 1상: 20%
  - 임상 2상: 40%
  - 임상 3상: 70%
  - 허가 신청: 90%

2차: EV/Pipeline 개수
  - 바이오텍 비교기업 분석

3차: M&A 거래 사례
  - 유사 기술이전 계약 금액
  - Upfront + Milestone 구조
```

**주의사항:**
- 임상 실패 시 가치 급락
- 특허 만료일 중요
- 파트너십/라이센싱 가능성

#### 금융업

**추천 조합: PBR + RIM (잔여이익모형)**

금융업은 자산=부채+자본 구조가 명확하고, 레버리지가 비즈니스 모델의 일부이므로 자기자본 기준 평가가 적합합니다.

**평가 프로세스:**
```
1차: PBR
  - 은행: PBR 0.3~0.8배 (국내 은행 저평가)
  - 증권: PBR 0.5~1.2배
  - 보험: PBR 0.4~1.0배

2차: RIM (Residual Income Model)
  자기자본가치 = BV + Σ[(ROE - Ke) × BV × (1+g)^t / (1+Ke)^t]

  - BV: 순자산가치
  - ROE: 자기자본이익률
  - Ke: 자기자본비용
  - g: 성장률

3차: P/E (보조)
  - 은행 PER 4~8배
```

**주의사항:**
- 대손충당금 적정성
- 자기자본비율(BIS) 규제
- 금리 리스크

### 2.3 기업 생명주기별 선택

#### 스타트업 (적자)

**추천: PSR + 사용자 지표 + VC Method**

현금흐름이 음수이고 순이익이 없어 전통적 평가법 적용 불가합니다.

**평가 요소:**
- 매출 성장률 (YoY 100%+ 목표)
- Burn Rate (월 소진 현금)
- Runway (생존 가능 기간)
- 시장 규모 (TAM > 1조원)
- 팀 역량

#### 성장기 (흑자 전환기)

**추천: DCF + PEG + Forward Multiple**

실적이 급성장하므로 미래 기준 평가가 중요합니다.

**평가 프로세스:**
```
1차: Forward PER (1년 후 예상 순이익 기준)
2차: PEG Ratio < 1.0 확인 (저평가 여부)
3차: DCF (고성장 5년 + 안정 성장 영구가치)
```

#### 성숙기 (안정 성장)

**추천: DCF + PER + 배당할인모형**

안정적 현금흐름과 배당으로 모든 평가법 적용 가능합니다.

**평가 프로세스:**
```
1차: DCF (성장률 GDP 수준 3~5%)
2차: PER (업종 평균)
3차: DDM (배당수익률 기준)
```

#### 쇠퇴기 (구조조정)

**추천: 청산가치 vs 회생가치 비교**

계속 기업으로 유지할지, 청산할지 판단이 핵심입니다.

**의사결정 프로세스:**
```
청산가치:
  자산 급매각 회수액 - 청산 비용 - 부채 = X억원

회생가치 (구조조정 후):
  DCF (매출 감소 반영) = Y억원

선택:
  If Y > X: 회생 추진
  If X > Y: 청산 권고
```

## 섹션 3: 교차 검증 방법

### 3.1 Football Field Chart

Football Field Chart는 여러 평가법의 결과를 시각적으로 비교하는 도구입니다.

**작성 방법:**

```
평가법          최소값     중간값     최대값
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DCF            [████████████████████████]  100~120억
EV/EBITDA      [██████████████████]        90~110억
PER            [███████████████████]       95~115억
NAV            [████████]                  80~85억
거래사례       [█████████████████████]     98~118억
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
종합 범위                                  95~110억
추천 가치                                  102억
```

**해석:**
- DCF와 거래사례가 비슷하면 신뢰도 높음
- NAV는 하한선 역할
- 중복 구간이 적정 가치 범위

### 3.2 가중평균 방법

복수 평가법 결과를 가중평균하여 최종 가치를 산정합니다.

**가중치 결정 기준:**

| 상황 | DCF | 상대가치 | 자산가치 | 근거 |
|------|-----|---------|---------|------|
| **M&A (제조업)** | 50% | 30% | 20% | DCF 가장 정교 |
| **IPO** | 30% | 60% | 10% | 시장 수용성 중요 |
| **부동산 기업** | 20% | 20% | 60% | 자산이 가치 핵심 |
| **스타트업** | 20% | 70% | 10% | 시장 비교 중심 |
| **법적 분쟁** | 30% | 20% | 50% | 객관적 증거 중요 |

**계산 예시:**
```
DCF:           110억원 × 50% = 55억원
EV/EBITDA:     100억원 × 30% = 30억원
NAV:            85억원 × 20% = 17억원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
최종 가치:                      102억원
```

### 3.3 Sanity Check (합리성 검증)

계산 결과가 상식적으로 타당한지 검증합니다.

**체크리스트:**

**1. 배수 비교**
```
산출된 PER = 최종가치 ÷ 순이익
→ 업종 평균 PER ±30% 범위 내인가?

산출된 PBR = 최종가치 ÷ 순자산
→ ROE > 자기자본비용이면 PBR > 1.0 맞는가?
```

**2. 성장률 검증**
```
DCF 매출 성장률 vs 업종 성장률
→ 업종보다 2배 이상 높다면 근거 있는가?
```

**3. 마진 검증**
```
EBITDA 마진 vs 업종 평균
→ 업종 대비 +5%p 이상이면 경쟁우위 명확한가?
```

**4. 자산 효율성**
```
매출 ÷ 총자산 = 자산회전율
→ 업종 평균과 비슷한가?
```

**5. 레버리지 검증**
```
순부채 ÷ EBITDA = 부채상환 배수
→ 3배 이하가 안전, 5배 이상이면 위험
```

### 3.4 민감도 분석

|------------------|-----|-----|-----|-----|

| **10%** | 85억 | 93억 | 102억 | 115억 |
| **12%** | 76억 | 82억 | 90억 | 100억 |
| **14%** | 69억 | 74억 | 81억 | 89억 |

**해석:**
