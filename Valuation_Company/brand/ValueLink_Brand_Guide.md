# ValueLink 브랜드 가이드

---

## 1. 브랜드 개요

### 1.1 브랜드명

| 구분 | 표기 |
|------|------|
| 영문 | **ValueLink** (붙여쓰기, 카멜케이스) |
| 한글 | **밸류링크** |

### 1.2 슬로건

> **"기업의 가치를 평가하여 투자자와 연결하다"**

### 1.3 브랜드 의미

**Value (가치)** + **Link (연결)**

- Value: 기업가치 평가
- Link: 투자자와의 연결

두 단어가 합쳐져 서비스의 본질을 담습니다.

### 1.4 서비스 정의

**AI 기반 기업가치평가 & 투자자 연결 플랫폼**

## 2. 로고 시스템

### 2.1 로고 컨셉

**V + L 조합 (3개의 선)**

```
 \      /
  \    /
   \  /
    \/______
```

### 2.2 로고의 의미 (2026-01-24 업데이트)

| 선 | 의미 | 색상 | 설명 |
|----|------|------|------|
| **선 1** (\\) | 평가 | 딥블루 | AI 기반 기업가치평가 |
| **선 2** (/) | 기업 | 딥그린 | 성장하는 스타트업 (👑 왕) - 가운데 위치 |
| **선 3** (\_) | 연결 | 앰버 | 투자자·파트너·서포터와 연결 |

### 2.3 브랜드 스토리 (2026-01-24 업데이트)

> "세 개의 선이 만나 하나의 가치를 완성합니다.
> 평가, 기업(왕 👑), 연결 — 밸류링크가 스타트업을 중심에 모십니다."

```
평가(딥블루) → 기업(딥그린) 👑 가운데 → 연결(앰버)
            투자자·파트너·서포터가 스타트업에게 달려옴
```

### 2.4 ValueLink 철학

> **"평가받은 스타트업 = 왕 👑"**
>
> 스타트업은 찾아다니지 않습니다.
> 투자자가, 파트너가, 서포터가 당신을 찾아옵니다.

### 2.5 로고 파일 (2026-01-24 업데이트)

| 파일명 | 용도 | 규격 |
|--------|------|------|
| `logo.svg` | 메인 로고 (심볼 + 텍스트) | viewBox="0 0 500 100" |

**로고 사양:**
- 심볼 크기: 40px 높이, stroke-width="12" (두껍게)
- 텍스트 크기: font-size="42", y="50" (세로 중앙 정렬)
- 텍스트 정렬: dominant-baseline="middle" (정확한 중앙)
- 슬로건: 포함하지 않음 (간결함 유지)

## 3. 컬러 시스템

### 3.1 Main Colors (3 메인 컬러)

세 가지 색상 모두 동등한 메인 컬러로 사용합니다.

| 구분 | 색상명 | HEX | RGB | 의미 |
|------|--------|-----|-----|------|
| **Main 1** | 딥그린 (Deep Green) | `#166534` | 22, 101, 52 | 기업, 성장 |
| **Main 2** | 딥블루 (Deep Blue) | `#1D4ED8` | 29, 78, 216 | 평가, 객관성, 정확성 |
| **Main 3** | 앰버 (Amber) | `#F59E0B` | 245, 158, 11 | 연결, 투자, 돈 |

### 3.2 색상 미리보기

```
■■■■■■  Main 1: 딥그린  #166534  (기업)
■■■■■■  Main 2: 딥블루  #1D4ED8  (평가)
■■■■■■  Main 3: 앰버    #F59E0B  (연결)
```

### 3.3 색상 관계

**Split-Complementary (분리 보색) 구조**

| 관계 | 색상각 차이 | 설명 |
|------|-------------|------|
| 딥블루 ↔ 앰버 | 183° | 거의 완벽한 보색 |
| 딥그린 ↔ 앰버 | 99° | 대비 관계 |
| 딥그린 ↔ 딥블루 | 84° | Cool 톤 인접 |

- Cool 톤 2개 (딥그린, 딥블루) + Warm 톤 1개 (앰버)
- 안정적인 Accent Color Scheme
