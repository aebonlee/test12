## 투자 수익률 (ROI) 분석

### 초기 투자 (부트스트래핑)
```
초기 투자: 150만원 (자체 자금)
3년 누적 순이익: 28.1억원

ROI = (28.1억원 - 0.015억원) / 0.015억원 × 100%
    = 187,333%

투자 회수 기간: 1개월 미만
```

### 외부 투자 가정 (시드 3억원)
```
시드 투자: 3억원 (1년차)
Equity: 20%
Exit: 3년차, 기업 가치 300억원

투자 수익 = 300억원 × 20% = 60억원
ROI = (60억원 - 3억원) / 3억원 × 100% = 1,900%

연환산 수익률 (CAGR): 300%
```
