## 시장 규모 분석 (TAM / SAM / SOM)

### TAM (Total Addressable Market)

**한국 기업가치평가 전체 시장**

```
┌──────────────────────────────────────────────┐
│  TAM: 연 3,000억원                             │
│                                              │
│  ├─ 상장 기업 평가: 1,500억원 (50%)           │
│  │   → 대형 회계법인 독점                     │
│  │                                            │
│  ├─ 비상장 기업 평가: 1,200억원 (40%)          │
│  │   → ValueLink 타겟 시장                   │
│  │                                            │
│  └─ M&A/상속세 평가: 300억원 (10%)             │
│      → 전문 평가기관 점유                      │
└──────────────────────────────────────────────┘
```

**산출 근거**:
- 한국 기업 수: 740만개 (통계청 2024)
- 비상장 기업: 738만개
- 연간 평가 필요 기업 (추정): 10,000개
  - 투자 유치: 5,000개
  - M&A: 2,000개
  - 상속/증여: 1,500개
  - 기타: 1,500개
- 평균 평가 단가: 3,000만원
- **시장 규모**: 10,000개 × 3,000만원 = **3,000억원**

### SAM (Serviceable Addressable Market)

**ValueLink이 실제로 공략 가능한 시장**

```
┌──────────────────────────────────────────────┐
│  SAM: 연 600억원 (TAM의 20%)                   │
│                                              │
│  타겟: 초기 ~ 성장 스타트업                     │
│  - 설립 1-10년차                              │
│  - 매출 5억 ~ 500억                           │
│  - 투자 유치 또는 M&A 준비                     │
│                                              │
│  기업 수: 5,000개                             │
│  평균 평가 단가: 1,200만원                      │
│  5,000개 × 1,200만원 = 600억원                │
└──────────────────────────────────────────────┘
```

**산출 근거**:
- 한국 스타트업 수: 약 15,000개 (중소벤처기업부 2024)
- 이 중 투자 유치 준비 기업: 5,000개 (33%)
- ValueLink 타겟 (저가 시장): 5,000개
- 평균 단가: 800만원 (DCF) + 400만원 (추가 서비스)

### SOM (Serviceable Obtainable Market)

**ValueLink이 3년 내 현실적으로 확보 가능한 시장**

```
┌──────────────────────────────────────────────┐
│  SOM: 연 36억원 (SAM의 6%)                     │
│                                              │
│  3년차 목표:                                  │
│  - 연간 평가 건수: 300개                       │
│  - 평균 단가: 1,200만원                        │
│  - 300개 × 1,200만원 = 36억원                 │
│                                              │
│  시장 점유율: 6% (SAM 기준)                    │
└──────────────────────────────────────────────┘
```

**성장 시나리오**:
- 1년차: 100개 × 1,000만원 = 10억원 (점유율 1.7%)
- 2년차: 200개 × 1,100만원 = 22억원 (점유율 3.7%)
- 3년차: 300개 × 1,200만원 = 36억원 (점유율 6.0%)

## 시장 트렌드 (Market Trends)

### 트렌드 1: 스타트업 투자 시장 성장

```
한국 VC 투자 규모 (단위: 조원)
2020년: 4.3조
2021년: 8.6조 (+100%)
2022년: 9.1조 (+6%)
2023년: 6.2조 (-32%, 경기 둔화)
2024년: 7.5조 (예상, +21%, 회복세)
2025년: 9.0조 (예상, +20%)
```

**영향**:
- 투자 증가 → 평가 수요 증가
- 2024-2025년 회복세 → 진입 타이밍 적절

### 트렌드 2: AI 도입 확대

```
AI 도입 기업 비율:
2022년: 15%
2023년: 25%
2024년: 40%
2025년 (예상): 60%
```

**영향**:
- AI 평가 수용도 증가
- "AI + Human" 모델 신뢰도 상승

### 트렌드 3: 플랫폼 경제 성장

```
플랫폼 이용 기업 비율:
2020년: 30%
2022년: 50%
2024년: 70%
```

**영향**:
- 온라인 평가 서비스 수용 증가
- 플랫폼 중개 비즈니스 성장

### 트렌드 4: 정보 투명성 요구 증가

```
투자자의 객관적 평가 요구:
2020년: 60%
2024년: 85%
```

**영향**:
- 랭킹 시스템 수요 증가
- 제3자 평가 신뢰도 상승

## 경쟁 환경 분석

### 직접 경쟁사

#### 1. 대형 회계법인 (Big 4)
**대상**: 삼일, 삼정, 안진, 한영

| 항목 | 회계법인 | ValueLink |
|------|---------|----------|
| **가격** | 1,500만 ~ 3,000만원 | 500만 ~ 2,000만원 |
| **납기** | 3-4주 | 5-10일 |
| **타겟** | 대기업, 상장 준비 기업 | 초기 ~ 성장 스타트업 |
| **강점** | 브랜드 신뢰도, 전문성 | 가격, 속도, 접근성 |
| **약점** | 높은 비용, 긴 납기 | 브랜드 인지도 부족 |

**차별화 전략**:
- 저가 시장 공략 (50억 미만 기업)
- AI 활용으로 비용 절감
- 플랫폼 네트워크 효과 (랭킹 + 매칭)

#### 2. 중소 회계법인
**대상**: 100인 미만 회계법인 (약 50개)

| 항목 | 중소 회계법인 | ValueLink |
|------|-------------|----------|
| **가격** | 800만 ~ 1,500만원 | 500만 ~ 2,000만원 |
| **납기** | 2-3주 | 5-10일 |
| **강점** | 지역 네트워크, 가격 | AI 기술, 플랫폼 |
| **약점** | 수작업, 표준화 부족 | 브랜드 인지도 |

**차별화 전략**:
- 온라인 완결 (오프라인 미팅 불필요)
- 표준화된 프로세스
- 추가 서비스 (랭킹, 매칭)

### 간접 경쟁사

#### 3. 투자자문사
**대상**: 기업 IR 자문, 평가 서비스 제공

| 항목 | 투자자문사 | ValueLink |
|------|-----------|----------|
| **서비스** | IR 자문 + 간단한 평가 | 정밀 평가 + 플랫폼 |
| **가격** | 500만 ~ 1,000만원 | 500만 ~ 2,000만원 |
| **강점** | 투자자 네트워크 | 평가 전문성, AI 기술 |

**협력 가능성**: 파트너십 (평가는 ValueLink, IR은 자문사)

#### 4. 자체 평가 (In-house)
**대상**: 기업 CFO/재무팀이 자체 평가

| 항목 | 자체 평가 | ValueLink |
|------|----------|----------|
| **비용** | 무료 (인건비만) | 유료 |
| **정확성** | 낮음 (편향 위험) | 높음 (제3자 검증) |
| **신뢰도** | 낮음 (투자자 불신) | 높음 (독립 평가) |

**차별화 전략**:
- 제3자 독립 평가 강조
- 투자자 신뢰도 확보

## 타겟 고객 분석

### 세그먼트 1: 초기 스타트업 (Seed ~ Series A)

**규모**:
- 추정 기업 수: 3,000개
- 연간 평가 니즈: 40% (1,200개)

**특징**:
- 설립 1-3년차
- 매출 5억 ~ 30억
- 직원 10-30명
- 투자 유치 금액: 5억 ~ 30억

**Pain Points**:
- 제한된 예산 (대형 회계법인 이용 어려움)
- 빠른 평가 필요 (투자 유치 타이밍)
- 투자자 설득 자료 필요

**ValueLink 적합도**: ⭐⭐⭐⭐⭐ (5/5)

### 세그먼트 2: 성장 스타트업 (Series B ~ Pre-IPO)

**규모**:
- 추정 기업 수: 2,000개
- 연간 평가 니즈: 50% (1,000개)

**특징**:
- 설립 3-7년차
- 매출 30억 ~ 300억
- 직원 30-100명
- 투자 유치 금액: 30억 ~ 300억

**Pain Points**:
- 정밀한 평가 필요 (대규모 투자 유치)
- IPO 준비 (공모가 산정)
- 투자자 IR 부담

**ValueLink 적합도**: ⭐⭐⭐⭐ (4/5)

### 세그먼트 3: 투자자 (VC, 엔젤)

**규모**:
- VC: 약 200개
- 엔젤 투자자: 약 1,000명

**특징**:
- 연간 투자 건수: VC 10-50건, 엔젤 1-5건
- 투자 규모: VC 10억-100억, 엔젤 5천만-5억

**Pain Points**:
- 딜소싱 비용 (기업 발굴 어려움)
- 정보 비대칭 (검증된 정보 부족)
- 비교 분석 어려움 (객관적 기준 부족)

**ValueLink 가치 제안**:
- 무료 랭킹 접근
- 검증된 기업 정보 (평가 완료)
- 객관적 비교 가능

## 시장 진입 전략

### 1단계: Early Adopter 확보 (0-6개월)

**타겟**: 혁신적인 초기 스타트업 30개

**전략**:
- 프로모션: 첫 10개 기업 30% 할인
- 파트너십: VC/액셀러레이터와 제휴
- 사례 연구: 성공 사례 3개 이상 확보

**목표**:
- 월 5건 평가 수주
- 고객 만족도 4.5/5.0 이상

### 2단계: 초기 다수 확보 (6-12개월)

**타겟**: 초기 ~ 성장 스타트업 100개

**전략**:
- 콘텐츠 마케팅: 유튜브 채널 (평가 교육)
- SEO 최적화: "기업가치평가" 검색 1페이지 진입
- 랭킹 시스템 론칭: 투자자 유입 증대

**목표**:
- 월 10건 평가 수주
- 100개 기업 랭킹 등재
- 200명 투자자 유치

### 3단계: 시장 확장 (12-24개월)

**타겟**: 시장 점유율 3-5%

**전략**:
- 매칭 사업 론칭: 투자자-기업 연결
- 파트너십 확대: 법률/특허 법인
- 글로벌 진출 준비: 동남아 시장 조사

**목표**:
- 월 15건 평가 수주
- 월 1건 매칭 성공

## 시장 기회 요약

### 긍정적 요인 (Opportunities)

1. **시장 성장**: 스타트업 투자 시장 연 20% 성장
2. **AI 수용**: AI 서비스 수용도 증가
3. **정보 투명성 요구**: 객관적 평가 니즈 증가
4. **플랫폼 경제**: 온라인 서비스 선호 증가

### 리스크 요인 (Threats)

1. **경기 변동**: 투자 시장 둔화 시 평가 수요 감소
2. **경쟁 심화**: Big 4 저가 시장 진입 가능성
3. **규제 변화**: 평가 업계 규제 강화 가능성

### 대응 전략

**리스크 1 대응**: 다각화 (평가 + 랭킹 + 매칭)
**리스크 2 대응**: 선점 효과 + 플랫폼 네트워크 효과
**리스크 3 대응**: 회계사 협력 체계 유지
