**작성일**: 2026-02-05
**버전**: 1.0
**프로젝트**: ValueLink - AI 기반 기업가치평가 플랫폼

---

## 비즈니스 모델 개요

ValueLink는 **플랫폼 비즈니스 모델**로, 기업과 투자자를 연결하여 양측 모두에게 가치를 제공합니다.

```
┌──────────────┐         ┌─────────────┐         ┌──────────────┐
│   기업       │────────→│  ValueLink  │←────────│   투자자     │
│              │         │  Platform   │         │              │
│ (평가 수수료)│         │             │         │ (무료 접근)  │
└──────────────┘         └─────────────┘         └──────────────┘
                               │
                               ↓
                        매칭 성공 시
                     중개 수수료 (1-3%)
```

## 수익 모델 (Revenue Model)

### 1차 수익원: 기업가치 평가 서비스

| 평가 방법 | 가격 | 납기 | 타겟 고객 |
|----------|------|------|----------|
| **DCF 평가** | 800만원 | 7-10일 | 투자유치, M&A 기업 |
| **상대가치 평가** | 500만원 | 5-7일 | 간편 비교 평가 기업 |
| **상증법 평가** | 1,000만원 | 10-14일 | 가업승계, 증여 기업 |
| **IPO 평가** | 2,000만원 | 14-21일 | 상장 준비 기업 |
| **자산가치 평가** | 600만원 | 7-10일 | 자산 중심 기업 |

**평균 단가**: 800만원
**예상 객단가 분포**:
- DCF (40%): 320만원
- Relative (30%): 150만원
- Tax (15%): 150만원
- IPO (10%): 200만원
- Asset (5%): 30만원

**월 목표 (단계별)**:
- 3개월: 5건 × 800만원 = 4,000만원
- 1년: 10건 × 800만원 = 8,000만원
- 3년: 15건 × 800만원 = 1.2억원

### 2차 수익원: AI Avatar IR 서비스

```
월 이용료: 200만원
최소 계약: 3개월
타겟: 투자 유치 진행 중인 기업 (평가 완료 기업)
```

**기능**:
- 24시간 투자자 대응
- 다국어 지원 (한/영/중/일)
- 맞춤형 Q&A 데이터베이스
- 대화 분석 리포트

**월 목표 (단계별)**:
- 3개월: 2개 기업 × 200만원 = 400만원
- 1년: 5개 기업 × 200만원 = 1,000만원
- 3년: 10개 기업 × 200만원 = 2,000만원

### 3차 수익원: 투자 매칭 중개 수수료

```
수수료율: 투자 금액의 1-3%
평균: 2%
```

**예시**:
- 투자 금액 10억원 → 수수료 2,000만원
- 투자 금액 20억원 → 수수료 4,000만원

**매칭 프로세스**:
1. 기업이 평가 완료 → 랭킹 등재
2. 투자자가 관심 표명
3. ValueLink가 양측 연결
4. 투자 계약 체결 시 수수료 발생

**월 목표 (단계별)**:
- 3개월: 0건 (랭킹 시스템 미구축)
- 1년: 월 0.3건 × 평균 2,000만원 = 600만원
- 3년: 월 1건 × 평균 3,000만원 = 3,000만원

### 4차 수익원: 프리미엄 노출 (광고)

```
월 이용료: 50-100만원
타겟: 랭킹 상위 노출 원하는 기업
```

**기능**:
- 랭킹 페이지 상단 노출
- 메인 페이지 Featured 섹션 노출
- 투자자 대시보드 추천 섹션 노출

**월 목표 (단계별)**:
- 3개월: 0건 (랭킹 시스템 미구축)
- 1년: 3개 기업 × 70만원 = 210만원
- 3년: 10개 기업 × 70만원 = 700만원

### 5차 수익원: 상세 보고서 판매 (투자자 대상)

```
가격: 10만원 / 1개 기업
타겟: 투자자
```

**내용**:
- 평가 보고서 요약본 (5-10페이지)
- 재무 하이라이트
- 리스크 분석
- 투자 포인트

**월 목표 (단계별)**:
- 3개월: 0건 (랭킹 시스템 미구축)
- 1년: 월 20건 × 10만원 = 200만원
- 3년: 월 50건 × 10만원 = 500만원

→ **월 3건 이상 평가 수주 시 흑자 전환**

## 비즈니스 모델 캔버스

| **성장 스타트업** | 매출 50-500억, Series B~Pre-IPO | 정밀 평가, IR 지원 |

**기업 대상**:
- 합리적 가격 (기존 대비 50% 저렴)

- 투자자 자동 노출 (랭킹 등재)

**투자자 대상**:
- 무료 접근
- 검증된 기업 정보 (평가 완료)
- 객관적 비교 (랭킹 시스템)

### 3. 채널 (Channels)

**온라인 채널 (주력)**:

## 경쟁 우위 (Competitive Advantages)

### 1. 가격 경쟁력
```
전통 회계법인: 1,500만 ~ 3,000만원
ValueLink: 500만 ~ 2,000만원
→ 50% 저렴
```

### 2. 시간 경쟁력
```
전통 회계법인: 3-4주
ValueLink: 5-10일
→ 70% 단축
```

### 3. 플랫폼 네트워크 효과
```
평가 기업 ↑ → 투자자 유입 ↑ → 매칭 기회 ↑ → 더 많은 기업 유입
```

### 4. 데이터 축적
```
평가 건수 ↑ → 벤치마크 데이터 ↑ → 평가 정확도 ↑ → 신뢰도 ↑
```

```
목표: 월 5건 평가 수주
전략:
```
