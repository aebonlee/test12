# ValueLink 비전 및 미션

**작성일**: 2026-02-05
**버전**: 1.0
**프로젝트**: ValueLink - AI 기반 기업가치평가 플랫폼

---

## 비전 (Vision)

> **"기업가치평가를 넘어 투자 생태계를 연결하는 플랫폼"**

### 장기 비전 (5-10년)

ValueLink는 단순한 기업가치 평가 서비스를 넘어, **기업-투자자-전문가**를 연결하는 **투자 생태계의 중심 플랫폼**이 되고자 합니다.

```
현재: 기업가치평가 서비스
  ↓
단기: 평가 + 랭킹 시스템
  ↓
중기: 평가 + 랭킹 + 투자자 매칭
  ↓
장기: 투자 생태계 플랫폼 (평가 → 랭킹 → 매칭 → 투자 실행)
```

---

## 미션 (Mission)

### 1. 기업을 위한 미션
**"객관적이고 신뢰할 수 있는 기업가치 평가를 제공한다"**

- AI와 전문 회계사가 협력하여 정확한 평가 제공
- 5가지 평가 방법을 통한 다각도 분석
- 합리적인 가격과 빠른 납기 (기존 회계법인 대비 50% 비용 절감)

### 2. 투자자를 위한 미션
**"검증된 투자 기회를 발굴하고 연결한다"**

- 객관적 평가 기반 기업 랭킹 시스템
- 무료 접근 (정보 비대칭 해소)
- AI 매칭으로 최적의 투자처 발견

### 3. 시장을 위한 미션
**"투명하고 효율적인 투자 생태계를 만든다"**

- 정보 투명성 증대 (랭킹 공개)
- 거래 비용 감소 (직접 매칭)
- 시장 효율성 향상 (데이터 기반 의사결정)

## 핵심 가치 (Core Values)

### 1. 정확성 (Accuracy)
```
AI 초안 생성 (60%) + 전문 회계사 검증 (40%)
= 신뢰할 수 있는 평가 결과
```

**실천 방안:**
- 5개 평가 엔진의 정밀한 알고리즘
- 회계사 2중 검증 시스템
- 평가 방법론 지속적 개선

### 2. 투명성 (Transparency)
```
평가 기준 공개 + 랭킹 알고리즘 공개
= 신뢰 구축
```

**실천 방안:**
- 랭킹 계산 방식 공개 (종합 점수 100점 만점)
- 평가 프로세스 투명 공개
- 데이터 소스 명시

### 3. 접근성 (Accessibility)
```
합리적 가격 + 빠른 납기 + 온라인 완결
= 누구나 이용 가능
```

**실천 방안:**
- 기존 회계법인 대비 50% 저렴한 가격
- 평균 5-10일 납기 (업계 평균 3-4주)
- 온라인으로 모든 프로세스 완결

### 4. 혁신성 (Innovation)
```
AI 기술 + 전문 지식
= 새로운 가치 창출
```

**실천 방안:**
- AI 초안 생성으로 작업 시간 70% 단축
- 24시간 AI Avatar IR 서비스
- 스마트 매칭 알고리즘

## 문제 정의 (Problem Statement)

### 기존 시장의 문제점

#### 1. 기업 입장
```
❌ 높은 비용: 평가 비용 1,500만원 ~ 3,000만원
❌ 긴 납기: 평균 3-4주 소요
❌ 접근성 낮음: 대형 회계법인만 가능
❌ 투자자 연결 부재: 평가 후 투자 유치는 별개
```

#### 2. 투자자 입장
```
❌ 정보 비대칭: 검증된 기업 정보 부족
❌ 딜소싱 비용: 기업 발굴에 많은 시간/비용
❌ 평가 신뢰도: 기업 자체 평가는 신뢰하기 어려움
❌ 비교 어려움: 객관적 비교 기준 없음
```

#### 3. 시장 입장
```
❌ 시장 비효율: 정보 비대칭으로 거래 비용 ↑
❌ 투명성 부족: 기업 가치 정보 비공개
❌ 매칭 비효율: 적합한 투자자-기업 연결 어려움
```

## 해결책 (Solution)

### ValueLink의 접근 방식

#### 1. AI + Human 협업 모델
```
┌─────────────────────────────────────┐
│  AI (60%)                           │
│  - 데이터 수집 및 분석              │
│  - 재무모델 자동 생성               │
│  - 초안 작성 (20-30 페이지)         │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  전문 회계사 (40%)                   │
│  - 정확성 검증                       │
│  - 전문가 판단 적용                  │
│  - 최종 승인 (22개 승인 포인트)      │
└─────────────────────────────────────┘
              ↓
       신뢰할 수 있는 평가 보고서
```

**효과:**
- 비용 50% 절감 (AI 자동화)
- 납기 70% 단축 (평균 5-10일)
- 정확성 유지 (전문가 검증)

#### 2. 평가 → 랭킹 → 매칭의 선순환
```
Step 1: 평가 완료
  → 기업의 객관적 가치 산정
  ↓
Step 2: 랭킹 등재
  → 업종별/규모별 순위 공개
  → 투자자에게 자동 노출
  ↓
Step 3: 투자자 매칭
  → AI 알고리즘으로 최적 매칭
  → 직접 연결 (플랫폼 중개)
  ↓
Step 4: 투자 실행
  → 투자 계약 체결
  → 플랫폼 수수료 수익
```

#### 3. 3가지 핵심 서비스

| 서비스 | 설명 | 대상 | 수익 모델 |
|--------|------|------|----------|
| **Valuation** | 5가지 평가법 적용 기업가치 평가 | 기업 | 평가 수수료 (500만-2,000만원) |
| **Ranking** | 객관적 기준 기반 기업 랭킹 시스템 | 투자자 | 무료 (투자자 유입) + 프리미엄 노출 (기업) |
| **Matching** | AI 기반 투자자-기업 매칭 | 기업+투자자 | 중개 수수료 (투자 금액의 1-3%) |

## 타겟 고객 (Target Customers)

### 1차 타겟: 초기 스타트업 (Seed ~ Series A)

**고객 프로필:**
- 설립 1-5년차 스타트업
- 매출 5억 ~ 50억
- 투자 유치 준비 중
- 예상 가치 50억 ~ 500억

**Pain Points:**
- 객관적 가치 평가 필요
- 투자자 설득 자료 필요
- 예산 제약 (대형 회계법인 이용 어려움)

**ValueLink 솔루션:**
- DCF/Relative 평가 (500-800만원)
- 합리적 가격
- 평가 후 랭킹 자동 등재 → 투자자 노출

### 2차 타겟: 성장 스타트업 (Series B ~ Pre-IPO)

**고객 프로필:**
- 설립 5-10년차
- 매출 50억 ~ 500억
- 추가 투자 또는 IPO 준비
- 예상 가치 500억 ~ 5,000억

**Pain Points:**
- 정밀한 가치 평가 필요
- IPO/M&A 대비 평가 필요
- 투자자 관계 관리

**ValueLink 솔루션:**
- DCF/IPO 평가 (800-2,000만원)
- AI Avatar IR (투자자 24시간 응대)
- 투자자 관심도 리포트

### 3차 타겟: 투자자 (VC, 엔젤, 기관)

**고객 프로필:**
- 초기 단계 투자자 (Seed ~ Series A)
- 연간 투자 건수 3-10건
- 딜소싱 어려움

**Pain Points:**
- 검증된 딜소싱 필요
- 객관적 비교 자료 부족
- 기업 접근 어려움

**ValueLink 솔루션:**
- 무료 랭킹 접근
- 검증된 기업 정보 (평가 완료)
- AI 매칭으로 최적 투자처 발견
- 직접 컨택 가능

## 경쟁 우위 (Competitive Advantages)

### 1. AI + Human 하이브리드
```
전통 회계법인:
- 100% 수작업 → 비용 ↑, 시간 ↑

ValueLink:
- AI 60% + Human 40% → 비용 ↓, 시간 ↓, 정확성 유지
```

### 2. 플랫폼 네트워크 효과
```
평가 기업 ↑ → 랭킹 신뢰도 ↑ → 투자자 유입 ↑ → 매칭 기회 ↑
→ 더 많은 기업 유입 (선순환)
```

### 3. 독자 기술 (5개 평가 엔진)
```
DCF Engine (504줄)
Relative Engine (487줄)
Asset Engine (497줄)
Intrinsic Engine (258줄)
Tax Engine (379줄)
→ 총 3,559줄 Python 코드 (영업 비밀)
```

### 4. 데이터 축적
```
평가 건수 ↑ → 업종별 벤치마크 데이터 축적
→ 평가 정확도 향상
→ 시장 인사이트 제공 가능
```

## 비전 실현 전략

### Phase 1: MVP (현재)
```
✅ 5개 평가 엔진 개발 완료
✅ 14단계 워크플로우 구축
✅ Supabase 백엔드 연동
⏳ Next.js Frontend 재구축 (진행 중)
```

### Phase 2: 랭킹 시스템 (3-6개월)
```
- 평가 완료 기업 자동 랭킹 등재
- 업종별/규모별 필터링
- 투자자 무료 접근
```

### Phase 3: 매칭 시스템 (6-12개월)
```
- 투자자 프로파일 설정
- AI 매칭 알고리즘
- 직접 컨택 기능
```

### Phase 4: 생태계 확장 (12개월+)
```
- AI Avatar IR 고도화
- 투자 계약 자동화
- 글로벌 확장 (동남아시아)
```
