### 1. 사용자 매뉴얼

**파일**: `docs/user-manual.md`

```markdown
# ValueLink 사용자 매뉴얼

## 목차

1. [시작하기](#시작하기)
2. [프로젝트 생성](#프로젝트-생성)

5. [보고서 다운로드](#보고서-다운로드)
6. [마이페이지](#마이페이지)

## 시작하기

### 회원가입

1. 홈페이지에서 **시작하기** 버튼 클릭
2. 이메일, 비밀번호, 이름, 기업명 입력
3. **회원가입** 버튼 클릭
4. 이메일 인증 (선택사항)

### 로그인

1. **로그인** 버튼 클릭
2. 이메일, 비밀번호 입력
3. 로그인 완료 시 마이페이지로 이동

---

## 프로젝트 생성

### 1. 프로젝트 목록 페이지 접속

- 상단 메뉴 **내 프로젝트** 클릭
- 또는 마이페이지에서 **새 프로젝트** 버튼 클릭

### 2. 프로젝트 정보 입력

- **프로젝트명**: 예) ABC 스타트업 기업가치평가
- **평가 방법 선택**:
  - DCF (현금흐름할인법) - 800만원
  - Relative (상대가치평가) - 500만원
  - Asset (자산가치평가) - 600만원
  - Intrinsic (내재가치평가) - 600만원
  - Tax (세법상평가) - 1,000만원

### 3. 프로젝트 생성 완료

- **프로젝트 생성** 버튼 클릭
- 프로젝트 상세 페이지로 자동 이동

---

## 평가 신청

### 1. 평가 방법별 신청 폼 작성

#### DCF 평가 신청

1. 프로젝트 정보 입력 (프로젝트명, 기업명, 산업분야)
2. 5년 매출 예측 입력
3. 주요 가정 입력:
   - 영업이익률
   - 법인세율
   - 가중평균자본비용 (WACC)
   - 영구성장률
   - 순부채
   - 발행주식수
4. **제출하기** 버튼 클릭

#### Relative 평가 신청

1. 프로젝트 정보 입력
2. 재무정보 입력 (매출, EBITDA)
3. 유사기업 추가:
   - 기업명
   - 매출 배수 (P/S)
   - EBITDA 배수 (EV/EBITDA)
4. **제출하기** 버튼 클릭

(나머지 평가 방법도 유사한 패턴)

---

## 진행 상황 확인

### 14단계 워크플로우

1. **프로젝트 생성** - 프로젝트 정보 입력
2. **견적 요청** - 견적서 확인
3. **협상** - 가격 협상 (선택사항)
4. **문서 업로드** - 재무제표 등 업로드
5. **평가 진행** - 회계사가 평가 수행
6. **데이터 수집** - 추가 데이터 요청
7. **회계사 검토** - 전문가 검토
8. **초안 생성** - 보고서 초안 작성
9. **초안 확인** - 고객 검토
10. **수정 요청** - 수정 사항 전달
11. **최종 준비** - 최종 보고서 준비
12. **최종 보고서** - 최종본 확인
13. **결제** - 무통장 입금
14. **보고서 다운로드** - PDF 다운로드

### 진행 상황 확인 방법

- 프로젝트 상세 페이지에서 **진행 상황 보기** 클릭
- 사이드바에서 현재 단계 확인
- 다음 단계로 진행하려면 현재 단계 완료 필요

---

## 보고서 다운로드

### 1. 프로젝트 완료 확인

- Step 14 (보고서 다운로드) 도달 확인
- 결제 완료 확인

### 2. 다운로드

- 프로젝트 상세 페이지에서 **보고서 다운로드** 버튼 클릭
- PDF 파일 자동 다운로드
```

### 2. FAQ 문서

**파일**: `docs/faq.md`

```markdown
# 자주 묻는 질문 (FAQ)

## 회원가입 및 로그인

### Q1. 회원가입이 필요한가요?
네, ValueLink의 모든 기능을 이용하려면 회원가입이 필요합니다.

### Q2. 비밀번호를 잊어버렸어요.
로그인 페이지에서 **비밀번호 찾기** 링크를 클릭하여 재설정할 수 있습니다.

### Q3. 회원 탈퇴는 어떻게 하나요?
마이페이지 → 설정 → 계정 삭제에서 탈퇴할 수 있습니다.

---

## 평가 방법

### Q4. 어떤 평가 방법을 선택해야 하나요?
- **DCF**: 안정적인 현금흐름이 예측 가능한 기업
- **Relative**: 유사기업이 많은 산업
- **Asset**: 자산 중심 기업 (부동산, 제조업 등)
- **Intrinsic**: 성장률이 중요한 기업
- **Tax**: 세법상 평가가 필요한 경우 (증여, 상속 등)

### Q5. 여러 평가 방법을 동시에 신청할 수 있나요?
네, 프로젝트를 여러 개 생성하면 가능합니다.

### Q6. 평가 비용은 얼마인가요?
- DCF: 800만원
- Relative: 500만원
- Asset: 600만원
- Intrinsic: 600만원
- Tax: 1,000만원

---

## 프로젝트 진행

### Q7. 평가 기간은 얼마나 걸리나요?
평균 10일 정도 소요됩니다. (영업일 기준)

### Q8. 진행 상황을 어떻게 확인하나요?
프로젝트 상세 페이지에서 **진행 상황 보기** 버튼을 클릭하면 14단계 중 현재 단계를 확인할 수 있습니다.

### Q9. 수정 요청은 몇 번까지 가능한가요?
초안 확인 후 1회 수정 요청이 기본으로 포함됩니다. 추가 수정은 별도 협의가 필요합니다.

---

## 결제

### Q10. 어떤 결제 수단을 지원하나요?
현재 무통장 입금만 지원합니다.

### Q11. 입금 후 언제 확인되나요?
영업일 기준 1~2일 내 확인됩니다.

### Q12. 세금계산서 발급이 가능한가요?
네, 입금 확인 후 이메일로 자동 발송됩니다.

### Q13. 환불이 가능한가요?
평가 시작 전에는 전액 환불 가능합니다. 평가 시작 후에는 진행 단계에 따라 부분 환불이 가능합니다.

---

## 보고서

### Q14. 보고서 형식은 어떻게 되나요?
PDF 형식으로 제공되며, 평가 방법별로 상세한 계산 과정과 근거가 포함됩니다.

### Q15. 보고서를 분실했어요.
프로젝트 상세 페이지에서 언제든지 다시 다운로드할 수 있습니다.

### Q16. 보고서를 수정할 수 있나요?
Step 10 (수정 요청) 단계에서 수정 요청을 제출하면 회계사가 검토 후 수정합니다.

---

## 기술 문제

### Q17. 파일 업로드가 안 돼요.
- 파일 크기가 50MB를 초과하지 않는지 확인해주세요.
- 지원 형식: PDF, XLSX, DOCX, JPG, PNG

### Q18. 페이지가 로딩되지 않아요.
- 브라우저 캐시를 삭제해주세요.
- 다른 브라우저로 시도해주세요.
- 문제가 지속되면 support@valuation.ai.kr로 문의해주세요.

---

## 기타

### Q19. 개인정보는 안전한가요?
네, 모든 데이터는 암호화되어 저장되며, RLS (Row Level Security) 정책으로 본인 데이터만 접근 가능합니다.

### Q20. 회계사는 누구인가요?
공인회계사 자격을 보유한 전문가가 직접 평가합니다.

### Q21. 영업 시간은 언제인가요?
평일 09:00 - 18:00 (주말 및 공휴일 제외)

### Q22. 외국 기업도 평가 가능한가요?
현재는 한국 기업만 지원합니다. 추후 확대 예정입니다.
```
