# ValueLink 사용자 여정 (User Journeys)

**작성일**: 2026-02-05
**버전**: 1.0

---

## 1. 고객사 여정 (Customer Journey)

### 전체 여정 개요

```
발견 → 신청 → 견적 → 계약 → 자료 제출 → 대기 → 검토 → 수정 → 승인 → 다운로드 → 사용
```

**소요 시간**: 평균 7-10일

---

### Phase 1: 발견 및 신청 (Discovery & Request)

**소요 시간**: 1일

#### 1-1. 웹사이트 방문
```
사용자: 스타트업 CEO
니즈: 투자 유치를 위한 기업가치 평가 필요

행동:
1. Google 검색 "기업가치평가 저렴한"
2. ValueLink 웹사이트 방문
3. 서비스 안내 페이지 탐색
   - 5가지 평가 방법 비교
   - 가격표 확인 (500만~2,000만원)
   - 기존 대비 50% 저렴 확인
```

#### 1-2. 회원가입 및 로그인
```
페이지: login.html, register.html

행동:
1. "무료 견적 받기" 버튼 클릭
2. 회원가입 (이메일, 비밀번호)
3. 이메일 인증
4. 로그인
```

#### 1-3. 프로젝트 생성
```
페이지: projects/project-create.html

입력 정보:
- 기업명: 테크이노
- 사업자등록번호: 123-45-67890
- 대표자명: 김철수
- 평가 목적: 투자 유치
- 평가 방법 선택: DCF + 상대가치 (체크박스)
- 목표 완료일: 2주 후

결과:
- project_id 생성: PROJ-20260205-001
- 견적서 자동 생성
```

---

### Phase 2: 견적 및 계약 (Quote & Contract)

**소요 시간**: 1-2일

#### 2-1. 견적서 확인
```
페이지: core/project-detail.html

견적 내용:
- DCF 평가: 800만원
- 상대가치 평가: 500만원
- 합계: 1,300만원
- 할인 (첫 고객 프로모션): -130만원 (10%)
- 최종 금액: 1,170만원
- 지불 조건: 계약금 20% (234만원), 잔금 80% (936만원)
- 유효기간: 7일

행동:
1. 견적서 검토
2. (선택) 협상 요청 (할인, 일정 조정)
3. 견적 승인 버튼 클릭
```

#### 2-2. 계약금 결제
```
페이지: valuation/payment.html

결제 방법:
- 신용카드
- 계좌이체
- 가상계좌

행동:
1. 결제 수단 선택
2. 234만원 결제
3. 결제 완료 알림 수신 (이메일)
```

---

### Phase 3: 자료 제출 (Document Upload)

**소요 시간**: 1-2일

#### 3-1. 필수 자료 업로드
```
페이지: valuation/data-collection.html

필수 자료:
✅ 재무제표 3개년 (2021-2023)
✅ 사업계획서
✅ 등기부등본
✅ 주주명부

선택 자료:
⬜ 감정평가서 (있으면 더 정확)
⬜ 사업보고서

행동:
1. 파일 드래그 앤 드롭 업로드
2. 각 파일 유형 선택
3. 업로드 완료 확인
4. "제출" 버튼 클릭
```

#### 3-2. 제출 완료
```
알림:
"자료 제출이 완료되었습니다.
평가는 약 5-7일 소요되며, 진행 상황을 이메일로 안내드립니다."

이메일:
- 제목: [ValueLink] 자료 제출 완료
- 내용: 담당 회계사 배정 안내, 예상 완료일
```

---

### Phase 4: 평가 진행 (Valuation In Progress)

**소요 시간**: 3-5일 (고객 대기)

#### 4-1. 진행 상황 모니터링
```
페이지: valuation/evaluation-progress.html

진행 표시:
┌────────────────────────────────────────┐
│ Step 5: 데이터 수집 [완료] ✅          │
│ Step 6: 평가 실행 [진행 중] 30% 🔄    │
│ Step 7: 회계사 검토 [대기]            │
└────────────────────────────────────────┘

실시간 업데이트 (WebSocket):
- "재무 데이터 추출 중..."
- "DCF 모델 생성 중..."
- "비교 기업 선정 중..."
```

#### 4-2. 알림 수신
```
이메일 알림:
- Step 5 완료: "데이터 수집이 완료되었습니다"
- Step 6 완료: "평가 실행이 완료되었습니다"
- Step 7 대기: "회계사 검토 대기 중입니다 (1-2일 소요)"
```

---

### Phase 5: 초안 검토 (Draft Review)

**소요 시간**: 1일

#### 5-1. 초안 확인 알림
```
이메일:
- 제목: [ValueLink] 평가 초안이 완료되었습니다
- 내용: "초안 보고서를 확인하고 피드백을 주세요"
- 링크: 초안 확인 페이지
```

#### 5-2. 초안 보고서 검토
```
페이지: valuation/report-draft.html

보고서 구성:
- 표지: 기업명, 평가일
- 요약: 평가 가치 범위 (50억 ~ 80억원)
- DCF 결과: 기업가치 70억원
- 상대가치 결과: 기업가치 60억원
- 차트: 5년 매출 예측, 멀티플 비교
- 부록: 재무제표, 가정 사항

행동:
1. PDF 뷰어로 보고서 열람
2. (선택) 수정 요청 버튼 클릭
3. 또는 승인 버튼 클릭
```

---

### Phase 6: 수정 요청 (Revision Request) - 선택

**소요 시간**: 1-2일

#### 6-1. 수정 사항 입력
```
페이지: valuation/revision-request.html

수정 요청 예시:
┌────────────────────────────────────────┐
│ 수정 유형: 수치 변경                    │
│                                        │
│ 내용:                                  │
│ "DCF 성장률이 너무 보수적입니다.       │
│  2024년 매출 성장률을 20%에서 25%로    │
│  조정 부탁드립니다."                   │
│                                        │
│ [제출]                                 │
└────────────────────────────────────────┘

제한:
- 최대 2회 수정 요청 가능
- 수치 변경은 재승인 필요 (회계사)
```

#### 6-2. 수정 완료 알림
```
이메일:
- 제목: [ValueLink] 수정 완료
- 내용: "수정 사항이 반영되었습니다. 다시 확인해주세요"
```

### Phase 7: 최종 승인 및 결제 (Final Approval & Payment)

**소요 시간**: 1일

#### 7-1. 최종 보고서 승인

```
페이지: valuation/report-final.html

행동:
1. 최종 보고서 검토
2. "승인" 버튼 클릭
3. 잔금 결제 페이지로 이동
```

#### 7-2. 잔금 결제
```
페이지: valuation/balance-payment.html

결제 금액: 936만원 (80%)

행동:
1. 결제 수단 선택
2. 결제 완료
3. 다운로드 페이지로 이동
```

### Phase 8: 다운로드 및 사용 (Download & Use)

**소요 시간**: 즉시

#### 8-1. 최종 보고서 다운로드
```
페이지: valuation/report-download.html

제공 파일:
- 평가보고서_테크이노_20260215.pdf (워터마크 없음)
- 엑셀 계산 파일 (DCF 모델)
- 요약 슬라이드 (PPT, 10페이지)

행동:
1. 다운로드 버튼 클릭
2. 파일 저장
3. (선택) 추가 사본 요청
```

#### 8-2. 활용
```
고객 활용 방법:
- 투자자 IR 자료로 사용
- 내부 의사결정 자료
- 재평가 참고 자료 (1년 후)
```


---

```
이메일:
- 제목: [ValueLink] 새 프로젝트 배정
- 내용:
  * 프로젝트 ID: PROJ-20260205-001
  * 기업명: 테크이노
  * 평가 방법: DCF + 상대가치
```

```
대시보드:

- 대기 중인 승인: 1건 (PROJ-20260205-001)
```

## 6. 감정 곡선 (Emotion Curve)

```
고객 만족도

😊 높음 |     ●                   ●           ●
       |    /                   /           /
       |   /                   /           /
😐 중간 | ●                 ●             ●
       |  \               /
       |   \             /
😟 낮음 |    ●─────────●
       |
       └──────────────────────────────────────> 시간
         발견 신청 대기 초안 수정 완료

높은 만족 시점:
- 견적 확인 (저렴함에 놀람)
- 초안 확인 (빠른 진행에 만족)
- 최종 다운로드 (완성도에 만족)

낮은 만족 시점:
- 대기 기간 (3-5일)
- 수정 대기 (1-2일)

개선 기회:
- 대기 중 진행 상황 실시간 알림
- 예상 완료일 명확히 표시
```
