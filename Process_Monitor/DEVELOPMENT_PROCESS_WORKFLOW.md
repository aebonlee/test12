## 개요

이 프로젝트는 **10단계 프로세스**로 구성됩니다:
- **Special (특별단계)**: P0 (작업 디렉토리), S0 (SAL Grid)
- **Preliminary (예비단계)**: P1 → P2 → P3 (사업 기획 → 검증된 프로토타입)
- **Stage (실행단계)**: S1 → S2 → S3 → S4 → S5 (개발 → 운영)

## 진행률 관리 시스템

### JSON 파일: `phase_progress.json`

경로: `Development_Process_Monitor/data/phase_progress.json`

```json
{
  "project_id": "YOUR_PROJECT",
  "phases": {
    "P0": { "progress": 100 },
    "P1": { "progress": 100 },
    ...
  }
}
```

### 프론트엔드 로드

`index.html`의 `loadPhaseProgressFromDB()` 함수가 JSON 파일에서 자동으로 진행률을 로드합니다.
