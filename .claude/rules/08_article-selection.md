# 08. 투자 뉴스 기사 선정 규칙

> 같은 기업의 투자 뉴스 기사가 여러 개일 때, Deal 테이블에 저장할 최적의 기사를 선정하는 기준

---

## 1. Deal 테이블 필드 (목표)

**기사에서 추출해야 할 정보:**
- ✅ 기업명 (필수)
- ✅ 업종
- ✅ 투자단계
- ✅ 투자자
- ✅ 투자금액
- ✅ 지역
- ✅ 직원수

---

## 2. 기사 선정 점수 시스템

### 필수 조건
```
❌ 기업명이 없는 기사 = 즉시 제외
✅ 기업명이 있는 기사만 점수 계산
```

### 점수 배점 (총 11점)

| 항목 | 배점 | 예시 |
|------|------|------|
| **투자금액** | 3점 | "100억원 투자", "Series A $10M", "50억 규모" |
| **투자자** | 3점 | "알토스벤처스", "삼성벤처투자", "카카오인베스트먼트" |
| **투자단계** | 2점 | "시리즈A", "프리A", "시드", "Series B" |
| **업종** | 1점 | "AI", "헬스케어", "핀테크", "푸드테크" |
| **지역** | 1점 | "판교", "서울", "부산", "강남" |
| **직원수** | 1점 | "직원 50명", "팀원 20명", "임직원 100명" |

**최고 점수: 11점**

---

## 3. 동점 처리 규칙 (우선순위)

```
1순위: 점수 (11점 만점)
   ↓ 같으면
2순위: 글자 수 (많을수록 우선)
   ↓ 같으면
3순위: 발행일 (최신 우선)
   ↓ 같으면
4순위: 사이트 랭킹 (상위 사이트 우선)
```

**사이트 랭킹 (Top 10):**
1. 벤처스퀘어 (9)
2. 스타트업투데이 (11)
3. 아웃스탠딩 (13)
4. 더브이씨 (8)
5. 스타트업엔 (12)
6. 블로터 (22)
7. 이코노미스트 (23)
8. 플래텀 (10)
9. AI타임스 (19)
10. 뉴스톱 (21)

## 4. 점수 계산 예시

### 예시 1: 고득점 기사 (9점)

```
제목: "AI 스타트업 '테크이노', 알토스벤처스·삼성벤처로부터
      100억원 규모 시리즈A 투자 유치"

본문:
"AI 기반 헬스케어 스타트업 테크이노(대표 김철수)가
알토스벤처스와 삼성벤처투자로부터 100억원 규모의
시리즈A 투자를 유치했다고 25일 밝혔다.
판교에 본사를 둔 테크이노는 현재 직원 50명 규모다."

점수 계산:
✅ 기업명: 테크이노 (필수 충족)
✅ 투자금액: 100억원 → 3점
✅ 투자자: 알토스벤처스, 삼성벤처투자 → 3점
✅ 투자단계: 시리즈A → 2점
✅ 업종: AI, 헬스케어 → 1점
✅ 지역: 판교 → 1점
✅ 직원수: 50명 → 1점

총점: 11점 (만점!) ⭐
```

### 예시 2: 저득점 기사 (3점)

```
제목: "스타트업 테크이노, 투자 유치 성공"

본문:
"AI 기반 스타트업 테크이노가 최근 투자를 유치했다."

점수 계산:
✅ 기업명: 테크이노 (필수 충족)
❌ 투자금액: 없음 → 0점
❌ 투자자: 없음 → 0점
❌ 투자단계: 없음 → 0점
✅ 업종: AI → 1점
❌ 지역: 없음 → 0점
❌ 직원수: 없음 → 0점

총점: 1점
```

## 5. 동점 처리 예시

### 시나리오: 같은 기업 "ABC 스타트업" 기사 3개

```
기사 A:
- 점수: 8점
- 글자수: 1500자
- 발행일: 2026-01-25
- 사이트: 벤처스퀘어 (1위)

기사 B:
- 점수: 8점
- 글자수: 2000자
- 발행일: 2026-01-24
- 사이트: 플래텀 (8위)

기사 C:
- 점수: 8점
- 글자수: 2000자
- 발행일: 2026-01-25
- 사이트: 뉴스톱 (10위)

처리 순서:
1. 점수 비교: 모두 8점 → 동점
2. 글자수 비교: B, C가 더 많음 (2000 > 1500) → A 탈락
3. 발행일 비교: C가 최신 (01-25 > 01-24) → B 탈락
4. 최종 선택: 기사 C ✅
```

## 6. 적용 방법

### 자동 적용
- Claude Code는 세션 시작 시 이 파일을 자동으로 읽음
- 투자 뉴스 기사 선정 작업 시 자동으로 이 기준 적용

### 수동 확인
- 사용자가 "기사 선정 기준이 뭐야?" 물으면 이 파일 참조
- 기사 선정 작업 전 항상 이 규칙 확인

### 점수 계산 프로세스
```
1. 기업명 확인 (없으면 제외)
2. 7개 항목 점수 계산 (0-11점)
3. 같은 기업 기사끼리 점수 비교
4. 동점이면 글자수 → 발행일 → 사이트랭킹 순 비교
5. 최고 점수 기사 선정
```

## 7. 키워드 매칭 가이드

### 투자금액 (3점)
**인식 패턴:**
- "100억원", "50억 원"
- "$10M", "1000만 달러"
- "약 50억", "총 100억 규모"

### 투자자 (3점)
**인식 패턴:**
- "알토스벤처스", "삼성벤처투자"
- "KB인베스트먼트"
- "~로부터", "~에게", "~의 투자"

### 투자단계 (2점)
**인식 패턴:**
- "시리즈A", "시리즈 A", "Series A"
- "프리A", "Pre-A"
- "시드", "Seed"
- "브릿지", "Bridge"

### 업종 (1점)
**인식 패턴:**
- "AI", "인공지능"
- "헬스케어", "의료"
- "핀테크", "금융"
- "푸드테크", "식품"
- "이커머스", "커머스"

### 지역 (1점)
**인식 패턴:**
- "판교", "강남", "서울"
- "본사를 둔", "위치한"
- "부산", "대구", "광주"

### 직원수 (1점)
**인식 패턴:**
- "직원 50명", "임직원 100명"

- "50여명", "약 100명 규모"

## 8. 예외 상황

### 기업명이 여러 개인 경우
- 메인 기업명 기준으로 그룹핑
- 인수/피인수 관계면 피인수 기업 기준

### 투자자가 여러 명인 경우
- 투자자 3점은 동일 (1명이든 10명이든)
- 리드 투자자 우선 표기

### 금액이 범위인 경우
- "50~100억" → 중간값(75억) 적용
- "최대 100억" → 100억 적용

## 주의사항

1. **점수는 객관적으로**
   - 명확히 명시된 정보만 점수 부여
   - 추측/유추는 점수 안 줌

2. **중복 기사 제거**
   - 같은 기업, 같은 투자 건 = 1개만 선택

3. **최신성 중요**
   - 같은 점수면 최신 기사 우선

4. **신뢰도 중요**
   - 사이트 랭킹 상위가 일반적으로 더 정확
